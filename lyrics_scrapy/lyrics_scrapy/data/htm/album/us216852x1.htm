Album Intro：

專輯名稱: 穿越
張聖子 – 小露Lucia

此張專輯為小露第一張個人專輯也是玩樂(ㄩㄝˋ)主義製作的第一張專輯.此張專輯為雙製作人(小埔&賴暐哲)共同製作.還有一首由凃惠源大師親自操刀製作.貫通了小露之前曾經出過但沒正式發行的幾首歌曲.包含由製作人&經紀人小埔創作.小露喜歡的中國風歌曲.於2016/4/30誕生的戰舞春秋.也在此張專輯中有了續作-戰非還.還有小露在2017/9/25的全創作單曲奇蹟.也改編曲後收錄亮相.本專輯主要的架構來自於小露出道至今的一些心路歷程以及生活經驗.很多很多的歌手&藝人在現今的大環境中被擊倒.但是慶幸的是小露一直在直播這條路上.堅持著自己喜愛的歌唱.並且也因為粉絲們的支持.先後拿到很多平台非常亮眼的成績單.目前也持續在17直播保持每月前10主播的殊榮.特別在這張專輯中.收錄了為粉絲寫的奇蹟改編版.用以感謝長久以來大家的支持.也謝謝大家給了小露新一代美聲歌姬的稱號

小露曾經受邀上過多個不同的新媒體節目.對小露熟悉的朋友.應該也都沒有忘記小露因為音樂這條路.曾經感受到超越自己承受極限的莫大壓力.也曾經為了這些傻過一次.但是小露最後跟大家說.感謝主.因為這些事.讓他看透很多其他的事情.走過以後更了解自己的不足.對曾經的夥伴也深感抱歉.只想在這一張專輯中.把歌唱好來回報過去的一切.這一段故事也藉由主打歌-穿越來做一個簡單有力的描述.並且加入了另外一個跟小露有關的特色.來自阿美族孩子的聲音.除了說故事以外.也想用這首歌.替那些走在這條路上的朋友們打氣

收錄曲目如下

1. 穿越
詞 : 張聖子&小埔 曲 : 賴暐哲 編曲 : 賴暐哲
原住民元素+EDM(future bass)元素.融合了小露的親身經歷後誕生的一首歌.裡面還有很多與小露的堂哥合作.具有震撼張力的阿美族母語.訴說了一個在外打拼的阿美孩子的故事

2. 你給的
詞 : 呂程程 曲 : 呂程程 編曲 : 凃惠源
由老師凃惠源以及師母呂程程精心編寫&製作.情感極其細膩.描述與戀人共同擁有過許多美好卻又分開的歷程.發洩過後重拾自我而求重生的一首歌曲

3. 戰非還(戰舞春秋續作)
詞 : 小埔 曲 : 小埔 編曲 : 賴暐哲
戰舞春秋續作.戰舞春秋描述的是一位將軍的出征及一位舞姬的不捨與盼望夫軍戰勝歸國.剛柔並濟的歌曲.而這首歌接續了前作揭曉了後續的故事.再次由小露與製作人小埔共同合作.訴說可惜天不從人願.戰爭的確贏了.舉國歡騰.但是卻不見凱旋行列中的夫君.由等待進而主動找尋線索.再到絕望的詢問蒼天.是否能夠有答案


4. 奇蹟
詞 : 小露 曲 : 小露 編曲 : 賴暐哲
以粉絲的角度寫出的一首帶有甜甜的愛戀的歌曲.敘述了粉絲觀看直播以及與主播相遇 相知 相惜 的情感


5. 殤 詞 :
小埔 曲 : 小埔 編曲 : 賴暐哲
一樣是一首小露喜歡的中國風歌曲.但不同的是此歌曲是用搖滾的方式著手.磅礡的呈現方式.敘述了戰爭的多個面相.闡述了悲喜無常的中國古歷史


6. 寂寞暴風雪
詞 : 羅凱 曲 : 羅凱 編曲 : 賴暐哲
這首歌也是一首小調悲傷情歌.在愛情的背叛與欺瞞中.選擇成全並且果斷的轉身.自己承受傷痛也不願意在對方面前落淚的骨氣.用最不屑的方式.灑脫的全身而退
