// 革命性蛻變 風格獨具的完美傑作

音樂鬼才 金曲製作人  陳星翰Starr Chen 全盤操刀製作
新世代超強女聲 關詩敏 詞.曲.演唱 創作能量 耀眼迸發
睽違樂壇四年，關詩敏以全新樣貌展開革命性蛻變。揮別『星光甜心』稱號，新作【MASTERPIECE】，揭露了沉潛時期的生活歷程與熟成體悟，一展豐沛的創作能量，塑造嶄新風格。加盟亞神音樂後，關詩敏力求音樂事業新起點，2020年發行第三張全新專輯【MASTERPIECE】，由金曲製作人陳星翰Starr Chen整盤掌舵。

// 新三關：國際關。感情關。人生關。
嶄新出擊 亮麗變身
全新關詩敏 直接表達 //

24歲的關詩敏，來自洛杉磯，在上海就讀音樂學校，現在生活於台北。求學期間曾在網路翻唱多首歌曲，引起關注與討論度，並且積極參與海內外各類歌唱比賽，屢獲得佳績，被譽為華語樂壇明日之星。基於海外獨立自主的成長背景，養成了關詩敏國際觀的視野與積極冒險的人生態度，而這些獨特成熟的思維與個性，也造就了她成長為藝人的重要養分，體現於各種領域之中。
跨界跨區的年輕世代成長背景，與獨立熟成的歷練，培養出關詩敏獨特敏銳的流行音樂品味與喜好，不僅在全新專輯大展她多變精進的歌唱技巧與創作才華，躍升為新世代女創作人身份，包辦多首極具國際水準的全新單曲，與新世代潮流創作者腦力激盪刷新三關：國際關、感情關、人生關，完全表達全新關詩敏！在音樂上更親力親為參與多首歌曲製作混音過程表達想法。

// 話題之作 首波回歸音樂主場單曲
《別弄花我的妝 feat. Karencici》//
關詩敏ＸKarencici 聯名《別弄花我的妝》新世代潮流創作人女力爆棚！
「美麗沒有標準 Love myself 就是信仰」，Girl power 魅力爆棚！
關詩敏 Ｘ Karencici聯名創作《別弄花我的妝》，創作開始兩人皆同意不以愛情為主題，並拋出「兩個女生面對這危險世界的看法」，以「妝」作為整首歌曲之主軸，編曲則為Reggaeton與Pop曲風，加入濃烈節奏的冒險風格，帶著反叛的性感氣息，同時也融合兩位新世代潮流女創作人不同的個性與魅力，講述著現代不平衡的審美觀，並挺身而出告訴女孩們在面對外界評價與衡量時，能一起保有不畏懼、不妥協的精神，不讓外面的評論影響到自己、讓自己哭或憤怒而弄花自己完美精緻的狀態。每個人都是獨一無二，不完美又何妨？請相信自己就是最美的！

MV影像力邀線上當紅導演 謝乾乾x蘇聖 SuZan為讓視覺有不同的體驗及融入於音樂節奏之中，不惜耗資成本大玩合成與特效，突破拍攝限制，打造濃烈電玩風格，並以關詩敏與Karencici的女力冒險歷程為腳本發想，兩人共同任務是開啟沉睡已久的神秘力量，神秘之鑰掌握在Karencici手中，在抵達目的地前，關詩敏早已安排好將鑰匙奪走的計畫，自行開啟這神秘的力量，而這股神秘力量跟她也有緊密相連的關係…這世界沒有誰是誰非，或是還黑白即知的真相與事實，在MV後段巧妙的暗喻安排，暗示著人性與個性本質，也隱喻著召喚「自我」的意識，同時也為重新出發的關詩敏開啟新的世界。

// About 關詩敏 Sharon Kawn //
關詩敏，音樂教父陶喆和天王周杰倫齊聲讚譽的 好聲音，2011年以15歲之姿參加『華人星光大道』歌唱節目，創下星光史上最年輕的總冠軍紀錄，賽程優異表現，讓她如願順利踏入樂壇並發行第一張專輯 【關在家】; 隨後參加浙江衛視《中國好聲音》第四季，並加入周杰倫戰隊，最終成功晉級戰隊四強，接著發行第2張個人專輯【綻放2.0 Blossom】，以甜美青春樣貌與完美歌藝深植人心。 2020年發行全新專輯【MASTERPIECE】，蓄勢待發，成績指日可待！
勇於接受各種挑戰的她，積極發表作品於專輯中，展現詞曲創作的實力；也勤練舞蹈在音樂影像中展露舞藝令人驚豔。為力求演藝事業突破，關詩敏更跨足戲劇圈參與影視劇，表演能力絕佳的她，並連三年受邀參與歌舞劇巡迴演出，人氣實力兼具。可歌可舞能創作的全方位形象，也讓關詩敏於華語音樂圈、戲劇圈備受各界期待與關注，表演佳績有目共睹！為樂壇不可多得的新世代優質全能歌手。
