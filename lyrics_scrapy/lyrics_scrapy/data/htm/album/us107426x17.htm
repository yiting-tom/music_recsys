Album Intro：

Wonder Girls,日攜新專輯“Why So Lonely”華麗回歸！
出道10年以來，第一首自作曲“Why So Lonely”再次堅定了作爲音樂家的志氣，幷爲第一首自創的主打歌。

◎Wonder Girls，攜收錄了第一首自創主打歌的新專輯“Why So Lonely”華麗回歸。
◎主打歌“Why So Lonely”展現了更多Wonder Girls的音樂才華，並展現了只屬於Wonder Girls的獨特音樂。
◎不需要額外修飾語的Wonder Girls，攜自創主打歌“Why So Lonely”全新回歸！這次不是JYP首長樸振英製作的歌曲，而爲嶄新的自創主打歌。

Wonder Girls攜全新專輯“Why So Lonely”華麗回歸。

Wonder Girls的全新專輯“Why So Lonely”，共收錄了同名主打歌“Why So Lonely”，“Beautiful Boy”和“Sweet & Easy”。

去年的正規專輯“REBOOT”展現了成員們的作詞作曲能力，幷成功轉變爲極具魅力的4人組樂隊，這次的Wonder Girls爲歌迷們展現出了更深一層的魅力，同時發揮出了更出色的音樂實力。

主打歌“Why So Lonely”爲Wonder Girls的第一首自作曲，出道9年以來，一直將樸振英創作的歌曲作爲主打歌的她們，這次卻離開了樸振英的懷抱，作爲實力歌手發表了一首自創主打歌，可稱之爲Wonder Girls成長的結晶。

“Why So Lonely”爲Wonder Girls第一次嘗試的雷鬼風格的流行音樂，由極具中毒性的Leaf Guitar和豐富多彩的節奏變化相結合構造出的一首魅力無窮的歌曲。將“你也要像我這樣沉迷”，“就你帥氣，就我的心受傷”等情侶之間所能感受到的冷嘲熱諷的內容，用可愛、風趣的Lyrics詮釋了出來。

另外一首“Beautiful Boy”是由作曲家Frants和Wonder Girls成員們攜手一起創作的歌曲，這首歌展現出了70年代樂隊曲風，幷傳達了在人與人之間越來越冷漠的數字化時代裡，希望成爲互相更珍惜、更疼愛的世界的一種心願。

最後一首“Sweet & Easy”爲適合夏天聽的流行搖滾風音樂，與歌名一樣甜爽又平淡的旋律與極具韻律感的吉他聲以及甜美的Lyrics相融合，給歌迷們留下不可磨滅的印像。

從女團開始，成功轉變爲樂隊的Wonder Girls這次則攜全新專輯展現出了更突出的音樂性，幷將展現出更高更確實的存在感。出道10年不停變化的“最高等級的音樂家Wonder Girls”，期待這次又將會是一個多個性、多美妙的變化。
