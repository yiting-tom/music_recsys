Crazy Summer Love
是Nora Says在2014年暑期推出的最新作品
藉由這張EP想傳達給大家三種層次的愛
08/22搖滾的愛，深情的愛，抒情的愛。
在這炎熱的夏季裡，放手瘋狂的再愛一次吧!!

關於Nora Says
Nora是團員Samuel大女兒的名字。在2012年Nora的出生是Samuel這輩子最好的禮物，就像音樂對Samuel來說也是一份上天掉下來的禮物。也因為如此，讓Samuel 和Greg 找回了對音樂最開始的堅持與執著，取這個團名透過音樂來表達自己。
Greg 與 Samuel 2007年於美國洛杉磯認識，在2009年從美國帶著組團的夢想回到台灣。今年他們決定要嘗試不一樣的東西，想透過跟其他音樂圈的朋友們合作，創造出不一樣的火花，於是找了很多擅長不同風格的歌手，依照每位歌手的風格創作所有的作品，另外也跟不同的混音師，攝影師，導演合作，就是想要帶給大家全新的聽覺和視覺上的衝擊。因為有這些人加進來各類型的元素，使得Nora Says的音樂、影像更加的精彩，而且讓聽眾們有耳目一新的期待。
Nora Says 就是不想把音樂侷限在樂團的模式.
於是找了不同聲線和詮釋的歌手，企圖創作不一樣的搖滾專輯。
Crazy Summer Love歌曲介紹
搖滾的愛 主打 : 州界女孩feat.鄭宏澧 ISRC CODE TW-V87-14-88001
主唱：鄭宏澧 詞：鄭宏澧/ Greg 曲：Samuel/ Greg
自從有暑假要發EP的計畫以後，我們想著，要寫什麼樣的歌最能夠代表此刻的Nora Says，如果說去年底的專輯，大部份都是邀請女歌手來詮釋，那這張應該是要由男性的角度出發，於是寫出了這首偏金屬的搖滾音樂，讓聽眾對Nora Says的音樂又有不一樣層次的認識。
Lyrics的方面，主要是在描述男女之間的一種曖昧關係，常常就像那一條跨越兩州的界限，時刻都在試探著對方，希望得到多一些證據，對方的壞又常常讓自己無法自拔，雖然很愛很愛，誰都不會輕易超越那條界線。

深情的愛 主打 : I Do ISRC CODE TW-V87-14-88002
主唱：Samuel 詞：Greg曲：Samuel/ Greg
“I Do”是一首深情創作，創作時就定位了EP裡面需要一首攪動內心的情歌，情歌歷久不衰是因為它們的故事性，另每一個人都可以深深的連結，這首歌有別於時下情歌賣弄文青優美氣息，我們從另一面很深很濃的切入點傳達，不經修飾的把情感放在每個人眼前，“I Do”這句話是對婚姻的承諾，是永恆的見證。
抒情的愛 主打 : 愛著你feat.吳海文 ISRC CODE TW-V87-14-88003
主唱：吳海文 詞：吳海文 曲：Samuel/ Greg
“愛著你”是一首抒情的中板情歌，其實這首歌的主要旋律寫出來已經好些日子了，但是一直沒有找到適合的時機點發表，正好這次EP的主題很適合把“愛著你”這首歌放進來成為一個完美的總結。另外，我們覺得邀請吳海文來詮釋是一個很棒的選擇，他的聲音非常溫暖，真誠，讓人沒有壓力，能完全把歌曲的表情詮釋出來，歌曲的副歌 “there’s no one like you. Oh, there’s no one like you.” 傳唱著你就是獨一無二的，關於愛，這不就是每個人都需要的嗎？沒有你，我怎麼完全呢？
