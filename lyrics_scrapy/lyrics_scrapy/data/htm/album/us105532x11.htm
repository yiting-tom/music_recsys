Album Intro：

正宗神曲重新定義｜正宗女神搖滾回歸
八三夭 x Sammi 鄭秀文
[ 神曲大帝國 ] 首發#01單曲 [ 眉飛色舞PLUS ]
港嗨電音女神鄭秀文 台語饒舌搖滾初體驗
重低音低車身 x 經典舞曲搖滾變身 = 超越2017神前衛台客搖滾

「曾經在香港演唱會上翻玩改編 [ 眉飛色舞 ]，如果真的找來原唱 Sammi 鄭秀文，一定很酷！」

吉他手劉逼以破音電吉他開場，直率宣告電子舞曲完全改頭換面，搖滾版本嶄新誕生。
solo 樂句低音高音齊狂飆，power chord 與霸天的 bassline、阿電的鼓點完全貼合，挑戰顛覆過去舞曲印象。團長兼鍵盤手小橘，以合成器上下震盪的旋律，扮演畫龍點睛、不可或缺的最佳配角。
搖滾充斥龐克氣味，輔助電器音色點綴，曲式熟悉又新鮮，曲風多變而新穎。

原唱鄭秀文在聽過八三夭 [ 眉飛色舞PLUS ] 的 demo 後，欣然答應邀約，讓一場異想天開的合作企劃開花結果。
Sammi 鼎力助陣，擔綱女聲演出，高亢有力的嗓音，一開口就令八三夭驚豔不已！
「我們竟然跟電音女神合唱！」八三夭高中下課時每天都在西門町店家聽到的神級金曲，竟有一天獲得改編合唱的機會。
主唱阿璞也特地為此寫了一段國台語新詞，用字饒舌有趣，莞爾寫出了現代愛情現象。巧妙地藏了「搖落」Lyrics，音似八三夭原創歌曲 [ 搖勒搖勒 ]，替改編曲注入八三夭的記號。

阿璞也力邀金曲製作人阿弟仔共同製作，一日生活圈來回香港配唱，打造出這首在經典原曲與精彩新曲之間，完美平衡又耳目一新的翻新作品。
讓我們以港嗨聯手台客搖滾新勢力的熱情，染紅12月的白色耶誕。


正宗經典注入音樂新觀點｜正宗翻玩精神神致敬神改編
八三夭
[ 神曲大帝國 ]
回歸音樂初心 x 神級翻玩企劃 = 音樂圈的潮流聯名神作

「我們真的這麼愛玩音樂，要不要試試把過去的好歌找出來，做點有趣好玩的改編！」

新生代搖滾神團八三夭，除了詞曲作品有驚人表現，編曲、演奏能力也是一等一神手。
改編翻玩的作品，無不受到觀眾重視，曾經翻唱詮釋謝金燕 [ 姐姐 ]、玖壹壹 [ 打鐵 ]…等，
網路傳播高居不下，分享點播數甚至引起原唱注目。

發完 2016 慘綠世代創作大碟 [ 生存指南 ] 之後，八三夭陸續起跑亞洲各站巡迴，
一邊醞釀萬人演唱會蓄勢待發的能量，一邊從生活中累積各種日後創作的可能與經驗。
本性愛好音樂，本職做出好音樂的他們，忙裡偷閒開發出支線思考：
「我們真的這麼愛玩音樂，要不要試試把過去的好歌找出來，做點有趣好玩的改編？」

憑恃對音樂旋律的神級狂熱與敏銳，八三夭推出回歸音樂初心的神級企劃 [ 神曲大帝國 ]。
一網打盡神級單曲，重新編曲演奏；拆散市面上慣有的專輯概念，全然回歸音樂本質，
重新詮釋出八三夭獨特搖滾味道的翻新神作。

氣勢首發與 Sammi 鄭秀文合作 [ 眉飛色舞PLUS ]，亦請期待後續其他聯名單曲。
