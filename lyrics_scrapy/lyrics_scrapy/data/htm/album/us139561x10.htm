Album Intro：

赤世代全新單曲抒情搖滾 「I need you」
失去你的吶喊！需要你的呼喚！
樂團赤世代第七張創作單曲[ I need you ]，用抒情搖滾的編曲，述說在感情中失去後的悲傷跟遺憾，以及渴望能回到從前美好時光的衝動……
這首全新單曲 「I need you」是主唱- 介作詞作曲的作品，以自己的角度來寫一段愛情故事。故事是一對相愛的戀人，男孩在一次次無知與莽撞中傷害了女孩的心，最後終於失去了這個一直守護著他的女孩。在後悔遺憾的心情下，渴望著能改變，希望可以回到最初美好的時候。
「I need you」是渴望挽回的呼喚與心中後悔不已的吶喊！
這首赤世代新單曲將於12月13日全網上線，12月14日台北赤世代 I NEED YOU演唱會現場實體唱片發行。
在I need you的編曲想法中，吉他手修覺得應該讓樂器的音色更展現出歌曲想講的故事情緒，所以編曲特別用加了一些破音帶著淒涼效果的吉他聲，做為歌曲的前奏導入，製造出孤單悲傷的感受。 赤世代在上次「從此以後」單曲開始加入電子元素後，這次也在主歌與副歌的段落加入一些Drum Loop，嘗試著讓樂團的編曲更有一致性的色彩 。而貝斯手尚，在編曲中巧妙的以貝斯加強了與主唱弦律的聲線相互呼應。
主唱介在創作過程中特別在最後的尾奏時，加入更有吶喊唱法的弦律，更呈現出歌曲中想傳達的後悔與遺憾情感。
「I need you」 同樣找來昔日好友一起共同合作
這次赤世代在歌曲的編鼓部分繼前一張單曲後，仍找來東城衛的鼓手 - 冥，由冥來編鼓的節奏，錄音也同樣是赤世代第一代的鼓手小育負責錄製鼓的部份。
當然在視覺設計上，赤世代與好友東城衛 Bass 手鐙繼續合作！
這次「I need you」單曲依然是鐙主導整個視覺設計！
視覺設計用知更鳥代表著愛情，在溫暖的色調中表達著愛情的美好，展翅的知更鳥也有著離去的含義，而在內頁照片中團員們拿著空的鳥龍也象徵著已經失去的愛情。
「I need you」全新單曲巡演場次將在12月14日台北-後台BackStage Café展開！這是繼2018年生存之道單曲後，首場的單曲巡演從台北開始，這場有著全新單曲「I need you」首演的台北專場演唱會！值得期待！不要錯過了！
