Album Intro：

林子良 第三張創作專輯 ' ORIGINAL FLAVOR '

ORIGINAL FLAVOR，也就是 ' 原汁原味 '。 最近大家一定很習慣，流行音樂目前已活在一個電音和舞曲的混合中。 林子良的 '原汁原味' 網路新專輯堅持強調自己原本的優雅質感旋律風格。 一直以來持著自己的 R&B 唱功唱腔風格，在新專輯裡可以感受到林子良這十多年經歷過的成熟與毅力。 對於感情，環境，與藝術的成長過程，都全寫在這八首歌裡面。 一瓶好的單一純麥成年威士忌，加了冰塊，也許還能接受；可是原味品嘗時，可以感受到這杯好酒的珍貴層次。 一道頂級好菜，加了過多鹽巴、辣椒，也可能還是好吃，可是原味品嚐時，肯定可以感受到所有鮮味的細節。 好功夫，不管是藝術、武術、還是音樂，原汁原味，就是所有本質的基本功。讓我們一起感受一下林子良的' 原汁原味 '！！！

' I Never Told You So '
夏天要來了，在高溫的季節中，天下的情人們永不變冷。 最熱情的告白方式表白， 好想擁有你，把你好好的寵壞……' I Never Told you So' ！！！

' 天黑以後 '
天黑以後，無處躲的低溫度，隨行跟著心的痛。 你的味道我還聞得到。 閉上眼睛，更加明顯。

' 從朋友開始 '
從朋友，慢慢的發展出火花，熱情，感情。 一首描述無法預料的單純輕快情歌。

' Get Closer '
火熱的挑逗和撒嬌，心跳中的衝動，只想被對方的眼神完整的融化掉。 無限幻想一直要 Get Closer ！

' 騎單車 '
青梅竹馬，從小到大騎單車我載著你。 隨著時光走，我依然還是載著你。 看著現在的小情侶們，也正在騎單車談戀愛。 一起迎接日落。

' 蜜兒 '
甜得不像話，找不出理由啦！

' 我的天后 '
妳是我的天后，讓我永遠為妳醉落……我內心為妳尖叫，希望自己是個處男所有歷史清掉。

' 蝴蝶結 '
切斷了我們綁的蝴蝶結，請帶走我們盡力愛過了的情節，好讓我哭泣放掉妳和我曾經的世界。
