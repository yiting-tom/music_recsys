姚謙+姚若龍+郭子+偉菘+齊豫+李劍青+小安 ..等全亞洲頂尖詞曲 全力打造!
在歌聲中慢舞 在愛情中漫遊 在no name的歌聲中 慢慢延伸



在2000年參加了唱片公司的歌唱比賽榮獲冠軍後，幸運地簽了約，卻又因唱片公司的人事變動而解約，2002年入伍服役，2003年退伍，一直為他的歌聲所吸引的資深音樂人姚謙就這麼等了四年，終於簽下這個好聲音。No Name 學過八年鋼琴，也會彈吉他，天賦音質特殊，一直以來他只是單純地愛唱歌，單純地想與大家分享他的心情，證明自己能夠讓聽者因為他的聲音而得到感動與安慰，甚至是一種幸福感。而2004年首張同名專輯問世後，在市場一片低迷的狀況下以實力賣出一片好成績，「愛情帶來的改變」、「守護天使的告白」、「愈愛愈愛」等主打歌首首成為兩岸三地KTV情歌點唱榜上金曲，也意外地與飛兒樂團在2004年成為最高尋問度的年度新人，這一次 No Name 余憲忠將以全職歌手的定位再戰樂壇，這一張「慢慢愛」將讓我們再一次溫暖感受到他的歌聲以及音樂忠的濃郁情感。

　　溫柔繫好男聲 No Name 余憲忠

　　慢工細活打造個人第二張全新專輯

　　因為 No Name 余憲忠 我們的愛情 有了名字…

　　慢·慢愛 慢 慢 感動 慢慢 聽見感情生活的溫暖光芒

　　等待，只是為了邁向成功之路所做的暖身。No Name 在音樂路程上，用比誰都漫長的時間，決定將自己的投身於音樂世界中。從正式簽約成為歌手到第一張音樂作品出聲，No Name 余憲忠的音樂之路並不像其他歌手來的順利，他回憶並笑著說「這是上天對他的考驗，測驗他對音樂的熱情與耐心。」後來遇上他音樂的伯樂─姚謙，No Name 余憲忠的音樂之路才漸漸出現一線曙光。正式出道後出版第一張同名專輯，特別以「出聲不露面」的企宣方式，果然在強片環繞的音樂市場中，打響「No Name」 的名號，讓消費者聽到真正的好音樂、好聲音，No Name 也因此烙印無數樂迷心中。第二張專輯《慢·慢愛》可說是這些音樂歷程匯入的總結，希望能讓支持好音樂的樂迷，透過這張音樂作品，更加貼近 No Name 余憲忠，感受他對音樂的熱情與付出。

　　No Name 余憲忠第二張專輯《慢·慢愛》，延續他溫暖動人的歌聲，在2006年初春再次用歌聲與每個尋求慰藉的人做一次內心感動的交流。現在傳媒講述的「慢速新生活」理念只談食衣住行，反而鮮少於「育樂」領域有這種概念的傳達，因此 No Name 希望藉由音樂力量，傳播能讓大家感受到「生活慢愛」與期待「新音樂運動」的第一人。

　　無論是慢活、慢食的推廣，不外乎就是希望讓每一個人在快速失調的壓力生活下，能重新調整生活步閥，將自己回歸生活起點，去享受生活的過程，無須在乎結果的定論。《慢·慢愛》專輯就是要闡揚這樣的生活理念。《慢·慢愛》收藏許多對親情與愛情的體貼與關心、對生活的期許與展望，No Name 余憲忠再次將他極具暖身的聲音，毫無保留盡情展現，如初春暖烘烘的太陽般，溫暖和煦地照耀每個人身上，專輯中使用大量生活化的簡單筆法，不唱高調、不走抽像，企圖在尋找平鋪直敘的過程中發展出一種屬於你我與 No Name 共同擁有、相互聯結，誠摯的真性情與真感動。希望每位樂迷聽見他的音樂，透過他歌聲，告別緊張煩悶的生活壓力，重新啟發每一個人的美麗人生。
