炎亞綸加盟索尼音樂 首張作品《摩登原始人》12/25誕生

2019年到2020年是我們身處的摩登地球進入巨大變化的野蠻時代。2019年到2020年也是炎亞綸歌手生涯的重要轉捩點。離開舊東家，成立自己的經紀公司，並在今年十月宣佈加盟索尼音樂。炎亞綸同時向大家預告年底前會有新作品《摩登原始人》的誕生。《摩登原始人》這張作品的詞曲收歌方向以“洞悉自己才能關懷世界”為主題出發。藉由架構一個原始時代的世界觀去帶到現代人活在摩登世界所面對的，無論是對自己，或對世界的各種省思。

陳珊妮 x 炎亞綸 x Dizparity葉柏成 x Andrew葉懷佩 x 林承佑DanDan U x 黃建洲 x 陳信延 聯手炮製摩登音樂大餐

「這張作品的概念可能就是我說的“心靈環保”吧。當我們越來越進步的同時，請不要忘了我們都該不斷檢視自己的心靈與了解世界上的飽滿和缺失。要保留初衷，以及別做無謂的浪費。」詞曲統籌炎亞綸邀請了陳珊妮擔任專輯製作統籌，加上眾多才華洋溢的音樂人打造色香味俱全的摩登音樂饗宴。曲風涵蓋了北歐house EDM、nu-disco、latin house、downtempo electronica、alternative synthwave rock、chill trap。全張作品的Lyrics視角有趣且饒富深意，引人省思。首播搶先聽同名主打歌「摩登原始人」於12/7首播，為炎亞綸的摩登原始世界率先端出一道令人驚艷的漂亮前菜。

聶永真 x KRIS KANG x 張以得 x 徐于筑 藝術家齊聚編織獨樹一格摩登原始世界精彩外衣

「我們永遠活在正在過期的前一秒。相對於未來，此刻的這一秒已是原始。」聶永真團隊利用視覺語言重譯時間的相對論，以摩登對比原始的反差，為這張作品注釋了一套cutting-edge的論述。限定版與常規版，雙版本實體CD以純白無邪的外衣包覆著的是一首首警世寓言。造型師徐于筑與炎亞綸討論後決定以多套非當季的精品作為搭配，以grunge元素緊貼著「摩登」與「原始」的概念，進行最狂野的美感碰撞，重新定義「時尚不會過期」的概念。攝影師KRIS KANG與3D虛擬設計師張以得分別以黑白底片和數位繪圖技術，搭配妝髮大師的集體創作，解構與翻玩了炎亞綸全新「摩登原始人」形象。

dosomething王宗欣 x 吳仲倫 率領新媒體團隊打造摩登原始人的先鋒建築美學

「炎亞綸作為一個明星，他不只是偶像，更是一個創作者。」dosomething主理人王宗欣受邀成為《摩登原始人》這張作品的新媒體藝術總監，從開案開始就不斷與炎亞綸和團隊頻密開會，試圖為這張音樂作品開創全新的藝術視角。王宗欣聯合眾新媒體創作者，包括影像導演吳仲倫，用超現實的描繪與想像，去思考現代人活在這個摩登世界中，所面臨到的自己對自己、社會、大自然之間的各種議題，也傳達理想中的世界是什麼模樣，架構起《摩登原始人》的前衛建築美學。

索尼音樂 x 晴空鳥 幕後團隊精心策劃一趟跨越時空之旅共同飛往摩登新世界

「藉由這張作品音樂上的轉型，重新定義炎亞綸的創作人形象，創造出跨時代的代表作。」從12月開始，索尼音樂與晴空鳥即將精心策劃一系列結合《摩登原始人》理念的新穎企劃宣傳規劃。從2020年橫跨2021年，宣傳活動鋪天蓋地而來，歡迎加入摩登原始人隊伍並留意最新消息，一起共赴摩登原始人之旅。
