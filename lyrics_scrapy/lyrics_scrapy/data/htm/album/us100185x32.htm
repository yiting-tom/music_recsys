十首一時之選 十種情緒切面
2020.3.9 《面面》 全面現聲


「你就是你每天生活的累積，那會讓你越來越清楚自己想要的樣子是什麼。快樂或不快樂，承擔或承擔不了，都不斷地累積成你的各種面向；面面代表著人的不同面向，而我的面面，其實也是每個人的面面」──蘇慧倫

橫越過馬不停蹄的風光年代，蘇慧倫不斷開創指標、自我重整，90年代她以「玉女掌門人」之姿橫空出世，再以怪奇前衛的《Lemon Tree》、《鴨子》、《傻瓜》變身三部曲引領流行，後來她從華麗喧騰出走，喚醒體內的文藝靈魂，唱出溫柔質樸的《戀戀真言》。在華語流行音樂的光譜裡，她是無法被取代的色票，如此獨樹一格的存在。2016年蘇慧倫發行了〈我們快樂地向前走〉單曲時，她已升格人妻人母，將重心挪移至家庭，但她仍然不忘愛歌唱的心，2017年技癢重返舞台，參加音樂節目《蒙面唱將猜猜猜》，清透溫柔的高辨識度嗓音，一開口便叫眾人懷念，當年席捲樂壇的玉女偶像終於歸來。爾後蘇慧倫再度告別鎂光燈的追捧，與樂壇保持若即若離，但她清楚明白許多死忠歌迷仍靜靜等待她的聲影出現，2019年底推出壓軸回歸單曲〈為你變成他〉，正式宣告睽違樂壇13年、慢火催生的全新國語專輯即將誕生。

極致陣容 細緻打造 頂尖音樂人聯手雕琢第16號蘇慧倫
陳君豪 黃韻玲 四分衛 陳如山 葛大為 HUSH 張震嶽 魏如萱
艾怡良 TIZZY BAC陳惠婷 黃建為 黃韻仁 Yellow黃宣

蘇慧倫第16張全新國語大碟《面面》，以生而為人本有各種面向為概念出發，無論快樂的、悲傷的、幻想的、脆弱的，甚至黑暗的各種層面，都是與生俱來的情緒表露，我們又何必過度隱藏真實的自我。由於蘇慧倫闊別樂壇許久，因此在設定全新專輯走向時，團隊也格外用心，嘗試在不同路線中迷走摸索，找到現階段最適合蘇慧倫的模樣。過往的蘇慧倫若是遙不可及的玉女偶像形象，現在的她更想以真實落地的姿態面向歌迷，不再片面、更加全面，透過十首歌曲表達十種情緒切面，蘇慧倫內心擁有的面面，其實也是你和我內心會投射出的面面。

而多年未投入完整音樂作品的蘇慧倫，在專輯開案時，深怕無法達到自己設下的標準，因此常滿懷忐忑不安的心情，但當開始進入收歌階段時，喜好分明的蘇慧倫迅速找回過往的精準耳感，陸續順利收到自己相當滿意的作品，其中許多音樂創作人一交歌，更是準確契合蘇慧倫對於音樂的期待，也讓久未發片的她吃了定心丸。蘇慧倫過往擁有多首合唱暢銷金曲，新專輯也特別邀請欣賞彼此許久的「娃娃 魏如萱」友情獻聲，實力派女聲加持，一首華語流行音樂女力合唱新經典即將誕生。而蘇慧倫從前端收歌期到後製混音，全心投入每個環節，展現更多音樂自主性，求好心切的專輯製作人陳君豪，不僅將專輯母帶特地送至美國製作，母帶後期混音處理更由曾操刀Adele、Lady Gaga以及Justin Bieber等歐美流行巨星專輯的Randy Merrill負責，一張「非常蘇慧倫」的精品大碟就此問世。

新銳創作視角 風格影像美學 面面呈現不一樣的蘇慧倫
時尚攝影師周墨 金曲裝禎設計師楊士慶 細膩美感極大化

為讓整裝再度回歸的蘇慧倫散發耳目一新的視覺形象，主視覺拍攝邀請曾操刀VOGUE、美麗佳人、Bella等時尚雜誌的攝影師周墨掌鏡，捕捉蘇慧倫柔美卻又不失個性的風貌。專輯封面主視覺則透過投影機的投射畫面，讓人看見蘇慧倫多重面貌的展現，集溫柔美麗與搞怪俏皮於一身，風格大器，卻也強烈呼應全新國語專輯《面面》欲傳達的意涵；金曲裝禎設計師楊士慶發揮極簡美感長才，在風格強烈的主視覺上，透過簡約的幾何與線條設計，彰顯面面傳達的內心情緒的不同層次。

首波抒情主打〈真面目〉真實袒露 解構自我
葛大為細膩察覺 你所認識的蘇慧倫 只是一部份的她

〈真面目〉做為專輯中的核心點題歌，團隊初期著實花費不少時間討論Lyrics走向，為更精準掌握蘇慧倫近年來的內心風景，身兼專輯製作統籌與作詞人的葛大為更親自請益擅長描繪女性心境的樂壇大哥李宗盛的看法，在交換觀點與拋出一些關鍵字的對談之後，整首歌的輪廓也逐漸成形，〈真面目〉也成為專輯概念的中心起點，由此開展出蘇慧倫欲透過音樂呈現的各種面向。葛大為細膩側寫蘇慧倫這些日子以來的心路歷程，從現階段的生活切面到身分轉換的心境，一句句Lyrics都深刻貼合蘇慧倫的內心世界，其中「當我和幸福交換條件 / 咬著牙走到今天」更讓蘇慧倫格外有感，甚至激動到想落淚；為充分體現〈真面目〉所描繪的細緻情緒，編曲歷經推翻修正，最終選擇以富有濃厚情感的鋼琴伴奏與悠揚弦樂為定案版本。

〈真面目〉不僅是一首貼近蘇慧倫現階段心境的歌曲，不同身份的女性也都能在這首歌裡找到描繪自己的樣貌。MV影像找來過往曾成功打造〈Lemon Tree〉、〈鴨子〉、〈傻瓜〉三部曲的陳宏一導演再創經典風格。以獨特美學見長的陳宏一，在此支MV安排了代表不同身份的女性角色，如家庭主婦、上班族、學生等，並巧妙為每個身份特製面具，隱喻每個人某種程度上都是戴著別人期待的模樣在生活，而藏著面具底下的真面目或許只有自己才懂。蘇慧倫初收到腳本提案時，看見導演為各種身分女性設計的不同面具，便靈機一動從Lyrics中的「當我和歲月短兵相接 / 回顧還是該往前」，聯想到自己不妨也嘗試穿劍道服、戴上帥氣的盔甲帽，拍攝當天導演完成了蘇慧倫的小心願，讓她化身帥氣女劍士，即使完全沒有劍道底子，蘇慧倫仍在MV中展現十足氣勢。而為展現每個女性心底都藏有一個真實自我的私密小世界，導演特別陳設了一個由彩色布幔所圍起的空間，象徵唯有躲在這裡獲得片刻寧靜，才能好好跟自己深度對話。導演也沿用了專輯主視覺的投影概念，將各種蘇慧倫臉部表情投射於布幔上，迷離的色彩映著柔美影像，讓人看見面面唯美的蘇慧倫。


十首一時之選 十種情緒切面

01.安和AnHe 詞 / 曲：HUSH
人的反叛面
「不顧一切遊蕩 / 人在安和路超商 / 物色不安分的流浪」
〈安和〉記錄著蘇慧倫現在對於過往某段生活切片的回味與遐想，以台北市安和路為中心座標，展開一段迷人的夜生活想像，以及自我現況對照後湧出的錯亂時差。穿梭過誘人的燈紅酒綠，年少的不羈，是酩酊邁出酒吧後的任性不想回家，而成家立業後，明亮的便利超商映著深夜加班後的身影，卻反而讓人掛念著想早點回家。曾經夜裡的肆無忌憚，如今只能在記憶裡發著光，對於夜生活的遐想，只能任它在內心的小劇場瘋狂上演。

Lyrics為更加體現遐想的真實感，歷經反覆多次修改琢磨才拍板定案，而此曲的旋律自由奔放，拼貼輕盈明亮的舞曲電氣搖滾，更讓人看好〈安和〉有機會繼「變身三部曲」後，晉身蘇慧倫最新傳唱經典。

02.為你變成他Disguise 詞 / 曲：陳如山
人的逞強面
「其實我並不那麼堅強 / 只是在白天裝模作樣」
無論在愛情或生活裡我們都曾渴望變成對方想要的模樣，為心愛的人修正自己、為家庭不顧一切、為工作奉獻心力，而一切的努力到頭來會不會反而遺失自我？那些白天的堅強模樣，其實只是勉強搬上檯面的裝模作樣，我們都可能不小心將自己演成一部沒有對白的喜劇電影。而為強化歌曲意涵，前奏特別取樣經典默片《摩登時代》片段聲響，彷彿領人走進一部無聲喜劇，隨之開展的British Rock，情緒層層堆疊，巧妙加入小喇叭的點綴，更增添些許倔強的悲傷色調──原來想要為你變成他，終究是徒留遺憾的想像。

03.永遠的一天Eternity and a day 詞：陳惠婷 / 葛大為 曲：陳惠婷
人的幻想面
「就讓我去假裝 / 我暫時不是誰的誰 / 去探險」
百無聊賴的下午時光，一陣雷雨轟隆作響，好像有什麼感覺在翻湧，那不如就任幻想綿延無疆。在固有的生活模式裡，我們將設定好的角色扮演得多麼淋漓盡致，如果有一陣閒暇空檔，我們何不讓大腦的小劇場上演偶像劇般的浪漫情節，畢竟人的幻想與欲望是無論到了幾歲都內建在身體的反應機制。〈永遠的一天〉早在2016年時便被蘇慧倫收進口袋名單，直至2020年才終於有機會曝光，此曲充滿Tizzy Bac主唱陳惠婷一貫跳躍風格，而在蘇慧倫柔軟纖細的演繹口吻，搭配電氣感十足編曲與層層堆疊亮麗音色之下，更讓人心醉神迷。

04.真面目True colors 詞：葛大為 曲：呂奇翰
人的赤裸面
「你認識的我 / 只是一部分的我」
每個人都有一部分的自己，藏在表面之下，堅強、脆弱、無助、惶恐，各種顏色的情緒也許因為旁人眼光壓力，而讓我們選擇壓抑心底。有沒有一個時候，你不再介意別人的想像幻滅，你願意無畏展示自己的真面目，你明白了，樂於欣賞自己的不完美，才能真正地快樂。葛大為透過〈真面目〉細膩側寫蘇慧倫這些日子以來的心路歷程，從現階段的生活切面到身分轉換的心境，一句句Lyrics都深刻貼合蘇慧倫的內心世界；為充分體現〈真面目〉所描繪的細緻情緒，編曲歷經推翻修正，最終選擇以富有濃厚情感的鋼琴伴奏與悠揚弦樂為定案版本。

05.遠行Voyage 詞 / 曲：黃宣YELLOW
人的缺憾面
「何時能靠岸 / 把遺憾留在一片沙灘」
決定遠行之後，就把那一片遺憾留在當時的沙灘吧，總有一天那些掛在心上的缺憾都會變得輕輕淺淺。耳邊的海風不厭其煩地呼嘯著，催促我們該抬起腳跟往前進，回頭看太陽已沉入海洋，而我們也都已往順風的方向遠行了。〈遠行〉詞曲出自以〈不開燈俱樂部〉奪下第30屆金曲獎最佳單曲製作人的Yellow黃宣之手，此曲洋溢海島愉悅風情，舒適宜人，Lyrics卻講述著內心因距離而產生的孤獨感，蘇慧倫以輕柔吟唱撫平了仍存活在心底的陣陣遺憾，即使只有自己一個人能也揚起風帆，遠洋下一段全新的未知風光。

06. 氣溫37度的遐想Summer delusion feat.魏如萱 詞：葛大為 / 蘇清清 曲：張震嶽
人的欲望面
「致襯衫的悶熱 / 治我身體的渴 / 什麼遐想都不過分」
夏天明明還沒正式報到，怎麼臉頰就開始熱了，原來是男神般的你朝了我走過來，究竟是氣溫先超過37度還是我的體溫先呢？被暖化的心如此騷動難耐，反射欲望的雙眼清澈又認真，認真極大化遐想我們之間會不會有什麼新鮮的可能。繼〈失戀萬歲〉後，蘇慧倫睽違許久再度推出女女合唱〈氣溫37度的遐想〉，甫收到由張震嶽譜曲的Demo時，眾人便決議將這首歌設定為女女合唱，歷經一陣腦力激盪後，最終邀來歌聲極具敘事感的娃娃 魏如萱助陣合唱。此曲將女人內心不為人知的欲望風景展露無遺，Lyrics也堪稱突破兩位知性女神的過往尺度，充滿遐想情節且挑逗人心的文字，讓每個人的身體也似乎渴了起來。

07. 親愛的全部 Dear all 詞：黃政彬 曲：黃韻仁
人的釋懷面
「都是你一幕幕 / 拼湊了全部 / 我全部」
不小心在口袋掏出一張手衝咖啡收據、打開衣櫃發現一條殘留眷戀氣味的圍巾、拉開抽屜原來某個紀念日的禮物還躺在那……我們的人生旅程留下了多少別人的溫度，即使現在再也感受不到當初晴朗的溫暖，心底似乎也不再感到難過或遺憾，時間繼續走，我們也繼續走，.偶爾回頭細數過往，仍心懷感謝那些，完整過我的全部。由新加坡金牌製作人黃韻仁作曲與製作的〈親愛的全部〉是專輯中的極致小品，蘇慧倫以釋然情懷回顧人生片段，在波瀾不驚的情緒起伏之下，更多是對生命透徹的領悟，是那一幕幕過往，拼湊成了我們的全部。

08. 與生俱來的黑 Darkness in the heart 中文詞：蘇慧倫 作詞：Hodge 作曲：회장님, 최철호, 유영준
人的黑暗面
「不拒絕那些遺憾 / 原來 黑色是我與生俱來」
如果人是明亮與黑暗的集合體，當我困在黑暗面的時候，你會怎麼看待流動在我身體的黑色？能不能給我一點空間，允許自己悲傷，允許自己軟弱，允許自己不對誰交代？有時候別急著給我光芒，讓我暫時躲在黑暗裡面，我才能好好活得像自己。蘇慧倫透過書寫〈與生俱來的黑〉的Lyrics，剖開靈魂，一層一層探索內心最深處，她不去刻意掩飾與生俱來的黑暗面，她知道人在脆弱的時候，就像困在深海裡般孤立無援，她知道每個人都有見不著光的時刻，她寫的是自己的情緒風暴，她寫的也是你和我的無助吶喊。

09. 量力而為 Between us 詞 / 曲：艾怡良
人的恐慌面
「多虧犯錯之前 / 我們不只量力而為」
漆黑的夜空，綻放了燦爛煙火，多想緊抱住一束煙火，讓它永遠點亮視線，但煙火總是短暫，於是我們拼命追、貪心追，也恐慌地追那稍縱即逝的光源，我們不只量力而為，甚至帶點瘋癲的犯錯，即使永恆只存活一眨眼的時間，也足以撫慰靈魂的惶恐。當蘇慧倫遇上了艾怡良，〈量力而為〉便散發了獨一無二的氣味，艾怡良起初創作概念源自想嫁卻又恐婚的獨白，而在蘇慧倫沉澱思緒、消化Lyrics意境後，她唱出了別緻的纖細情緒，〈量力而為〉也成了專輯中最為靜謐深沉的一首歌。

10. 你會想念我嗎 Will you miss me 詞：葛大為 曲：黃建為
人的坦然面
「當生命自有它的節奏 / 你學會輕盈也要了解沉重」
謝謝親吻過我生命的每個你，過往的回憶仍像昨日美夢般生動，我們都犯過執迷的錯、流過
