　　2005年，So What 有備而來，即將以他們大無限的舞台魅力和音樂熱浪。

　　2005年「123一起跳」 So What 踏上開始的旅程。

　　一如他們眉宇之間透露的年輕驕傲、狂妄的勇氣；

　　一如他們勇敢堅強對自己所愛的事物展開追求，

　　一如他們這個樂團的名字 So What，率性而為。

　　「123 一起跳」顯現一個不做作、自然、直接的 So What。三首歌，除了鋼琴伴奏「 狂舞吧！！勇氣 (浪漫版)」讓人感受到一絲憂鬱氣質外，其餘都以快樂進行。就連示愛的歌曲都很激昂。太快樂別忌妒、太感傷說不一定下一刻的Lyrics和旋律又讓你恢復好心情，總有千奇百怪的方式要你怒中帶笑，喜極而泣，字裡行間透露這個樂團率真的個性，爽朗的旋律讓你不得不記住。

　　「123 一起跳」 三首歌挑戰你奔放的極限、愛的能力、給你迎向未來的勇氣！

　　「123 一起跳」 不得不穿戴在身上的夏日旋律！

　　「123 一起跳」 緊貼著 So What 才聽得見青春在你心臟裡跳動的頻率。

⊙發片創奇蹟 電視零曝光 首周銷售排行衝上第15名

　　So What 樂團首張專輯由華研唱片代理發行，6月4號推出之後，發片首周就創下了奇蹟。因為沒有預算作宣傳，因此電視零曝光；因為要節省成本，所以根本沒送出公關片給媒體；電視沒打歌電台沒播歌平面沒報導，但卻一口氣衝上銷售排行榜第15名，首批 3000 張幾近銷售一空，簡直讓唱片同業跌破眼鏡。榜單一公佈，唱片同業與許多媒體朋友紛紛來打聽 So What 到底是誰？

　　2004 年的酷夏，台灣搖滾充滿聲嘶力竭的怒吼，是飛的、還是跳的、還是猛的，百花齊放到處都是；2004 年同時間 So What 獨立自製首張專輯《有人在外面嗎？》其悅耳不喧囂的音樂在眾多樂團中清新脫俗。

　　2004 年，當初辛勤賣力的製作只為留下玩團的美好回憶，卻不經意的走上一條搖滾不歸路。而為了讓人見識 So What 的騎士精神，這群小子在當時幾乎踏遍全台北大街小巷、地下街道。甚至由北到南、中部、跨越中央山脈遠到花蓮，只要可以到達的地方，只要願意分享他們的音樂，So What 就必定抵達。而那股就是要衝的任性和勇氣終在發片第一周登上 G-Music 排行榜。

　　別企圖找出一個脈絡尋找及窺探這個樂團的曲風，《有人在外面嗎？》說他是冷風過境下的英倫迷幻、是狂野奔跑、不輕不重的搖滾也好、清新可人的 Pop Rock 也可，《有人在外面嗎？》那時的 So What 只想一股腦給你滿滿的 So What，想說的太多、就怕你不了何謂 So What。

　　《有人在外面嗎？》開宗明義宣示 So What，每一首歌像是教條、宣言、註解，每一首歌都代表 So What 的精神、企圖與理想，自信與眉宇帶著殺氣、帶著憤怒，傲氣徹徹底底的完全彰顯。

　　2004 年 So What 首張專輯《有人在外面嗎？》是一個美麗的意外、也是旅程的開始。

⊙SoWhat 的真心話：讓我們繼續 我最後的一擊

　　而 愛 讓我們最後讓他一起蒸發 光 光 光

　　首先 So What 要在這邊感謝所有支持我們的朋友！讓 So What 獨立製作的首張專輯《有人在外面嗎？》從去年發行到現在所生產的五千張專輯即將銷售一空！！很開心有這麼多人在外面，這麼多人肯聆聽我們的聲音，這對獨立發行來說已經是相當好的成績了，謝謝你們給了 So What 繼續下去了力量！

　　然而得到了你們給予的力量之後，我們又把所有爭到的版稅，繼續做著我們的搖滾夢，現在 So What 已經製作完第二張專輯，聚集所有能量，蓄勢待發！

　　在這之前 So What 為了不讓支持的朋友等太久，在今年的夏天，我們先推出了青春狂夏三部曲 EP：《123 一起跳》為第二張專輯作暖身。

⊙每年夏天 我都會去海邊

　　每年夏天福隆的海洋音樂祭，總是吸引了大批人潮一起去朝拜聖音。2005 年 So What 再度入圍國際海洋音樂際大賞，期待在沙灘上的大舞台，把他們的搖滾能量全部釋放，在眾多歌迷與其他樂團的矚目下，希望 So What 能在這次海洋發光發熱！

　　這世界真是美好，讓我們同時擁有音樂和電影。

　　今年的海洋音樂祭擴大舉辦，除了變成一個長達五天的音樂祭，另一個焦點就是，這次除了音樂還結合了影像，很榮幸 So What EP 單曲「夢想倒數的時間」受到第一屆海洋影展的青睞，成為 7-11 貢寮國際海洋音樂祭海洋國際影展主題曲。

　　如果你曾經看過 7-11 收銀機上播放的短片，你就一定聽過這首歌：「夢想倒數的時間」，收錄在 So What 青春狂夏三部曲 EP：《123 一起跳》

　　然而再這長達五天的海洋音樂祭，So What 除了在大舞台上演出，更是拼了命，攜家帶眷五天都待在福隆的海邊，每天都在沙灘上，舉行簽唱會。

　　所以千萬不能錯過今年的海洋音樂祭，更不能錯過今年夏天最受矚目的 EP！

　　還有史上最賣力的搖滾樂團 So What！

⊙歌曲簡介

　　青春的奮力一躍從他們身上開始起跳！

　　青春大無畏的自由與快樂在他們身上，一一彰顯。

　　青春夢想勵志歌，百分之百給你信心的鼓勵。

　　撇開那些擾人的情感贅述、撇開那些批判以及忿恨不平，至於大力蠻幹或狠勁的搖滾基底都暫時拋開，一切就以輕鬆、平易近人，坐著、躺著，青春狂夏三部曲，從「夢想倒數的時間」，「而我們繼續」，「狂舞吧！！勇氣 (浪漫版)」每一首歌一一為您揭示青春的本質。

01 夢想倒數的時間 (7-11 貢寮國際海洋音樂祭海洋國際影展主題曲)

　　勇氣還剩多少？而你什麼時候會走掉？結局又會是如何？這一切都不再重要。

　　So What 用進行曲，像是軍隊鼓舞士氣，把勇氣握在手上，讓我們跳過夢想倒數的時間，讓我們緊緊擁抱，唱出一個最快樂的倒數計時。欲靠近0點0分0秒，就欲靠近夢想的開始，讓我們 123 一起跳，我們要幸福、我們要驕傲。

02 而我們繼續

　　一反對愛情的溫柔公式，So What 用暢快到幾乎跳躍的方式呈現等待的愛的到來！不浪漫也是浪漫的一種，至於怎麼一個不浪漫法，So What 居然用棒球比擬對愛情的告白。

　　從一開始熱鬧登場的喇叭聲，遇見廣大的球場，上面正沸騰的歡呼加油聲，和場內投手做出準備投球姿勢…故事以棒球選手作為主軸，「到兩好三壞才發現我愛你」、「九局下兩出局而我們繼續」、「故事的結局近期上映--不再是秘密」Lyrics上演一段「心靈投手」的橋段，男女主角從秘密的開始、演變成地下戀人到最後成為大家祝福的戀人。

　　除了Lyrics意境營造球場的歡樂氣氛，So What 還特別邀請曾與台北愛樂合作的小號手「鄭北京」客串演出，那不斷出現的喇叭聲簡直畫龍點睛，彷彿看見全壘打紅不讓的畫面。據說為了讓他迅速進入音樂情境，團長麥基搬出 PS2 的棒球遊戲仔細解說其中的遊戲規則、和描述電玩配樂的刺激場景氣氛。到底是古典音樂底子的鄭北京老師，從頭到尾一次 OK。

03 狂舞吧！！勇氣 (浪漫版)

　　狂舞吧！！勇氣 (浪漫版)，這將會是 So What 的經典抒情。曾經收錄在《有人在外面嗎？》的「狂舞吧！！勇氣」，其版本較為澎湃激昂。

　　寫歌的人叫做麥基，當初寫下這首歌是在他和媽媽通完電話之後，一把眼淚一把鼻涕，字字寫下。玩團都需要家人支持和鼓勵，從這首歌我們除了聽見這群小子出征前對自己的宣言之外，更重要的是希望支持他們的家人、親朋好友，希望他們一直都在，希望他們永遠不要離開。

　　至於 EP 這首「狂舞吧！！勇氣 (浪漫版)」的版本極為輕柔緩慢，整首歌只有一種樂器，鋼琴。而為什麼只有鋼琴？麥基的回答：「就想像和我媽在跳舞。」

　　據說這首歌在未來會以各種不同的面貌出現在 So What 每一張專輯。假若你錯過《有人在外面嗎？》的「狂舞吧！！勇氣」激昂版，那就一定要聽聽浪漫版，保證哭到哽咽！
