咖啡可以待用，那 愛情呢？
陌生的溫暖我們感激 熟悉的胡夏讓感動越沉越香
關於”待用愛情”全新大碟
近期十分受到矚目的暖心網路話題”待用咖啡 Suspended Coffees ”，起源於義大利拿波里的咖啡店消費者，在購買自己喝的咖啡外，還會貼心的多付了幾杯咖啡錢，目的是讓在外居無定所或是衣食匱乏的人，可以進入店內得到前面的好心人預留的免費咖啡，而這樣的行徑在得到宣揚之後，立刻獲得全球許多店家的跟進，而這樣的溫暖且善良的舉動與巧思，在胡夏的{傻瓜探戈}專輯裡也希望可以給予每個期待愛情，渴望好的事情的人，可以藉由胡夏在這張專輯裡這份聽得到愛、人生、正向思考的”待用”理念，他真切誠懇、滿懷希望、無私微笑、清泉般透徹的情感，讓我們可以輕而易舉、唾手可得的領取 ，並一起持續推廣這個良善之舉...

傻瓜探戈 (讀音：傻瓜tango)
這是胡夏正式完全自我介紹的概念大碟【傻瓜探戈】
第 3 張專輯，第 3 年個人舞台，
多了些許歷練，多了更多獨白，
「以前，我唱的是別人的愛情﹔
這次，我想試著唱唱關於自己...
無論過去或未來...亦有關於生活自述或是愛情那件小事」
愛情對於胡夏，狀態有點經驗不足，抽象得像電影一樣，
天真的腦海中想像那種關於愛的喜、樂、悲、歡，
真摯的唱出快樂、溫暖、悲愴甚至是遺憾...

每當胡夏拿到了一首新的Demo，便會開始嘗試想像那個畫面...
愛情片裡的優雅溫柔、特效片裡的壯闊奇想、時代片裡的動蕩不安、懸疑片裡的曖昧不明、抑或是 歌舞片裡的華麗爛漫......翩翩登場，這一回 胡夏展現他的奇想！

浪漫抒情的音樂舞台在2013重生之年更加的炫彩奪目...
爭奇鬥艷、繽紛精彩的國標舞世界裡，男男女女舞動著愛情裡暗藏曖昧心機的tango...
這是一種情人之間的祕密舞蹈，胡夏選擇了這一種頓錯鮮明的舞步，是因為tango的情緒如他，認真且直率有力的態度...異想一位期待愛的女孩，還有一位離愛情很近的男孩，
他像個遲鈍的傻子，只懂得 快 快 慢...的緩步進退，
這步伐讓女孩著急且更加的心跳加速，
她試圖掌握著節拍，一點點熱情、一點點主動...
他努力學著那些旋轉、拉手、抬腰，生澀且艱難...
在這個懵懂年華，彷彿只懂要用功、專心跳著tango，愛情，就像這支舞一樣，一步步跳躍而來... 然而，總是在一個失神的無意鬆手，愛情終究像手中的細沙流失殆盡，
最後，我們只能看到spotlight下，那個獨自迴旋、繼續練習的孤獨舞者，

經驗不足也許是個敗筆，
但，誰不是這樣茁壯而堅強...

30年代經典再現 全新復刻流行風格抒情新專輯 {傻瓜探戈}
在這首極具復古翻新風格的[傻瓜探戈]主打歌裡，抒情基底的曲風樣貌，卻帶有極大企圖心融合亞洲音樂市場新興的玩轉復古編曲創意調味，仿效30 ～ 40年代，最美好的年代也是一切經典的開端，電影、音樂等娛樂產業蓬勃發展，黑白電影漸變邁向炫目奪眶的彩色世界。
在當時... 每個人處於對未來充滿期待與面臨改革新意的蛻變階段，
「 期待改變，做更多自己想做的事，唱自己想唱的歌，是我現在最想做的事情... 」
胡夏帶著他倔強又堅定的口吻說著，卻也知道有些努力不是技巧就可以勝任，所以他帶著他最自信的招牌清泉嗓音，開始為自己祈求的愛情勇敢發聲，溫潤且情緒飽滿的嗓音，很真實也很柔情的娓娓唱出，屬於自己渴求那份溫暖的情感卻技巧明顯不足，落得原地獨舞那種遺憾感受，[傻瓜探戈]描繪真切，探戈並不傻... 是愛情 太傻。

耗資20萬 打造多組30年代復古經典造型裝扮
書香男孩 變身 音樂紳士 時尚流行上身
造型上，在過往兩張專輯極簡的書卷氣息風格之後，胡夏也不斷成長著，並開始越來越見其輕熟的溫文氣息，彷彿緬懷過往的古典優雅風情，從衣著上經由亞洲天王天后合作班底 王威翔，造型團隊赴香港採買、於當地定裝，耗資台幣20萬打造多組30年代經典時尚，復古翻新的新風格，「復古」一直是流行時尚不敗的流行趨勢，近期的 20 至 80 年代依序輪番上陣，從歐洲皇室英國的維多利亞女王、愛德華時期，到法國的拿破崙，甚至中古的神秘哥德，都是靈感泉源 ：各式各色的領結、西裝背心、硬挺西服、紳士帽、吊帶boy look、窄版九分褲...，從{傻瓜探戈}專輯音樂本質出發，打造而成令人玩味的「復古」美感與本質融合裝扮 ，這一次除了聽得到音樂的改變外，胡夏吾家有子初長成的時尚輕熟感，此次好感度更加分，令人刮目相看…
最美好的年代，最能創造出最經典的饗宴...
封面主視覺的拍攝企劃，特別邀請到時尚雜誌御用攝影師- Tommy陳世杰 (Men’s Uno、 ELLE、 Harper’s BAZAAR)，為胡夏掌鏡拍攝2013年全新個人專輯的嶄新主視覺封面，同時把場景拉到台北近期新開幕的典雅紅磚屋改造餐廳，並大量使用木造桌椅、老舊厚重書籍、舊式機械打字機、銀質檯燈以及極具時代氣息的圓形收音麥克風當作拍攝美術道具，穿著深色合身西裝look、白襯衫、真皮皮鞋的小紳士胡夏或坐或站其中，活脫展現他的深具風格新專輯的曲風樣貌，
沒錯！
這是一場傳達著充滿傳統、復古、民謠、爵士...等賞心悅目聽覺與視覺饗宴的玩轉新經典。

寫真詞本內翻開，更可見前所未現胡夏表現吃喝的輕鬆愜意生活感，看他一口咬著香噴噴的可頌麵包、一首喝著透明玻璃罐裡的牛奶，不小心還會沾在嘴唇上而不自覺，讓周遭工作人員直呼”真的太萌了吧... 連吃東西都這麼惹人疼愛！”

1 \ 靜電 詞：葛大為 / 曲：鄭宇界
緩慢搖擺的節奏，緩步進入專輯開場，音樂紳士的渾厚嗓音以及深情口吻，摩擦出模糊曖昧、若有似無的復古節奏，越接近尾聲越是大鳴大放，聽得出專輯企圖巧思的開門見山。

2 \ 改變 - 偶像劇 求愛365 片尾曲 - 詞：嚴云農 / 曲：陳忠義
愛的念頭 會讓你煥然一新，
演藝圈的喜好是善變 所以胡夏決定有所【 改變 】，
還沒有好好談過戀愛的他，在突破層層關卡、離鄉背井的適應與挑戰種種難關，
發現，以不變應萬變，有些狀態卻默默的在改變...不變的是，那份單純、清新、無心機... 真摯且充滿溫度感的聲線，開始期待自己，應該多一點男人該有那種一肩扛起的臂膀，
而眼神不自覺多了一點渴望愛的撲朔，熟悉依舊且期待度更高的全新抒情作品。
改變需要時間 再等一下 你會發現一切都對了....

3 \ 傻瓜探戈 - 詞：鄔裕康 / 曲：饒善強
探戈不傻，是愛太傻...一個迴旋轉身，只見他獨舞在spotlight下。首波話題主打胡夏+宋紀妍 豪華共舞，挑戰”新嘻哈探戈”全民矚目復古新經典強勢攻佔媒體版面。

4 \ 我們會不會 詞：林夕 / 曲：薛忠銘
青春期的詩反覆在腦中盤旋，喃喃自語著對於成長的庸人自擾，忙碌生活裡戀上一那個人後，察覺起...我們會不會不夠堅強?根本不敢勇敢?天亮了，等著迎接下一個多愁善感...詞神 林夕為胡夏異想創作那些關於愛情惆悵的日子，成為美麗的詩歌< 我們會不會 >。

5 \ 微笑的人 詞：陳珊妮 / 曲：陳珊妮/徐千秀
只懂得微笑的人，也許只能夠坐在個人座上，等待下一位願意微笑回應的她靠近...
嚴師音樂人 陳珊妮風格獨具的抒情搖滾曲風，這是愛情心跳的重拍結合胡夏真摯情感的多重音域層次，展現鮮明個性的精彩演繹。

6 \ 學著走 - 偶像劇 求愛365 - 插曲 - 詞：十方 / 曲：譚志華
面對眼前現實與叛離構圖成的失落世界，
搭配呼之欲出的絃樂緊跟著鋼琴的步調開場，像是一部slow motion的特效電影，
「回憶像荒蕪的樂園 / 歡笑聲逃出從前 / 海盜船廢 / 摩天輪斜 / 木馬不旋... 」
【 學著走】是胡夏對於成長的經歷體悟，教師世家的熏陶讓面對這個世界的真實，有著充滿意境的形容與描繪，不是刻意逃避，而是因為視野不同，胡夏比誰都更懂得... 疼痛的時候 要像孩童般 學著走...

7 \ 離家以後 – 詞：丁曉雯 / 曲：鴉片丹
唱歌對於胡夏來說，是一種天賦，卻也沒像喝水那麼輕鬆簡單...
站上舞台唱歌給大家聽，讓他迷惘自己的自信其實沒這麼高，會害羞也會膽怯，尤其是在面對人群，不管一個人還是上萬個觀眾...
這首【 離家以後 】 是胡夏對於離開老家出外4年累積，體悟未來原來不是一股腦認真奮鬥，就會如己想像所願，經歷這個世界黑暗也光明的矛盾， 會是男孩蛻變的的過程......
木吉他沈沈的伴奏，彷彿輕鬆的步伐，也夾帶著踏實走下去的決心。

「 如今我 還跌跌撞撞 / 想起 百里外的家 / 我的天涯 是什麼模樣 /
一個能給我溫暖給我愛的地方 / 還有更長的路要走 / 還要多一點的成熟 /
在我離家以後 / 找到生命的彩虹 」

離家以後的日子，胡夏還是忠於自己那份認真、直接、開朗的真摯性情，用歌聲傳達溫暖與愛到每個天涯海角...直達到蝦米們心底。

8 \ 愛情離我一公尺-韓劇 浪漫滿屋2 片頭曲- 詞：姚若龍 / 曲：陳小霞
從來沒有如此赤裸真實的告白自己的感情觀，胡夏：「愛情曾經離我一公尺，但現在應該有十公尺、一百公尺了吧... 」 充滿空氣感的吉他和絃，營造出貼近愛情時那份神祕未知、充滿曖昧的錯綜心情。

9 \ 黑框眼鏡 詞：吳易緯 曲：蕭恆嘉
這是胡夏的自我介紹，從沒脫下的黑框眼鏡，只為可以完整紀錄與歌迷們回憶的點點滴滴，那些一路以來的掌聲與喝彩，都是想要好好珍藏且歷歷在目的鏡框中回憶。

10 \ 一定有個妳 詞：葛大為 / 曲：金大洲
滿場絃樂在開頭一併奏起，偶像劇般的華麗開場抒情曲，胡夏低吟的唱出幻想愛情彷彿少年維特的煩惱般，祈禱那個她降臨，倔強等待卻也擔心受挫... 像是經歷一場難度頗高的愛情般：「這首歌是我錄音期配唱最久，也是音域最廣的一首歌，讓我回想起歌唱比賽時挑戰高難度的當下... 」胡夏說，而因此也成為專輯最後一首曲目，別具巧思。


胡夏在選秀節目以及自己私底下也常常唱著這些老師們的音樂作品，
對他來說，當初可以唱這些參賽歌曲他就已經很滿足、很開心，
只是沒想到當初純粹可以站上舞台，唱唱這些神人等級的音樂作品，
如今還可以與老師們合作，唱出屬於自己的音樂創作，自己心中已經是默默獲得極大的成就感!

這個世代該有的經典音樂、該被留下來的絕佳嗓音，不過度修飾與包裝，索尼音樂一直抱持著極度珍惜與力捧珍藏，很開心，這樣純粹飽滿且溫暖的聲音，是該被呵護保存並雅俗共賞的！
