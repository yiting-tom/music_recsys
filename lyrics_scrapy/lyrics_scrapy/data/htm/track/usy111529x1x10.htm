國王的新衣(安徒生童話)

Lyricist：柴可夫斯基 - 胡桃鉗 ( 進行曲 )
Composer：柴可夫斯基 - 胡桃鉗 ( 進行曲 )
Arranger：何俊明

從前有一個很喜歡穿新衣服的國王，他一天到晚都在要求皇宮裡的裁縫師替他製作各式各樣的新衣裳。每個月，國王都花費大量的時間和金錢在他的衣服上，不管大臣們怎麼勸說都沒有用。很快的，國王就發現原本皇宮裡的那些裁縫師已經設計不出新花樣的衣服了，於是就下了一道命令，只要誰能製作出與眾不同的衣服，國王就要給他一大筆的獎賞。

於是，皇宮外每天都有來自全國的裁縫師排著長長的隊伍，要向國王展示自己所製作的衣服。可是國王卻一件也不滿意。一直到有一天，有兩個騙子來到了皇宮外，他們宣稱可以用魔法製作出最特別的衣服來獻給國王。國王一聽高興極了，於是就把他們留在皇宮裡替國王製作衣服。可是這兩個人每天在皇宮裡閒晃，完全都沒有在製作衣服，只有當國王的使者來詢問衣服的進度時，才會碰一碰紡紗機，假裝十分忙碌的樣子。

終於有一天，國王忍不住了，親自來看看他的「新衣服」。這兩個騙子立刻跪在地上，雙手捧的高高的，於是對國王說：「親愛的國王，您的衣服已經完成了。這是一件神奇的衣服，只有有智慧的人才能看得見！」可是明明裁縫師的手上根本沒有任何東西，國王卻對裁縫師說：「唉呀，這是我所見過最美麗的衣服了，來人啊趕快替我換上，我要穿著它去遊行。」在場的人沒有人敢出聲音，並且假裝看得到這件衣服，紛紛讚美國王穿上衣服後多麼的好看。因為大家都害怕被別人認為是沒有智慧的人。


於是，國王就這樣光著身體去遊行了，當全國人民看到國王的時候，大家都嚇了一跳，可是沒有一個人敢講話。最後，當國王經過人群的時候，有一個小男孩拉著媽媽的裙子問：「媽嗎，為什麼國王沒有穿衣服呀？」一旁的路人聽到了小男孩的話都紛紛笑成了一團，「哈哈哈，國王沒有穿衣服！」國王害羞地脹紅了臉，匆匆忙忙的回到皇宮裡去。而那兩個騙子早就拿著國王的賞金，大搖大擺的離開了王國了。

這一位愛漂亮的國王就是花了許多時間和金錢在挑選和製作新衣服上，才會讓兩個騙子有機會來愚弄國王。而大臣也因為擔心自己被別人嘲笑不夠聰明，失去了判斷是非的能力。小朋友，我們對自己要有自信，不要因為人云亦云而失去了真正的價值判斷喔。
