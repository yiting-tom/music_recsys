2008 - 2012的陳零九
「只是有些話」想對我們說
「總是有些話」用唱的才能出口

20首心情創作
感動溫柔褓抱你


夢想是需要勇敢去灌溉的，感情是需要感動來滋潤的，讓陳零九用親情、愛情、友情的生活體驗，深情傾訴他內心的世界 ; 總是有那麼一首他的歌，能夠命中紅心，使人揪心。

「不自覺」地愛上一個人，卻終究屈服於現實的拆離 ;
安慰朋友們有沒有孤單都無所謂的「正負寂寞」 ;
「乎阿嬤的批」道盡了一個人離鄉背井在台北打拼的思親 ;
充滿愛與感動，「只是有些話」想對你說 ;
五味雜陳的遠距離戀愛，不管如何「我只是害怕不在你身旁」;
每個人心中的自己是什麼模樣？「我」這首歌送給我自己 ;
無法呼吸，「我很想你」是一種窒息式的思念，讓人停止呼吸地聆聽 ;
世界上最難堪的無奈就是在「情人結」前夕分手 ;

或許，你曾在某處聽過陳零九的某些歌，然後訝異地驚嘆Lyrics是如此刻入心底， 旋律與文字代替你說出「那些話」，毫無保留，正當我們為了尋找一首讓自己哭泣的歌，卻常遍尋不著時，就聽聽陳零九的歌吧！
或許，你並不認識「已讀不回」之前的陳零九，總以為他就如同那些突然竄紅的網路歌手一樣，乍現，然後隕落消聲匿跡，事實上，他的能量已累積多年，就等待那個專屬時刻，讓你聽見不一樣的聲音。

現在，給自己一個機會，在音樂中重新認識陳零九 ; 他，有許多話想要跟我們分享，也想聽你說，屬於你的那些話。
