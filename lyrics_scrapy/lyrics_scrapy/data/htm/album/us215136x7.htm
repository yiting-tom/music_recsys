Album Intro：

風華爵代！ 一展爵士調配嘻哈的迷人風味！
中文說唱指標人物路壹Lu1首張現場大碟「 孤島唱遊 / Nomad Studio Session: Live at Blue Note Beijing 」優雅登場！
澳洲爵士先鋒Toby Mak 攜手TTechmak 國際級樂手豪華助陣！


不容錯過的現場錄音唱片
2017年，路壹Lu1攜手TTechmak爵士樂團 以及DJ LJ接連登陸上海北京兩地爵士名所，為一眾樂迷帶來了名為「 ‬孤島唱遊 / ‬Nomad Studio Session ‬」的精彩演出，兩場演出門票在Lu1超強號召力下迅速售罄，更有媒體人稱讚其為“年度最高級的演出，沒有之一！”。‬‬‬‬‬‬
十余首全新編曲的作品在現場的極佳地呈現了嘻哈的律動與爵士的自由，讓現場歌迷讚不絕口。
12月28日在爵士名所Blue Note Beijing的精彩更是進行了現場錄音，並製作成了這張有著迷人風味的現場專輯。



The Nomad Studio Session的概念由來
約翰•多恩說：“No man is an island,沒有誰是一座孤島。”
而在這個強調自由、自主、自我的時代，似乎我們每個人都是一座孤島。
我們披星戴月地漂泊在擁擠的大街上，在標籤裡尋找方向，各自前往下一個目的地；
就像茫茫宇宙裡面一顆顆有著自己獨自軌道的星星。
我們從一段關係流浪到另一段關係；
每一次相遇，都可能是我們在同一時空裡的唯一的交集。

Lu1為以上這番思考找到了一個關鍵字：Nomad ，意為遊牧。
這個單詞也被定為他限定合作巡演的概念主題。
一直以來，Lu1都想和爵士樂隊合作一場小型現場演出，邀請朋友來聽歌，喝酒，聊天。但一場好演出的意義一定遠不止於此，能把自己內心的所感所想傳遞到台下每位聽眾的心裡，引發更多的思考，才是真正令表演者開心的事。
所以，「 孤島唱遊 / Nomad Studio Session 」是Lu1借由爵士樂的自由和嘻哈的自我表達，以獻給孤島們的一場一期一會的表演。
