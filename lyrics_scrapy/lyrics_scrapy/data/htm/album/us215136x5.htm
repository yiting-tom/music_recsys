Album Intro：

都市背包客 ( Live at Blue Note Beijing )

風華爵代！場場爆滿！
爵士調配嘻哈的迷人風味！

中文說唱與爵士樂融合的指標性演出，路壹Lu1領銜之「 孤島唱遊 / Nomad Studio Session 」北京 Blue Note 場現場錄音即將登場！
先行單曲《都市背包客》，展現迷人都會風貌。

2017年，路壹Lu1攜手 TTechmak 爵士樂團 以及 DJ LJ 接連登陸上海北京兩地爵士名所，為一眾樂迷帶來了名為「‭ ‬孤島唱遊‭ / ‬Nomad Studio Session‭ ‬」的精彩演出，兩場演出門票在路壹超強號召力下迅速售罄，更有媒體人稱讚其為“年度最高級的演出，沒有之一！”。‬‬‬‬‬‬

12月28日在爵士名所 Blue Note Beijing 的精彩更是進行了現場錄音，即將製作成現場專輯發行。
先行單曲《都市背包客》，最早收錄於Lu1與Cee的合作專輯《午夜列車上的告別》，雖然發行初未被列為主打單曲，卻意外受到眾多樂迷追捧，此番現場獨唱版在一眾樂手的合作下展現出更加迷人的都會風貌。
