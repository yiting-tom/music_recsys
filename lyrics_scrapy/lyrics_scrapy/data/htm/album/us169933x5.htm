趙泳鑫新專輯《躺著》世界等你用新姿勢迎接！更特性更多樣性更成熟曲風！

世界等你用新姿勢迎接，趙泳鑫選擇“躺著”面對！繼上張專輯大獲好評後，歷時兩年，趙泳鑫潛心創作，詞曲唱親力親為，編曲製作一手把控，以更具特色更成熟的音樂風格，Funk、R&B、Future Bass、Jazz Pop、House舞曲等信手拈來，打造了這張2018個人全新專輯《躺著》，10首音樂作品，再次將自己的音樂才華與偶像氣質展露無遺。

新專輯《躺著》不僅有重新詮釋2018都市男女最佳情感關係的《稀客》；還有藏著他自己的私密故事，如《Lili》、《自首》等；也有他看到的這個世界正在發生著的現象，如《躺著》、《杜英才》等；更有自我創造的多元空間，如《就算你還在》等。可謂是內外兼修的一張屬於他個人的代表作。由於全部親力而為，趙泳鑫更巧妙地在歌和歌之間用Lyrics和概念埋下各種彩蛋，細聽會發現，每首歌其實互為因果互有連結。

要麼喪，要麼燥！新專輯表態年輕人享受生活熱潮，反諷現代都市“躺現象”！

玩轉生活，要麼喪，要麼燥，該宣洩就宣洩！
隨著現代都市生活節奏越來越快，人們壓力越來越大，有人甚至不願拼搏，被喪文化主導著自我，於是乾脆在哪裡跌倒就在哪裡躺下。針對大多年輕人都開始喜歡躺著這一現象，大家手機不離手、吃啥都沒夠、有座絕不站、能躺絕不坐的日常，趙泳鑫通過反諷口吻的音樂作品，向這個浮躁的社會傳達著自己獨特的新（心）態度，好好學習，躺著向上！

生活也許殘酷、無情，但生活也是一面鏡子，世界看你用什麼樣的姿態來迎接它，你若喪，世界比你更喪，你若微笑，世界開滿鮮花。沒有不勞而獲的機會，沒有不努力就能實現的夢想。天從來不妒英才，而是英才妒英才，更多時候要相信自己天生我才必有用，總有人會發現那不可方物的才華和價值。我們可以任性的躺著、可以忙裡偷閒、可以去相信生命在於靜止、可以劍走偏鋒、可以特立獨行，但是主導我們命運的是我們對生活的心態，所以說心有多大，舞臺就有多大。趙泳鑫從“躺現象”裡，思考出自己的“躺文化”，用年輕人喜歡的音樂風格，與這個時代溝通。

《躺著》就要躺出天翻地覆，趙泳鑫躺著定義2018最佳姿勢！

既然越來越多人喜歡“躺著”，於是趙泳鑫在Lyrics裡呼籲：“來，我們一起躺著！”不跟大家唱反調，而是用音樂雪裡送炭唱高調。諷刺“躺現象”背後對生活喪的一面，而不是拒絕表面的“躺行為”，高調地把自己思考的“躺文化”傳遞出來，給予大家正能量。從之前首張專輯大獲成功後，趙泳鑫一直思考著自己下一張專輯要怎麼做？如果重複宣揚自己音樂的酷炫就沒意思了，所以在這張專輯裡，趙泳鑫從心思考，從新出發，與這個時代接軌，為這個時代發聲。

這一次，歡迎大家躺著聽趙泳鑫。聽音樂是最佳減壓時間，躺著是聽音樂的最佳姿勢。松掉所有緊繃和壓抑的神經，隨時隨地就像躺在最舒服的床上，不去想討厭的人和事，把全身心交給趙泳鑫。“不信你閉上眼，跟我一起躺著，來，來，我們一起躺著”，躺進趙泳鑫的音樂世界裡，夢遊這10首歌的時間。躺著，是2018的最佳姿勢。

2.《稀客》
Lyricist：趙泳鑫 / 作曲：趙泳鑫 / 製作人：趙泳鑫
你是誰的稀客？誰是你的過客？
耳朵一亮的Chill R&B 曲風
2018最佳情感關係《稀客》, 歡迎光臨都市男女的世界！

不做情感的脆弱者，該宣洩就宣洩！打破夜的平靜，掙脫寂寞枷鎖！稀客稀客，歡迎光臨！

趙泳鑫最新創作的《稀客》悅耳來襲！寓意深刻的Lyrics展現出現代都市男女超帶感的 chill R&B節奏，搭配副歌動感輕快旋律，跟著趙泳鑫在每一個夜色迷離的時刻搖擺起來！潮愛起來！

你的生活裡有沒有幾個稀客？每當夜深孤獨寂寞時又非常想念？很多時候正是這種友達以上，戀人未滿的關係令人瘋狂著迷，享受著彼此如同知己又不受對方束縛的自由狀態。在人來人往快節奏的現代都市里，夜色紛雜的霓虹燈讓人容易迷失方向，越來越多的年輕人，渴望能尋找到一個身心的寄託，可以向彼此傾訴，盡情宣洩出孤獨帶來的落寞。這首《稀客》直擊人們內心敏感地帶，希望大家不做情感的脆弱者，該宣洩就宣洩！打破生活平靜，掙脫寂寞枷鎖！

歌曲令人耳前一亮的是前奏結合了一些chill R&B的元素，令整首歌聽下來非常悅耳，特別是進入到副歌的時候，動聽的旋律一聽就會，第二遍就能跟著哼唱起來。

歌曲可以理解為一種孤獨中的慰寄，是一種平靜後的宣洩，趙泳鑫說：“很多人都是我們人生當中的過客，可相遇，可偶遇，可擦肩，可錯過，但唯有稀客，即便不願承認，也一直存在於我們生活當中。”不要讓稀客變成過客。

走過的時間旅行、散場的寂寞表情、錯過的美麗風景等等，都是生活中每個人都會經歷過的，無法拒絕就學會接受，不用遺憾，在一起時就盡情享受。每個人都有幾個不常見，但會偶爾思念的稀客。

3.《自首》
Lyricist：趙泳鑫 / 作曲：趙泳鑫 / 製作人：趙泳鑫

全唱作人趙泳鑫情歌曲風再度昇華，抒情之餘亦氣勢磅礴
光明與黑暗只在一線之隔，趙泳鑫“自首”人性的多面性

《自首》是全唱作人趙泳鑫的最新情歌之作，但區別於以往的是在這首歌裡，趙泳鑫加入了更多哲學式的思考，對人性、對生活。就像整張專輯《躺著》一樣，可以明顯看到趙泳鑫的成長，除了依然新潮酷炫外，音樂也越來越有深度，獨具個人特色。

歌曲開始以鋼琴柱式和絃進入，充滿了訴說感，加上大編制弦樂的烘托，也有一種懸疑劇的氛圍，與Lyrics所表達的內容和展現出的氣質極其融合。演唱方面在抒情之餘也有氣勢磅礴之處，是內心訴說和情緒呐喊的標配，這是趙泳鑫一整年的所有感悟，Lyrics中藏了一些只有他自己知道的故事。

整首歌像是在對人性的探討，人的固執、貪婪、奸詐、無恥、自私等多重陰暗處十面埋伏在我們生活中，有誰願意或者說是放心向他人坦白出來？坦白就是自首，在如同動物世界的現實中，這不是一般食肉者的風格。既然避免不了，那就應該學會自我保護，同時也要逐漸去適應環境。就像趙泳鑫在Lyrics裡唱的：“學會自我保護也適當的接受/這精妙在裡頭/指揮所有都並行在自己的左右/能懂就不怕食肉者目標永遠選個沒夠”如果這首歌從愛情的角度來理解，其實也無不可，個中滋味留給聽者自行領悟。

4.《不可方物》
Lyricist：趙泳鑫、張敬豪 / 作曲：趙泳鑫

這是一首典型的Future R&B作品，也是趙泳鑫和編曲周禹成的又一次新鮮嘗試。趙泳鑫在一個朋友提到“孤獨讓人思考”後進行的思考。進而產生的一首作品。正如Lyrics中所描繪的，趙泳鑫作為一個夢想主義者，不會因現實而妥協，也不會被暫時的困難所擊倒，但又要適應和習慣很多世事無常，但過後又不計後果地朝著自己嚮往的方向狂奔的內心過程。

《不可方物》是不想裝不可方物，有些事難辨真偽，有些事難免針對，不想跟風站錯隊，不想內心孤獨表面裝無所謂，不想活得太虛假，不想班門弄斧，就不用裝不可方物，照樣可以活得生龍活虎。真實的享受人生，真實的面對孤獨，做自己想做的事，從習慣中變得自然，感受樂趣。
5.《杜英才》
Lyricist：趙泳鑫、張敬豪 / 作曲：趙泳鑫

天從來不妒英才，而是英才妒英才
趙泳鑫唱給每個追夢的人，請相信天生我材必有用

請相信天生我材必有用，天從來不妒英才，因為天降大任於斯人也，必先苦其心志，勞其筋骨，給點時間，總會發光。

歌曲為了體現旋律有記憶點，所以有加強snare drum。全篇添加了少許中國樂器的元素，不追求多而滿，點到為止，正好讓整體氛圍立體起來。

關於歌曲的創作靈感，是來自趙泳鑫和張敬豪在無意中的一段關於“天妒英才”的聊天激發而來的，於是兩人就立刻開始以擬人的方式進行了創作這首歌，最後的成果是令人意想不到的滿意。他倆也笑稱，繼“張有才”（張敬豪）後，又有了“趙英才”（趙泳鑫）的出現。

朗朗上口的旋律，配上輕鬆詼諧的Lyrics風格，令人聽完大感愉悅，就算你心中有不得志的鬱悶也會被一掃而空。其實仔細看Lyrics，其深度也可反復推敲體味。故事中的“杜英才”是一個天賦異稟，但無奈世事不如意，胸懷大志，卻不能得志，充滿悲情色彩的人。想要證明自己卻又無能為力，而在鬱悶至極時，歌曲C段的神來之筆，以孩童的視角，道出了歌曲的內核：在我們的生活中，其實每個人都是“杜英才”。我們只是因為生活的壓力，忽視了身邊的人與事，忘卻了生活的美好。

6.《Mr. Sorry》
Lyricist：趙泳鑫、張敬豪 / 作曲：趙泳鑫

“都請叫我Mr. Sorry/我就是誰都不愛”

《Mr. Sorry》是趙泳鑫和耀樂團的一次深度合作，Lyrics是和耀樂團的張敬豪共同創作的。編曲由耀樂團的馮宣元完成，歌曲講述了一個關於謠言體質的故事，體現出面對生活的自由態度。

快節奏的POP House 流行舞曲的風格，極能體現趙泳鑫性格氣質的一首歌，有趣的Lyrics，唱出了瀟灑的感覺。Mr. Sorry是生活裡每個人都會遇到的，瀟灑自如、不拘小節、不畏閑言的個性卻又令人無比嚮往。我們都會有被誤解的時候，謠言太多，對不起我聽不過來，清者自清，有時候不需要過多的解釋就會真相大白，何必費口舌！

7.《就算你還在》
Lyricist：趙泳鑫 / 作曲：趙泳鑫

如果當初我們在一起會怎樣？很多情歌都這樣唱，失去了才知道後悔。可是有沒有勇氣去面對後悔呢？可能大多數情況是，就算當初我們在一起，估計也不會怎樣！所以，在這首歌裡，趙泳鑫用釋懷的口吻，對過去的遺憾畫上句點：“就算你還在/我們還是一樣/不懂去愛”，勇敢地面對情傷，對曾經的愛說goodbye！

該歌是趙泳鑫作品裡比較少見以弦樂為主體的情歌，也是這張專輯開始寫的第一首歌，靈感來自趙泳鑫工作臺上的一幅畫。他給畫中人設計了故事情節，而編曲採用大編制的弦樂團錄製，並選擇了43拍的節奏，讓歌曲顯得更復古，同時希望營造一種電影配樂的意境，用簡單的Lyrics，唱出畫中人的對白。

8.《邪念》
Lyricist：趙泳鑫 / 作曲：趙泳鑫

愛情裡的《邪念》，都是青春時想要放肆的甜。以回憶的角度唱出曾經愛情的美好，優美的旋律令人共鳴。一首小情歌，仿佛能帶人回到初戀時期，面對一張單純的臉，難道你就沒有過一絲絲“小邪念”？這些念想並非出於惡意，只是每個人內心本能對美好的渴望。回憶也是美好的，而正因為這些美好，才組成了我們不能忘卻的過去，而敢面對過去，才是成長。

9.《Lili》
Lyricist：趙泳鑫、王天放、張敬豪 / 作曲：趙泳鑫、張敬豪

趙泳鑫用自己的乳名力力首次為父母為家鄉而歌，歌曲從深情的回憶訴說，緩緩推進到情感爆發，自然而合理。Lyrics描述了自己的成長經歷和親情感受，這是趙泳鑫第一次在音樂作品中涉及這一部分。為了避免過於主觀和閉塞，他還找來了兩位合作夥伴王天放和張敬豪來共同完成。

歌曲前半部分用鋼琴慢慢敘述，比較偏原聲音樂感的chill out，後半部分採用future bass的風格體現內心的翻湧，以增加節奏感和情緒的遞進。這首歌也是趙泳鑫在現階段最心儀的作品之一。

10.《下場》
Lyricist：趙泳鑫、周禹成 / 作曲：趙泳鑫、周禹成

“我和你的關係永遠都不能說/朋友在的時候交流不能太多”
趙泳鑫式爵士流行，打造新專輯最虐心情歌之作
細膩的人總是在感情裡沒有好下場

不能公開的感情，自始至終都像是在地下戀。這樣的愛可能註定了就沒有什麼好下場。“你給我的安心讓我一直犯錯/現到了這個如今什麼都不能做”既然知道了是這樣的下場，那麼何不主動下場，自我療傷。

歌曲是極具趙泳鑫個人風格的爵士流行作品，爵士樂對趙泳鑫的影響很大，所以在他的作品中不難發現Jazz和Gosple“福音音樂”的影子。而人們印象中的爵士音樂都是以舒緩慵懶為基底的，但這首《下場》恰恰相反，可以聽到樂手們的激情演奏，配合揪心的Lyrics，展現情緒的湧動，自我又深情，是一首不常見的爵士樂和流行樂的結合。趙泳鑫把故事的“下場”和“下場”謝幕都寫在了歌裡，面對感情，是歌頌上場的是英雄，還是惋惜下場時的瞬間，只能各自體會。
