#重新定義紀儀羚 #與你展開新交集的城市唱作女聲

紀儀羚，也許，你聽過她的名字，也許，你聽過她的聲音，也許，你對她零記憶。但從現在起，此刻的紀儀零，將自己重置歸零，再與你重新交集，希望這一次讓你記得她的名字，她的聲音，她寫的故事。

#難以捉摸的吉他全創作才女 #風格獨具的音樂口味 #流行民謠爵士多樣面貌

此次專輯作品皆由紀儀羚全創作，製作人廖晉儀LIAOBOY在聽到DEMO樣帶時，常常驚呼她使用的創作和弦甚至是演唱旋律，不在常人思維之上，難以捉摸卻又展現令人驚喜的獨特性。在製作專輯時，最大的困難就在於保留紀儀羚最原本的創作想法，又需讓作品透過完整編曲表現時，能夠更搶眼。

為了凸顯紀儀羚對作品的獨特味蕾，製作人在每首歌曲裡，都請紀儀羚親自演奏木吉他，在不失本位又能加分的前提之下，將紀儀羚擅長民謠、爵士風格揉入流行、電音元素之中，製作風格獨具的音樂作品。

然而在音樂作品中，更不能忽視的是紀儀羚收放自如的唱腔，時而呢喃，時而高亢輕柔，時而堅決有力，在每十首歌曲都給予多樣的聲音展現，突破一般流行音樂市場習慣且熟悉的演唱技巧，這次紀儀羚將會帶給你更多想像力！

#熱愛生活的觀察家 #將時間轉回記憶零時分

紀儀羚，喜歡稱自己像個熱愛生活的觀察家。水瓶座的她總是有自己看世界的角度跟節奏，有時並不是都能讓別人追得上。所以她寫歌，把自己的，別人的，親身經歷的，只是聽說的…幻化在她的音樂筆記裡，在歌曲裡，她是自己，也不是自己。

最後停留幾分鐘，讓記憶時鐘回到最初的交集時分，
在紀儀羚的歌聲中找尋你我的痕跡。

20:00　Superhero　「在黑暗無人角落裡，才綻放的愛意」
#愛情定義 #秘密超能力 #Savemefromtragedy
丟出直球後的忐忑心情，在內心小劇場裡，偷偷刻畫屬於我們的故事，無止盡地揣測你的回應。當我跌入自己的小小悲劇時，過往的傷心回憶，揉入未來的不安困惑，這一刻，只有你可以成為拯救我的超級英雄/遮風擋雨的光明鬥士/無人知曉的黑暗騎士。只要是你，I’ll be saved.

//

21:00　說愛我　「我的心蹦蹦跳，只為你蹦蹦跳」
#大聲說愛 #心動進行式 #愛情勝過麵包
怎麼證明心動的進行式？你像活在外星的天才，在生活中的不落俗套，喜歡你偶爾任性胡鬧耍怪招，無聊的放空時刻，也因為你變得美好。你相信愛情勝過麵包，堅定地無法撂倒，心跳無法停止脈動，不自主地我也對愛大膽索求，要你大聲說愛我。

//

22:00　不懂　「所謂愛的秘密，放在心底，化作孤寂」
#單戀執迷 #不怕虛實相牴 #偷計算你我的距離
你不懂沒關係，雖然嘴上這麼說著，心裡這份愛你的秘密卻因無處安放而心煩意亂。我們之間的距離，時近時遠，想猜出最合適也能讓你悸動的距離，想計算你心裡面我佔的比例，是否足夠打破界限？哪怕只是自己的幻想風景，哪怕冒著被你看輕的危機，現在的我也不想逃避這樣愛你的心情。

//

23:00　偏安　「你愛過我嗎？你沒有回答」
#愛情敗仗 #偏安完美夢境 #救贖與希望
面對愛情敗仗，是擁抱，是逃避，還是偏安在完美幻覺？沈醉在一段不被認可也不存在的關係裡，心中規劃的種種未來，好像只是自己眼裡的海市蜃樓。選擇不說破，一步一步退讓城池，偏安在自己的愛情角落，等待那份救贖的回答。
「你 愛過我嗎？」

//

00:00　記憶零　「這一刻，就讓記憶返轉為零」
#愛情之後 #打破平靜再相遇 #你我的過去
常常會想著，我們分開後，如果再相遇會是什麼場景？看見彼此的第一眼，會是微笑還是哭泣？而當預期成真時，內心波瀾難以平靜，試圖想回到遇見你之前的場景，將回憶歸零裝作沒有交集，或許，那一天的晚安就不會如此動心。但，想忘記，卻還是無法割捨乾淨。

//

01:00　說晚安　「等你回頭看看昨天，是否和我一樣想念」
#愛情倦怠 #像獨角戲對白 #不再甜蜜地說晚安
往昨天的方向走近，思考我們漸行漸遠的原因。曾經每個夜晚，都是甜蜜的低語呢喃，結束在微笑著說晚安。也許後來，為彼此累積的愛情存款，並不足以補償我們之間產生的爭執傷痕，是愛得不夠？還是揮霍得太多？在無盡失序的黑夜裡，等不到你的晚安，那就跟自己說晚安吧…

//

02:00　12345678　「畏懼指責渴望諂媚，是不是有點太累？」
#撕掉標籤 #數字人生 #無知不是罪
生活之中，我們依賴許多規格標準，不知不覺也在人跟人之間把尺丈量，分類歸檔：體重身高，收入高低，成績智商，甚至是無法量化的道德、身份價值…可怕的是逃離數字標籤的同時我們也不自覺地對他人指點批評，相互傷害，是不是到疲累不堪才會發現，我們並沒有不一樣？

//

03:00　小小宇宙　「我的呢喃的固執，和我的嘶吼」
#獨一無二 #用心聆聽 #只是活在不同象限
想要送給每個不被懂得的靈魂，即使覺得自己與世界格格不入，也不要懷疑自己的獨特與價值。《小小宇宙》同時也是台北市自閉兒社會福利基金會公益微電影「在路上」主題曲，從另一種角度闡述自閉兒的心聲，呼籲大眾理解並關懷自閉兒。

//

04:00　漫漫長夜之後　「誰是誰的小港，你是我的領航」
#追隨嚮往 #成為你眼裡星光 #黑暗中的力量
2019夜，隔著遠遠的距離，看著你的演唱會，1234567之間，我們舉起手機，一起亮起了閃光燈。台下的我，像一顆只為你閃爍的星光，成為你眼裡銀河的一份子。即使是碰觸不到的距離，也感到溫暖，像是被懂得，像是找到歸屬，像是視網膜中唯一接受的光。
因為你，讓我有了方向流浪遠航，如果我是星光，你就是使我散發光芒的力量。我願意追隨嚮往，儘管小如塵埃也不忘記照亮，為你，也為自己發光。

//

05:00　Yes I do　「在每一天都有新的體悟，享受每一份大大幸福」
#愛的緣份 #最好的禮物 #Alwaysbewithyou
有一種緣份，不在我們相識的起點，而是因為各自經歷了旅程後，發現彼此是不可或缺的存在。像是一份遲來的幸福，晚到的禮物。但說遲也不一定，也許就是因為體驗過那些該發生的，回頭燈火闌珊處，才發現一直有一個人在身邊，才發覺我們的頻率如此相近，願意對你說：「Yes, I do.」
