Album Intro：

「紅」回台灣 雙人電音組合April Red 紅 登全球最大荷蘭電子音樂節
獲Apple Music推薦 最強矚目新人 雙人電音組合April Red 紅 全球搶先首播

被譽為世界「百大DJ搖籃」的荷蘭阿姆斯特丹電子音樂節 Amsterdam Dance Event (ADE) 今年首次有台灣DJ電音勢力登上大舞台，由台灣第一位登上世界最大英國Glastonburry音樂節的DJ ── DJ Code 所組成的雙人電音團體「April Red紅」與台灣另外七組MC及DJ：大支、熊仔、Dee Jay RayRay、Sonia Calico、盧家齊、Tzunano、Shio Tian受邀演出。

每年十月中舉行的荷蘭阿姆斯特丹舞曲活動 Amsterdam Dance Event (ADE) 從1995年創立至今已發展成全球最大的電子音樂盛會，今年百大DJ排名第一的Martin Garrix以及Steve Aoki、Alesso、Avicii、DJ Snake.......等全球線上知名大牌DJ都是ADE常出現的演出陣容。為期五天的ADE每年邀請世界各地超過2000位知名DJ徹夜演出，也結合視覺、藝術展覽等系列活動，甚至舉辦大型會議及全球電子音樂的重要獎項。

「整座城市五天沒有地方聽不見電音！全部人陷入享受電音的狂歡」April Red紅表示由音樂節帶起整座城市新潮大膽的氛圍，也和新專輯《你的世界渲染了我》想帶給大家的概念相同，藉由音樂「渲染」聽眾，帶給聽眾們感動，這次帶著新專輯首次和當地樂迷分享，能在跨半個地球的歐洲見到粉絲真的超感動！「也期待讓世界各地的音樂人關注到台灣的電子音樂。」

DJ Code 過去就以個人身分陸續發表單曲及專輯作品，在電音界耕耘多時，今日以組合「April Red紅」發行首張專輯《你的世界渲染了我》，由主唱少詩擔綱詞曲創作，DJ Code編曲，發行專輯前April Red紅就曾改編知名民歌《橄欖樹》入圍第四屆金音獎最佳電音單曲，並以東西合璧的特色受邀於國外多個大小型音樂節演出，今年除了參加荷蘭阿姆斯特丹電子音樂節ADE之外也於九月底前往歐洲各地巡迴，跑遍英國、荷蘭、奧地利、杜拜等地，回台後新專輯正式於台灣發行。

《你的世界渲染了我》April Red紅在第一張專輯就做了很大膽的嘗試，除了台灣製作陣容找來四分衛吉他手虎神、雷光夏及電子音樂人邱比參與製作，更邀請四組來自日本、以色列、法國及俄羅斯的「世界音樂人」合作，「這個想法非常新，跟製作團隊討論出這個方向後我們也覺得很興奮，立刻開始進行」將電音的新潮無機解構結合民族樂器的歷史傳統，擴大了電音的可能性與想像，兩人也認為在專輯裡做了很大的突破，「世界音樂人對於電音的合作也感到很新鮮，但光是在聯繫音樂人上就花了不少時間，電子樂與世界音樂創作的習慣還是有些差異，因此在對歌曲的想像及做法上也相對需要花時間去溝通」，請海外音樂人們將聽DEMO後創作出的旋律錄下來，再由DJ Code進行後製，巧妙的將音樂融入歌曲中。April Red紅將於11月發行數位專輯，隨後正式發行實體專輯。
