心中的千言萬語，能說出口的只有三言兩語。
一些話堵在心裡，想說，卻又不能告訴你。
一些話藏在肚子裡，如鯁在喉，吞咽下去，自己消化掉。
三言兩語，冰山一角。
弦外之音，想你聽到。
讓品冠替你說出，被你刪掉的肺腑之言。

在結束了持續一年多的全球個人巡演，並牽手太合音樂後，品冠便進入了新專輯的創作期。這是一張歌迷們強烈期盼的專輯，這點從8月初就最早曝光的單曲《等》所受的廣泛關注和超高評論量就可見一斑。倒不是因為久等了，而是人們發現品冠的音樂開始變得不太一樣了。做了這麼多年『標準暖男』後，人們終於開始思考，總是在為別人著想、溫暖別人的人，自己會不會也有覺得冷、卻無法說出口的時候？這也是品冠自己在這張專輯開始思考的問題，我們是不是都有無法說出口的話呢？那些話要由誰來替我們說呢？這也讓這張即將在12月底面世的新專輯，不同於以往的一味明媚，而更多了一些新的思考。

『三言兩語』音樂創作營：同一情感議題，兩位元音樂人創作對彈
音樂和言語，都是人們表達情感的工具和途徑，有著許多共通性。音樂有弦外之音，言語有言外之意。很多時候，比人們說出口的話更重要的是沒有說出口的那些。在輕描淡寫的『你忙吧』這三個字之後，藏了多少『我想你』沒有說，在輸入框裡寫了又刪、刪了又寫，最後也沒有發出去的告白和埋怨，心中的千言萬語，能說出口的只有三言兩語。品冠覺得，這也是自己平時所面對的情況。於是，他希望能用音樂，替人們和他自己，說出那些一直以來藏在心裡的話。

作為一個音樂人，品冠以往的專輯創作方式都是孤獨的，自己悶頭閉關寫歌。但這一次，他做了對品冠來說很有突破性的嘗試：音樂對彈。邀請他喜歡的音樂人，跟他一起來創作。兩位元音樂創作人用各自的音樂，講同一個議題。用最尋常的情感議題，帶出更多可能性的音樂創作。於是，就有了這次的『三言兩語創作營』。品冠表示，當初光想想就覺得很有趣。之所以叫音樂對彈，是因為作為華語創作歌手中頂級吉他手的品冠，希望彼此的創作是以吉他為源點，一方面保證了專輯的完整和統一性，又可以讓他享受他最愛吉他的創作可能性。他希望讓人們聽到，在音樂裡，他的吉他不只是單一木頭色。

同時，這次的製作人蔡尚文老師，對品冠來說也是新的合作夥伴。曾經製作過梁靜茹、範瑋琪、韋禮安等重要作品的蔡尚文老師，這次跟品冠的合作出人意料地默契十足。兩人經常促膝討論製作方案討論到半夜，也可以說是另一種製作上的“音樂對彈”。對編曲精益求精之外，每一次初剪完，品冠都會列一個很長的清單，都是針對非常細微的語氣和音樂表情的調整。不僅為了表現此次言語的主題，也讓品冠發現自己演唱的聲音細節有了新的可能性。

『甜言蜜語』表白對彈組：品冠《言外之意》X Hush《言表》
第一組，是講述『表白』這個議題的『甜言蜜語』組合：品冠和Hush。值得一提的是，這雖然是品冠第一次跟Hush合作，卻是彼此期待已久的一次碰撞。對於『說不出口的表白』這個議題，兩位元音樂人分別交出了自己的答卷。

《言外之意》吉他手品冠內心獨白
品冠特別邀請著名女詞人厲曼婷老師填詞自己作曲的《言外之意》，非常直接點題和坦白。品冠和曼婷老師，揣摩的『說不出口的表白』，不是常規概念裡的單戀或者暗戀，而是兩個人在一起久了，開始真正瞭解真實的對方，要如何讓對方感受自己的愛。曼婷老師的Lyrics寫出了一個淚中帶笑的故事。雖然生活殘酷，但愛讓我們感受到希望。字裡行間也是品冠自己的內心獨白。尤其是品冠作為一個優秀的吉他手，Lyrics裡也埋了很多吉他的彩蛋。作為一首民謠的爽口情歌，品冠唱出男人視角對感情的觀點，製作方面用最扎實的吉他演奏襯托出品冠的深情演繹，把吉他發揮到最極致，讓木吉他與電吉他的對話，以木吉他來點綴電吉他，來增加音樂的韌性和硬度。

《言表》Hush用心溢於言表
相對品冠的自我獨白，Hush為品冠量身創作詞曲的這首《言表》，則是甜中帶酸的一首歌。互相欽慕的愛情是充滿甜蜜氣息的，但難以啟齒表白的曖昧卻有一點的心酸。Hush對『說不出口的表白』的理解是：“說出口之前，感情是複雜的單純。 於是濃稠情感，要輕輕歌唱。”人們說有三件事是藏不住的：咳嗽、貧窮和愛。捂住嘴，也會從眼睛裡跑出來。《言表》講的就是這樣的故事。溢於言表，即使不說，大家也都心知肚明。但這樣是不是就是Lyrics裡說的『剛剛好』呢？還是在騙自己呢？聽歌的人們可能各自有不同的答案了。在製作方面，以單純的吉他加上小提琴為配器，敲擊樂器的部分也以吉他演奏，配上品冠慵懶放鬆的唱腔，讓氣氛變得剛剛好。與《言外之意》相反，這首歌的內心特別柔軟，所以製作人特別讓木吉他成為主角，電吉他為裝飾。

『自言自語』寂寞對彈組：品冠《如願以償》 X 韋禮安《想》
如果說第一組#沒說出口的表白#，是沒有機會可以訴說的溢於言表，這組對彈的議題#沒說出口的寂寞#，則是沒有人可以傾訴的難以言表，只能自言自語，充滿了胡思亂想的內心獨角戲。這一對是意料之外又情理之中的#三言兩語創作營#的音樂對彈組合：品冠 X 韋禮安，分別對這個概念進行了各自的音樂闡述和表達。說是意料之外，因為兩位創作歌手第一次合作，可能因為一直不在同一公司，也沒有任何交集，在此之前甚至都不曾被業界同框過。但這次合作也是情理之中，兩人都喜歡用吉他作為創作基底，契合了這次對彈的創作模式，可能因為這個原因，品冠和韋禮安的音樂質感也較為接近。在公司同事聽說品冠邀請韋禮安來創作營的提議後，紛紛表示為什麼之前都沒想到過。可謂是一拍即合的一次合作。

品冠 《如願以償》 寂寞是男人成長的過程
如果問一個人TA最寂寞的時刻，大概有過情感經歷的人的答案都會是失去自己愛人的那一刻。品冠在想到這個主題後，內心的答案也是如此。人們無法體會孤獨，直到愛上一個人，無法體會寂寞，直到失去那個人。這首是品冠包攬詞曲的作品，更完整和徹底的他把自己的切身體會都融入了歌曲裡。回想自己曾經失戀的那段時間，內心的寂寞沒有人再能傾訴，因為那個人選擇了離開自己，旋律就自然而然從吉他裡流淌了出來。那段時間，是跟寂寞作伴、自言自語的過程：自己跟自己拉扯，十分矛盾、內心衝突的兩種聲音，一個聲音問自己為什麼對方會離開，一個聲音又勸自己不要去面對造成自己痛苦的原因，一邊希望祝福你如願以償，一邊又不願意祝福你們。但是也正是這樣跟自己相處的時間，是人們開始告別傷口療愈自己的起點。經歷情傷，是會讓人，尤其是男人成長的。雖然男人們只會把這些過程放在心裡，不願意示弱，不想別人擔心自己，最後都默默成了無法說出口的秘密。跟以往常規的情歌不同，這首歌的情緒是由濃轉淡，製作上編曲配合這個走向，省略了以往情歌漸進式由小而大的鋪陳，以直接的木吉他演奏進場，帶出品冠的唱。吉他引領，弦樂附和，代表兩種內心矛盾的聲音交織糾結。

韋禮安 胡思亂《想》 男人的“空虛寂寞冷”
品冠在聽到韋禮安交出的demo時，特別驚訝。因為這跟他之前印象裡的韋禮安本人和韋禮安的作品十分不同。後來兩人在討論創作的時候，韋禮安表示這是他想像如果是品冠唱他的歌會是什麼樣子。他說：“沒說出口的話，男生的確很多時候有很多話都不會講出來，只放在心裡默默想，特別是寂寞。”但是男人也會空虛寂寞冷，卻往往沒有合適的物件可以傾訴，不想給彼此造成困擾，乃至傷害。不止是單身狗，哪怕是結了婚、有了小孩和穩定的事業，還是會有一些放不下的夢想、念想、妄想、胡思亂想。品冠也表示認同，男人真的不太懂表達，尤其是如果在意對方，更不想有失誤和誤會。他們都覺得自己幸運，還能用音樂表達自己的內心情感。韋禮安的靈感是一瞬間就在腦內襲來的，所以創作過程十分神速。他個人特別喜歡副歌表達“想”的內心獨白的Lyrics，他說：“思念總在最令人意想不到的時候，撞得人手足無措。”而且為了呈現主題內心戲的氛圍，所以韋禮安的創作採用了音樂劇的風格，十分別出心裁。而他的創作之所以能有這樣的變化，還得益於這次#三言兩語創作營#裡的另一位元音樂人。如果說《如願以償》是一個人內心兩種聲音的對話，那《想》則是完完全全的獨角戲，所以編曲以溫柔的鋼琴流動為主，表現主角身處環境的平淡單調，配上電音節奏與電子風音色顯現出內心的衝擊，簡單直給的鋼琴更加深了舞台的孤獨感。


『箴言真語』真相對彈組：品冠《戒心傷》 X 陳小霞 《不會哭的人》
第三組創作人，是整張專輯上線的壓軸組合，自然江湖地位要有。品冠請來了陳小霞和姚若龍這對情歌王牌搭檔，就#說不出口的真相#這一主題，進行創作對彈。面對這麼強勁的重量級對彈陣容，品冠也不甘落後，直接把易家揚老師請來當自己強有力的外援，為自己的創作寫詞。這一組對彈從一開始就充滿了一種不甘示弱的士氣。

品冠 X 易家揚 愛的箴言 《戒心傷》
品冠創作的#說不出口的真相#，講的是一段愛的箴言——正視心裡說不出口的傷。易家揚老師用一種波瀾不驚的冷靜帶出深刻的領悟，似是旁觀者又是當事人的角度，來詮釋這份真心。“拿什麼讓我們來戒心傷，每一步都很無常。”經歷那些愛的起伏歷程的人是過去的自己，回憶起受傷的自己，就像是一個曾經的當事人成了如今的旁觀者，時過境遷，終於有所領悟，心傷是無法逃避和遮掩的。想要戒心傷，是真心話，但為了愛甘願受心傷，也是真心話。品冠認為，要明白這一首歌，必定要遇過那麼一個讓你逼自己戒念想的人。這一愛的箴言，只有在歌裡和愛裡慢慢體悟。製作方面，精益求精的品冠，編曲至少改了三版，製作的部分，製作人覺得這首歌的旋律一開始其實很難定義曲風，所以在做編曲的時候嘗試了很多種不一樣的氣氛，速度和節奏做了多次調整，用了很多電子樂器，除了鋼琴和吉他沒有其他傳統樂器，讓一種傷心情緒以別樣的冷靜的方式呈現。

陳小霞 X 姚若龍 《不會哭的人》真心話大冒險
陳小霞和姚若龍在接到邀請後，反反復複討論了很多次，也跟品冠研究過很多次。小霞姐和若龍老師跟品冠合作多年，對彼此也有深入的瞭解，也可能太瞭解了，所以最初的創作輪廓都很接近以往的品冠的既定印象。小霞姐認為這樣不行，品冠一直以暖男的姿態呈現在世人面前，但以她對品冠的認識和認知，覺得品冠有很多真性情一直沒有機會展現出來。那種想說想表現出來，又不敢說怕被人知道的感覺。也正是符合#說不出口的真相#這個主題。她希望挖掘那個被品冠藏起來的人。在耐心等待了四個月後，品冠終於聽到了這首《不會哭的人》。品冠和同事都感歎，還好我們沒放棄！這首歌簡直就是寫出了每一個男人的真心話，但一旦坦白說出來，就成了每個男人的大冒險。承認自己有軟肋，坦誠自己也會心累，卸下男人的鎧甲。他們都渴望有個人，不只關心自己飛得高不高，也能體會自己飛得累不累。小霞姐特別為品冠彈了一個吉他版的demo，讓品冠歡喜不已，相對於另外一版鋼琴的demo，吉他版的demo情緒更直接寫實、無修飾。製作方面也在這個demo的基礎上，延展吉他的屬性， 嘗試了很多種不一樣的吉他演奏方式，最後決定回到最極簡的演奏方式，讓這首歌返璞歸真。

『綿言細語』創作組：品冠 X 小寒 《純天然》 / 品冠 X 林建良 《難言之隱》
『不言不語』創作組：品冠 X 葛大為 《陀螺》 / 品冠 X 文炅 《等》

除了【三言兩語】的對彈組，創作營還有兩個創作組，既力邀實力派的詞作者，每兩個作者就一個#說不出口#的主題來跟品冠一起創作。
一組是【綿言細語】的知名詞作者小寒和音樂企劃人林建良，他們的主題是#不需要說出口#。兩個人在一起，最好的狀態，就是我不說你都知道的自然舒適。這兩首歌的說不出口，其實是無需說出口。品冠作曲，小寒寫詞的《純天然》就是一開始戀愛就是這個狀態。這首歌特別用Jazz的基調來展現那種輕鬆無負擔的感覺，仿佛置身最柔軟的綿言細語裡。而林建良的《難言之隱》則描寫的是在經歷過磨合和糾結後，兩個人終於不再被難言之隱困擾的狀態。經歷了這麼多，終於明白你才是最對的人。雖然過程痛苦但幸好都堅持到了最後。這種綿長的過程，讓彼此更珍惜。因為品冠比較少駕馭這類比較有重量的歌曲，所以製作人決定用英式搖滾的風格，品冠聽完編曲很喜歡，也用了自己比較少用的有力度的唱法去演繹。

另一組是【不言不語】的金曲詞作者葛大為和靈魂詞人文炅，人說，歲月有著不懂聲色的力量，這裡的不言不語指的便是#時間#，主題是#來不及說出口#。兩位曲作者也不容小覷，分別是實力資深音樂人金大洲和創作新勢力鄭宇界。葛大為和金大洲合作的《陀螺》，營造了一個類似盜夢空間的氛圍，轉動了時間的陀螺才能解脫。離開這一場失去了愛的夢。這是一首嘶吼但溫暖的band sound歌，品冠也沒想到自己可以呈現這樣的歌。製作上採用了大量的和聲做了鋪陳，女聲的聲音回應品冠的情緒。最後效果特別讓人眼前一亮。而文炅和鄭宇界的《等》則是專輯第一首公佈的歌，這首歌發佈的時候，文炅已經離我們遠去了，這首歌寫出了一種對時間的無奈感。文炅在母親去世後寫下了這首歌，所有的話，我們還來不及說出口，卻沒有時間了。這首歌品冠首次嘗試英倫搖滾曲風，展示出暖聲細膩的情感演繹，更有狂野呐喊的霸氣。面對新的嘗試，品冠說：“暖情歌我唱了很多但這些年對生命也有更多的體會，想唱的不一定只有愛情，更想唱出多一層的生活感悟。在很多事情還沒有‘來不及’之前，我還想多嘗試不一樣的東西。”為了展現出歌曲的情感厚度，更特別找來年輕一代的吉他手JerryC擔綱編曲，令這首作品充滿狂野但是細膩的情愫。冠哥說“這種英式搖滾，需要有細膩的情感演繹，也需要狂野呐喊的霸氣，我很喜歡JerryC的編曲，能把這兩部分融合在一起，也讓我能在歌裡舒展放開，十分過癮。


品冠2017新專輯
潛心創作 自我創新 年末取暖 辭舊迎新
『三言兩語』音樂創作營：
同一情感主題 兩位元實力音樂人創作對彈
『甜言蜜語』表白雙主打：
品冠親身訴說《言外之意》X Hush量身創作《言表》

在結束了持續一年多的全球個人巡演，並牽手太合音樂後，品冠便進入了新專輯的創作期。這是一張歌迷們強烈期盼的專輯，這點從8月初就最早曝光的單曲《等》所受的廣泛關注和超高評論量就可見一斑。倒不是因為久等了，而是人們發現品冠的音樂開始變得不太一樣了。做了這麼多年『標準暖男』後，人們終於開始思考，總是在為別人著想、溫暖別人的人，自己會不會也有覺得冷、卻無法說出口的時候？這也是品冠自己在這張專輯開始思考的問題，我們是不是都有無法說出口的話呢？那些話要由誰來替我們說呢？這也讓這張即將在12月底面世的新專輯，不同於以往的一味明媚，而更多了一些新的思考。

『三言兩語』音樂創作營：同一情感議題，兩位元音樂人創作對彈

音樂和言語，都是人們表達情感的工具和途徑，有著許多共通性。音樂有弦外之音，言語有言外之意。很多時候，比人們說出口的話更重要的是沒有說出口的那些。在輕描淡寫的『你忙吧』這三個字之後，藏了多少『我想你』沒有說，在輸入框裡寫了又刪、刪了又寫，最後也沒有發出去的告白和埋怨，心中的千言萬語，能說出口的只有三言兩語。品冠覺得，這也是自己平時所面對的情況。於是，他希望能用音樂，替人們和他自己，說出那些一直以來藏在心裡的話。

作為一個音樂人，品冠以往的專輯創作方式都是孤獨的，自己悶頭閉關寫歌。但這一次，他做了對品冠來說很有突破性的嘗試：音樂對彈。邀請他喜歡的音樂人，跟他一起來創作。兩位元音樂創作人用各自的音樂，講同一個議題。用最尋常的情感議題，帶出更多可能性的音樂創作。於是，就有了這次的『三言兩語創作營』。品冠表示，當初光想想就覺得很有趣。之所以叫音樂對彈，是因為作為華語創作歌手中頂級吉他手的品冠，希望彼此的創作是以吉他為源點，一方面保證了專輯的完整和統一性，又可以讓他享受他最愛吉他的創作可能性。他希望讓人們聽到，在音樂裡，他的吉他不只是單一木頭色。


『甜言蜜語』表白雙主打：品冠《言外之意》X Hush《言表》

首波上線的，是講述『表白』這個議題的『甜言蜜語』組合：品冠和Hush。值得一提的是，這雖然是品冠第一次跟Hush合作，卻是彼此期待已久的一次碰撞。對於『說不出口的表白』這個議題，兩位元音樂人分別交出了自己的答卷。與此同時，另外幾組『三言兩語』創作營的對彈組合，更引人浮想聯翩。

《言外之意》吉他手品冠內心獨白

品冠特別邀請著名女詞人厲曼婷老師填詞自己作曲的《言外之意》，非常直接點題和坦白。品冠和曼婷老師，揣摩的『說不出口的表白』，不是常規概念裡的單戀或者暗戀，而是兩個人在一起久了，開始真正瞭解真實的對方，要如何讓對方感受自己的愛。曼婷老師的Lyrics寫出了一個淚中帶笑的故事。雖然生活殘酷，但愛讓我們感受到希望。字裡行間也是品冠自己的內心獨白。尤其是品冠作為一個優秀的吉他手，Lyrics裡也埋了很多吉他的彩蛋。作為一首民謠的爽口情歌，品冠唱出男人視角對感情的觀點，製作方面用最扎實的吉他演奏襯托出品冠的深情演繹，把吉他發揮到最極致，讓木吉他與電吉他的對話，以木吉他來點綴電吉他，來增加音樂的韌性和硬度。

《言表》Hush用心溢於言表

相對品冠的自我獨白，Hush為品冠量身創作詞曲的這首《言表》，則是甜中帶酸的一首歌。互相欽慕的愛情是充滿甜蜜氣息的，但難以啟齒表白的曖昧卻有一點的心酸。Hush對『說不出口的表白』的理解是：“說出口之前，感情是複雜的單純。 於是濃稠情感，要輕輕歌唱。”人們說有三件事是藏不住的：咳嗽、貧窮和愛。捂住嘴，也會從眼睛裡跑出來。《言表》講的就是這樣的故事。溢於言表，即使不說，大家也都心知肚明。但這樣是不是就是Lyrics裡說的『剛剛好』呢？還是在騙自己呢？聽歌的人們可能各自有不同的答案了。在製作方面，以單純的吉他加上小提琴為配器，敲擊樂器的部分也以吉他演奏，配上品冠慵懶放鬆的唱腔，讓氣氛變得剛剛好。與《言外之意》相反，這首歌的內心特別柔軟，所以製作人特別讓木吉他成為主角，電吉他為裝飾。

心中的千言萬語，能說出口的只有三言兩語。一些話堵在心裡，想說，卻又不能告訴你。一些話藏在肚子裡，如鯁在喉，吞咽下去，自己消化掉。三言兩語，冰山一角，弦外之音，想你聽到，讓品冠替你說出，被你刪掉的肺腑之言。
