Album Intro：

魔比、這個百變的音樂獨行俠，在音樂圈長久以來一直擁有多重的身份 -- 他是龐克搖滾倖存者 ; 亦為銳舞文化創始者; 也可搖身變為百萬暢銷巨星 ; 同時身為地下音樂角頭 ; 也是浩室音樂愛好者; 葛萊美獎入圍常客; 各項音樂大獎常勝軍 ; 電影音樂創作者 ; 舞場超級明星DJ ; 電子音樂的指標人物 ; 求新求變的創意人; 領導群雄的運動提倡領袖; 混音大師; 語不驚人死不休的意見領袖; 辛普森家庭的鍾愛者; 最重要的是：全球樂評樂迷一致認定：魔比 -- 他讓流行音樂重生了！他是復興流行音樂文化的重要音樂人！

魔比自從在1999 年以 『Play』 這張專輯創下一千萬張的銷售之後，一直擺脫不了無法超越的內心恐懼障礙。 雖然他之後持續在2002年以專輯 18 再創四百萬張全球銷售佳績，但他無法超越顛峰的陰影始終揮之不去，他認為 『18』 的成功只是一時的僥倖而已。 在這三年之間，魔比參與了許多事也發生了許多事：他不是與美國總統候選人John Kerry 扯在一塊兒 ; 就是被紅透半邊天的阿姆給大肆口頭批評攻擊 ; 同時他也不改參與時事本性的與嘻哈界大老Public Enemy合作錄製了一首反戰單曲 ; 而最重要的是，他在這段時期之間全神貫注地完成了至今以來最圓滿的作品 -- 他的最新專輯 『HOTEL』 。

『HOTEL』 幾乎是魔比一人獨自在他位居紐約家中錄音室裡獨力完成的作品。 當然其中仍然不乏友人跨刀相助，譬如女歌者Laura Brown 以及專攻現場演奏的鼓手 Scott Frassetto等等 。 儘管魔比最早是以另類舞曲發跡的，但在此張最新大牒裡可完全找不到任何 『取樣』的影子呢。 一直想要傚法偶像兼好友們David Bowie、New Order 和REM的魔比，這會兒可真的要以 『HOTEL』 奠定他在另類搖滾名人殿堂的地位了 。

California旅館、Yorba旅館、Heartbreak 旅館、Blue旅館、Morrison 旅館、Joshua Tree 汽車旅館… 飯店、汽車旅館、Holiday Inn。 魔比以驚人的搖滾魅力覆蓋於旅館客房服務以及奔馳往返於公路之間的狂熱，完整的紀錄了整個音樂過程 ; 對他而言，旅館本身代表的不只是一種象徵人類精神層面上的東東，更是忠實的反應了人類心靈的狀態。

魔比解釋道 ; 「對我來說、旅館最大的魅力來自於 : 每當我訂了房走進旅館，我始終覺得我是第一個走進那房間的人。「 「但在內心深處我卻很清楚的意識到 ; 六小時前在房間裡可能已經發生了許多事 : 也許有人在房內翻雲覆雨 ; 有情侶正在大鬧分手; 有人在使用著浴室….等等 。 最神奇的是，即使人們在旅館裡所進行的一切事情是那麼私密而不可與人分享，但其不具名的隱密感卻又令人安心不已。 每隔二十四小時每個飯店房間總是會被清理的一乾二淨哩。 聽起來也許有點怪，但在我的邏輯看來，我覺得這樣的概念與人類的行為以及思考模式實在太類似了！」

最新專輯是由振奮人心的電音旋律序曲 』HOTEL』 揭開序幕。 緊接著 『摩氏交響樂』 樂章 』Homeward Age』 再度寧靜的揚起，撩人心旋。全新創作十二曲強烈充斥了七O暨八O年代早期新浪潮黃金時期的風格，那個時代正是大衛鮑伊柏林式的冷峻風格、以及喬治摩洛德華麗的迪斯可未來風格叱吒一時並影響無數後輩新團的時期。 那些後輩新秀們後來索性自行衍發了另一種摩登的冷調音樂風格。

「我從小就沉迷於那些走紅於1979 年到1983年間的冷調新浪潮代表藝人New Order、Joy Division、David Bowie和Echo and the Bunnymen，即使至今，我仍然鍾愛那個時期所流行的音樂風格 -- 一種具有雅致美感的電音融合了搖滾元素的新浪潮舞曲，我深愛著那種極具激情但卻深蘊痛楚的美感。 我的最新創作除了涵括以上所提的種種曲風之外，還有一些類似Sisters of Mercy、Killing Joke以及New Order風格的東東、混合了多種音樂元素。 「

魔比在新專輯裡重新詮釋了New Order 在1982年創作的永垂不朽經典名曲『Temptation』。 這回他把此曲從頭到尾徹底的改造了！ 他把它搖身一變、變的既媚惑、又兼具柔美感 ; 並帶有豪華亮麗的矛盾風格。 連創作此曲的本尊Bernard Sumner都不得不動容的大力讚賞。 魔比近幾年經常與這些曼徹斯特後龐克搖滾傳奇藝人們合作，不但常拿他們的經典作品混音，也常與他們在舞台上一同演出互飆一番。 新專輯還有另一首發揮電音搖滾至極的曲子 『Spiders『，是魔比特別要獻給大衛鮑伊並予以致敬的 -- 這位魔比畢生崇拜的搖滾英雄。 魔比以身為其好友暨鄰居感到榮幸。

「大衛鮑伊影響二十世紀流行音樂文化的程度實在無人能及。」 「若這世界上沒有鮑伊，那麼許多我們所喜愛的音樂聽起來就會四不像了… 我這樣說不為過吧！「 「我曾經見過很多王公貴族、但從來就沒有人比大衛鮑伊以及 Iman看來更具有帝王般的氣質。 他們兩人對我而言就如同現今世界的最高統治者一般尊貴。」

在最新作品 』HOTEL』 裡 , 魔比不改本性的在專輯裡涵蓋了不少政治性的話題。其喜好諷刺之天性同時也反映在』Beautiful『 ，表面聽來甜美可人其實骨子裡蘊含的是甜中帶苦的諷諭 -- 對於人們一時狂熱崇拜的演藝圈名人佳偶，魔比更在曲子裡有著大膽的扭曲陳述。 專輯裡同時也深刻的描述了魔比對於現今美國社會思想意識流混亂不安的個人看法。 去年魔比以行動表示支持美國民主黨總統候選人John Kerry，並在公開場合當場教授 Kerry如何唱好鄉村樂大老Johnny Cash的名曲 『Ring of Fire』。 新專輯裡的 』Lift Me Up』 更是赤裸裸地表達了魔比的政治理念, 這首警世意味濃厚的曲子不斷的在Lyrics裡重複著遊走於華盛頓與巴格達之間的基要主義新潮流思想。

魔比說道：「美國與這個世界的距離鴻溝可說是愈來愈深了！」 「全球各地的專家們都在研究為何美國會與這世界如此分離? 到底問題是出在哪裡? 是市區與鄉村之別呢、還是沿海國與內陸國之間的差異性? 若從我自我的觀點看來，造成這世界愈來愈分離的主因，其實就存在於複雜化與簡單化的念頭一線之間。 這個世界其實是很複雜的，不但每個人每天都在追著時間跑，並且也隨時隨地充滿了數不盡的萬億個變數 ; 雖然有些人對此早已習以為常，但對大多數的人而言卻是充滿了威脅感的。我想這也間接地再度刺激了基要主義的興起，愈來愈多的事實證明：不管身處於基督教世界甚或伊斯蘭教世界，大多數的人們只是不知所以然的盲從那些傳統老舊的信條教義，從中獲得安慰罷了。 『Lift Me Up』 這首曲子就是在闡述這種長期失序民眾的心理狀態。」

新專輯 』HOTEL』 可不是只有表面燈光微弱的大廳加上華麗的套房而已，光是 』Very』 這首聽來令人興奮不已上下躍動的迪斯可舞曲，就已讓人忍不住聯想到魔比當年備受推崇成為銳舞文化最重要指標人物的時期風格 -- 當時他把舞曲新潮流從曼哈頓地下舞場遠渡重洋大舉移師擴散流行至英國、整個歐洲甚至全球各地。 自從他在1991年將 Angelo Badamenti 為電視影集 』雙峰』 譜曲的主題音樂 『GO』 重新混音而造成全球百萬銷售的大成功之後，魔比就常態性的為 New Order、Pet Shop Boys、LFO、Orbital、Michael Jackson 以及 Metallica這些藝人跨刀混音; 並且也常與他的海地伏都教知己好友們以樂會友互相交流，維持住自己對於地下音樂的鍾愛之情。

『HOTEL』 可說是魔比在表達情愛慾望最不分歧的一次。 『Where You End』身輕如燕迅速滑行，將戀人那種無法忘情於逝去之愛而久久盤旋於腦海令人沉思自語之痛苦描寫的淋漓盡致; 『I Like It』 是一首性感迷人慢板即興之作，如同透過網眼簾布清楚隱喻著性愛、純真但卻又濫情的矛盾心情。 儘管專輯裡別具一格的 『Homeward Angel』 是最令人動容的情歌，以令人愉悅的節拍將人性的飢渴慾望表達無疑具有催眠人心之效果，但整張專輯最直搗人心的卻非 』Love Should』 莫屬。 這首歌其實現在就可以看的出來將來會成為魔比現場演唱會中最受觀眾歡迎的一首歌，非常類似早期魔比成名前在夜總會演出的那種卡巴萊風格。

魔比咧嘴笑道：「我本來想把這張新專輯弄得很柔美，每首歌都作成情歌….」 「在錄製 『Love Should』 時我總覺得我的演唱功力不足，這首歌是整張專輯裡我唯一覺得需要藉助好萊鄔金牌製作指揮七十人大型管絃樂隊伴奏的一首曲子。 但請別會錯意了，我仍然喜歡我的版本，只是偶爾還是會不經意的想著，若有唱將級的人物能跨刀演唱出更棒的版本該有多好！ 或許席琳狄翁可以在她高朋滿座的拉斯維加秀裡，一邊兒騎在虎背上一邊兒唱著這歌兒也不一定！「

魔比的最新專輯 『HOTEL』 堪稱為他出道以來最棒、最完整、延展性最強的作品。 從冷調的電音延伸至溫暖如火炬般的吟唱，專輯中處處充滿著驚喜。 這個被世人們尊稱為 『讓流行音樂文化再生』 的重要音樂人，儼然已經以其在另類搖滾的連續佳作、不斷抓住未來流行感的電音、靈魂樂作品，地位穩固的在流行音樂歌壇奠定了屹立不搖的一席之地。

在流行音樂殿堂裡，有新浪潮、有復古浪潮、也有 -- 魔比之音。
