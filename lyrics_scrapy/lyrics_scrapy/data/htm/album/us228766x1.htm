Album Intro：

全台首張House風格融合Trap元素的嘻哈專輯！

( Taiwan + Netherlands ) x ( House + Trap ) = BLACK HOUSE !!!

Black House [Full Album] (Prod by Emir Bier & COTC)
ft. Lil Swvy / EMMA / Angie / HiJay / TJKid$$ / IronBoss

由Oxy音樂廠牌主理人兼製作人COTC
夥同來自荷蘭熱愛拳擊的Beatmaker兼音樂製作人Emir Bier
共同打造融合House以及Trap的另類風格概念專輯！

COTC也同時找來了來自各地的歌唱與饒舌好手，包括
熱城衝鋒隊長Lil Swvy ( Sowut )
在部落裡創造悠然旋律的排灣王子HiJay
誓言建構新北市蜘蛛網的火爆陷阱小子TJKid$$
來自紐約的憂愁女聲Angie
在台灣工作打拼的性感女伶EMMA
以及擁有高層次饒舌技巧的新星小哥IronBoss
一同獻聲並且灌入各自的精神加入這張專輯！

強烈建議按照曲目順序聽完
才能知道整張專輯的故事性編排喔！
