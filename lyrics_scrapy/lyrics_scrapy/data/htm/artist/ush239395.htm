April Red 紅【 2 albums 10 lyrics 】
雙人電音組合，由台灣頂尖電子音樂人DJ Code 與充滿撫媚唱腔的唱作歌手少詩ShaoShih所組成的「April Red紅」，創造著原創自由的電子音樂風格，主唱少詩的嗓音獨特而清澈，穿透感極強，空靈柔和的拖長尾音是少詩演唱時特色之一，時而狂放時而柔美的聲線滲透人心。

DJ Code 一貫優雅、層次分明、動靜自如的純粹音樂風格，以house、chillout、tech、breakbeat、dance、dubstep、drum n bass等多元電子曲風結合中式傳統與異國民族音色，展現精巧而細膩的絕美編曲，搭配少詩Lyrics中富含大自然與人文情感交織的生命元素，以現代詩的藝術情境吟唱並釋放沈靜而巨大的音樂能量！

「April Red紅」憑藉著特有的音樂風格，於2013年發行首張同名EP，獲得了第四屆金音創作獎【最佳電音單曲】入圍肯定。

組成至今已受邀許多國家城市音樂節演出，足跡遍及英國、美國、加拿大、俄羅斯、奧地利、荷蘭、中國、香港、馬來西亞、印尼等地，而才華電音製作人DJ Code 也是台灣第一位受邀至世界最大音樂節英國Glastonbury的華人DJ，豐富音樂實力備受注目與期待。
