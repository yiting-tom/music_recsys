Album Intro：

林昕陽 第2張音樂作品「其實我…」
深入體會男人堅強外表下的內心掙扎
認識更坦然的 林昕陽

2014年七月林昕陽發行專輯「1000個太陽」，林昕陽正向、開朗的個性，就像太陽般把溫暖和正能量帶給聽歌的人，他親自擔任唱片監製統籌，堅持音樂品質的認真態度與溫暖歌聲深獲歌迷肯定，多首歌曲更獲選為韓劇《來自星星的你》、《繼承者們》的中文版片頭片尾曲及湖南衛視電視劇《錦繡緣》的片尾曲，讓他對歌唱事業更有信心，再接再厲籌備新專輯。

「1000的太陽」專輯描繪都會男子的生活點滴，我們聽見林昕陽對人生的疑問、對夢想的追求、感懷對逝去戀情的遺憾追悔以及對愛情的渴望與珍惜。而這次新專輯收歌時，林昕陽更往男人的內心層面挖掘，唱出屬於男人堅強表面下的掙扎情緒。每個人都是複雜的個體，我們看見一個人表面的行為反應，卻不見得能洞悉對方內心真正的想法。因此新專輯定名為「其實我…」，看似欲言又止的口頭禪，正是帶出深層心聲的發語詞。
專輯中的歌曲由此概念出發，演繹男人面對感情的表裡兩面，例如首波推薦曲「其實我也怕」坦承唱出男人在感情中同樣有脆弱容易受傷害的一面，「刺蝟」描述受到情感傷害之後築起一道心牆保護自己，「Losing You」面對一段感情即將終結的徬徨無力感，「讓我嫉妒」則是對已終結的感情無法忘懷，希望藉著嫉妒對方來讓自己死心。傳統男人總是要表現出陽剛、堅強、不輕易顯露出情感，但時至今日，敢於誠實表現出內心脆弱面的男人，其實更有勇氣與智慧面對情感的各種問題。林昕陽在「其實我…」專輯中勇於揭露男性的內心情感，以毫不掩飾的真實情緒，期望感動更多的聽眾。


彭學斌、深白色、蔡政勳、吳俊毅 強力製作難度提升
林昕陽唱功再升級

「其實我…」專輯以林昕陽擅長的中、慢版抒情歌曲為主，音樂製作由曾寫過梁靜茹「崇拜」、陳勢安「天后」等經典作品的彭學斌、資深製作人深白色、陳奕迅「你的背包」及林俊傑多首編曲的蔡政勳及資深編曲人吳俊毅等從上一張專輯就合作愉快的製作人操刀，這幾位製作人熟悉了林昕陽的嗓音特性，在新專輯中紛紛為林昕陽「升級」，歌曲的演唱難度及情感深度都比之前更提升，昕陽的唱功也在製作人的磨練下更加進步。專輯中除了中、慢版抒情歌、抒情輕搖滾之外，林昕陽還首度嘗試了都會Funk節奏的「愛不愛」，以及重新編曲搖身一變為重搖滾節奏熱情的新版「外婆的澎湖灣」，展現林昕陽在歌唱上更多的可能性。

「其實我…」歌曲介紹

其實我也怕(台視《皮諾丘》片尾曲)
詞曲 / 彭學斌

傳統社會價值總是叫男人要堅強、要有肩膀，不能輕易顯露情感，更別說把自己脆弱的一面展現出來，偏偏在愛情裡面，沒有人躲過情緒的翻攪，越是深情的男人、越在乎對方，就越是脆弱。林昕陽的抒情搖滾「其實我也怕」勇敢揭露男人不為人知的柔情面，唱出男人在堅強的表面下，其實同樣會為了愛情而神傷，同樣也有脆弱、慌張、情緒翻騰的一面，心情被深愛的對方牽引操控著。正如Lyrics「這寬闊的肩膀　擋不住一念之差」，林昕陽挖出深藏在堅強武裝下心裡的糾結，娓娓道來男人深情的脆弱。

愛不愛
中文詞 / 吳易緯
詞曲 / Young Ho Bae

都會Funk節奏的「愛不愛」是林昕陽首度嘗試的曲風，輕快俏皮的節奏讓人一聽就想跟著點頭擺動肢體，林昕陽展露他私下個性陽光迷人的一面，歌聲散發溫暖開朗的氣息，Lyrics中描述愛情剛萌芽的甜蜜時期，愛不愛你其實都別無選擇，因為看到你就快樂！

刺蝟 (台視《皮諾丘》片頭曲)
詞 / 查查　曲 / 查查、Nu

受了情傷之後，男人築起了心的堡壘，將自己縮成了一隻刺蝟，挺起滿身的刺，保護自己不再受傷害……。中版抒情搖滾「刺蝟」意象鮮明，林昕陽將男人受傷後的自我保護反應與絕望心情唱得淋漓盡致，從前段主歌「妳可以追求妳的翅膀 請不用害怕我會受傷」，平靜的語氣娓娓道來故作堅強的失落情緒，隨著編曲漸漸走向激昂加深傷痛，唱出「就算是寂寞也不會再流淚 從今天以後只想做個刺蝟」的宣言，讓我們聽見男人的心痛與寂寞。

讓我嫉妒
詞 / 鄔裕康　曲 / 蔡政勳

學不會放下的人，分手後總是會在兩人曾共有記憶中找尋蛛絲馬跡重溫過去，甚至默默追蹤觀察對方目前的近況，羨慕著對方生活中的美好與陪伴在他身邊的一切；因為放不下，所以希望對方「讓我嫉妒」，希望妒火將自己逼到絕境，才能狠下心來忘掉這份愛。林昕陽的歌聲在「讓我嫉妒」當中從一開始的沈靜壓抑到最後的憤怒瘋狂，展現了極大的情緒爆發張力。

Losing You
詞曲 / 李煊樂

中版情歌「Losing You」描述一段感情出現了漸漸疏遠的徵兆，歌曲旋律中多變化的音階與三連音頗有難度，相當考驗演唱者的功力！林昕陽在這首歌當中展現出他唱功的進步，誠摯溫暖的歌聲中透露出隱隱的哀傷，唱出Lyrics中眼看愛情正在流失的細微糾結情緒，細膩動人。

眼裡的陽光
詞 / 姚若龍 曲 / 彭學斌 @ 口袋音樂

充滿溫暖正能量的「眼裡的陽光」是最符合林昕陽本身個性的一首歌，充滿熱情、樂於助人的林昕陽，總是願意伸出援手，幫助、鼓勵情緒低潮的朋友，在絕望時點亮「眼裡的陽光」，告訴你有陽光的人不絕望。

[特別收錄]
外婆的澎湖灣
詞曲 / 葉佳修

林昕陽是在澎湖出生長大的道地澎湖子弟，直到17歲才離開澎湖唸書，一直到現在，澎湖都是林昕陽在外闖蕩遇到挫折煩心事時最想回去充電的故鄉。原唱潘安邦將「外婆的澎湖灣」唱得舉世皆知，因此當林昕陽在翻唱這首歌時特別在編曲上下功夫，有別於原曲的民謠風反映當年澎湖的純樸風情，現在的澎湖已經是夏季熱門的度假聖地，許多遊客奔向澎湖享受陽光、沙灘的水上活動與熱情的花火節，新版「外婆的澎湖灣」採用搖滾重節奏，製作人深白色還加入了口琴、三味線以及原住民演唱組合「南王姊妹花」的和聲，讓林昕陽版本的「外婆的澎湖灣」充滿陽光活力，反映出今日澎湖的熱情與活力。
