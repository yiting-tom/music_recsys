Album Intro：

《十級中文聽力 ( 搖滾版 ) 》單曲文案

信老師曾說：
口齒不靈活的 Rocker 不是一個好 Rapper！
信老師還曾說：
現在開始認真學習中國話，未來走到哪裡都不怕！

信老師擔心大家的中文聽力還不夠達標
找來教導主任NINEONE# 一起出題

兩個月前，信 ( 蘇見信 ) 饒舌回歸，推出 2022 年第一首錄音室作品《中文十級聽力》！唱出新花樣，反轉新想像，致力新鮮改造，呈現創意拼貼，以流行風唱出全新音樂樣態。

一開始還叫苦不迭，吐槽Lyrics太難唱，結果錄完流行版，信越唱越上頭，又萌生全新想法，要把歌曲改編搖滾版，希望以搖滾信的姿態，再來和歌迷玩一次“燙舌頭”的遊戲，不僅自己玩上癮，還要和別人一起來玩，最終，信老師邀請到 Rap 組“教導主任”NINEONE# 來一同輔導歌迷繼續苦練《中文十級聽力 ( 搖滾版 ) 》，練不會怎麼辦？信老師和乃主任一起用重搖滾+金屬說唱轟炸你耳膜！

如果說“每個漢字都是一幅畫”，信和NINEONE#合唱的《中文十級聽力 ( 搖滾版 ) 》大概就是筆法豪放、畫風狂野的潑墨山水畫！
