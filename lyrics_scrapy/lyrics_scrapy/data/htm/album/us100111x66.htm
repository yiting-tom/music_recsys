Album Intro：

陳奕迅全新廣東專輯 H3M 由九位戰友一起打造十首全新廣東作品，包括 No. 1 作品 Allegro, Opus 3.3 a.m. 及動人情歌 七百年後、於心有愧、還有什麼可以送給你。

　　H(系)___ H(系)___ H(系)___ M(咩)___？

　　H(系) H(好) H(好) M(賣)！

　　H - Hyper

　　陳奕迅終於推出廣東大碟，並聯同 Moving On Stage 演唱會的大樂隊組成 H3M，Eason 表示與樂隊成員巡迴演出超過一年，十分希望能夠與他們出一張唱片，他說:「希望可以與 Moving On Stage band 做一張大碟，他們每一個人都會為我寫歌，是十分值得紀念，好興奮的事情。」Band 友包括 Gary Tong、Joey Tang、Pat Lui、Davy Chan、Pam Chung、孫偉明、柳重言、黃仲賢及 C.Y. Kong。

　　H - Horrendous

　　2008年整年沒有於專題封面露面，今次終於見面，未知相隔一年，Eason 的尊容有何改變？Eason 早前對封面照大賣關子，表示改變很大卻又很天然，果然，成年冇見過人，當然滿面須。Eason 十分享受拍攝過程，興起跳霹靂舞，又與 band 友齊齊跳高。有不少工作人員取笑未剪須的 Eason，說他像《鹿鼎記》小說中的鰲拜。

　　H - Hybrid

　　Allegro Opus 3.3 a.m.：詞中少將小克及太極級上將 Gary Tong 聯手出戰

　　如此有趣味的大碟，派出第一首主打歌曲 Allegro Opus 3.3 a.m.，真的令人十分期待，由著名漫畫插圖家小克下筆，Lyrics十分有趣，提及一個填詞人心力交戰，嘔出作品前的15分鐘，「唔願意太跟阿 Y 相似 / 又會諗夕爺 / 又怕抄大師 / 太崇拜林振強好點子 / 其實要寫 / 什麼都要試 / 唔知點 / 開始」。歌名 Allegro Opus 3.3 a.m. 完全表達了歌曲的節奏及製作，「Allegro」是意大利文，意思是「歡樂的快板」，而「Opus」是拉丁文，是作品編號之意，作曲和作詞家或音樂出版商會以此編排作品，通常作品編號愈小，亦即是作曲和作詞家愈早期完成的作品。而「Opus 3.3 a.m.」，就是小克詞作改了三次的第三稿，剛剛就在歌凌晨三點三完成，這一切就成了 Allegro Opus 3.3 a.m.。

　　還有什麼可以送給你：Davy Chan 周耀輝混出複雜心情

　　就如陳奕迅所說，這首歌到了三十多歲聽的時候，就會有感覺。「悠然的秋色 / 是否欣賞過都會有點膩 / 然而想不起 / 剩下什麼給你 / 明媚的春光 / 是否不可以年年無限美 / 從此想起 / 遺憾不應給你 為何不捨不棄 / 懷疑不應一起」。大飛的曲總是伴隨著思想，而周耀輝的詞就是優美地勾勒出感情上的迷思，就算給你世界，也許亦不是最好的決定。

　　於心有愧：鍾達茵、林夕夾埋要你慚愧

　　鍾達茵（Pam Chung）第一次為 Eason 作曲，歌曲感情澎湃，Pam 表示作曲的時候，腦海中就是 Eason 於台上縐著眉頭左右搖擺的景象。而林夕的詞更大玩《Six Feet Under》，講到年少氣盛的感情一不小心，可能會累人一世，最後入土未能為安。

　　今天只做一件事：Joey Tang 周耀輝互動純愛表現

　　整首歌曲都有和音伴隨，又有流水聲，是十分柔和輕快的歌曲。花一整天做一件事的專注是現世間難以達到的美事。

　　一個旅人：雷有暉、余紹祺兩個人假日本旅行

　　這首歌有日劇歌曲的輕快味道，充滿旅行感覺，Eason 大叫過癮！

　　七百年後：柳重言、林若寧與《Wall E》談情

　　七百年後滿有活化氣息，活化的卻不是一幢大樓，一個碼頭，或一條街，而是跨越物質，逾越生死的感覺、回憶。即使是一隻被遺棄的破黑膠，歌聲依稀卻可以暸亮得照起生命，情懷如是，日月如是。柳重言的旋律流麗，林若寧的Lyrics意景叢生，一幕幕的畫面，有看過《Wall E》的朋友，應該更有共鳴。七百年後開首慢慢的，如帶人進時光隧道，副歌開始速度增加，回憶突襲，歷歷在目，結尾回復平淡。

　　Life Goes On：孫偉明 DASH 放工過後，生命延續

　　一首放工回家路上聽的歌曲，抬頭望月亮，轉過電燈柱，活在世上也有很多事值得慶賀。孫偉明的音樂加上 Gary Tong 編曲，開首的鋼琴聲就像酒吧的 last order，卻沒有悲哀，是打烊後等待另一個新開始。

　　太陽照常升起：黃仲賢、林夕齊齊掛上太陽

　　不是每首歌都要激昂進場，悲愴離開，太陽照常升起 簡單的曲目，卻愈聽愈有味。林夕的Lyrics亦為此曲生色不少，從香檳開始，喻作喜，啤酒喻為愁，無論發生什麼事情，「車廂也照讓大門開了」，太陽又如常升起，一樣要返工，一樣要嘟咭。

　　不來也不去：C. Y. Kong 林夕啱啱遇著剛剛

　　C. Y. Kong 歌曲的感染力不容置疑，林夕去到化境的Lyrics，令人明白人海每日發生的情愛，緣份使然，剛巧主角是我們，啱啱遇著剛剛，沒有欠誰。「回望最初 / 當喪失是得著可不可 / 可痛若驪歌 / 樂如兒歌 / 像你沒來過 / 沒去過」。

　　沙龍：陳奕迅、黃偉文要拍照的事可不少

　　時代曲的濃濃的情感再度重現，中間一段 solo 帶回最感性的 Eason Chan，親自作曲，找來好友 Wyman 把關。「登高峰一秒 / 得獎一秒 / 再破紀錄的一秒 / 港灣晚燈 / 山頂破曉 / 摘下懷念 / 記住美妙 / 升職那刻 / 新婚那朝 / 成為父母的一秒 / 要拍照的事 / 可不少」。近日 Eason 經常手持相機，把身邊的人事、美事留作紀念，「音樂 話劇 / 詩詞和舞蹈 / 揉合 / 生命 / 千樣好 / 攝入相部」，藝術的可貴之處就是結集生活的精粹，保留人性。

　　M - Maniac H3M 口痕 band 友談 Eason 系系系咩，早前 band 友示範作中，Pam Chung 表示陳奕迅H(謙)___ H(虛)___ H(好)___ M(Man)___？因為她本為 Eason 的和音，現在竟然可以作曲 於心有愧 給他唱，她覺得 Eason 是個大好人，亦好顧及各人的感受，沒有架子。而鼓手 Davy 則認為 Eason 繫好開明，經常會接納別人的意見。
