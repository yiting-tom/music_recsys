Album Intro：

譚詠麟(校長)即將在4月推出全新國語大碟《欣賞》，每一首國語歌都邀請了不同歌手合唱，包括中港台星馬各地校長所欣賞的歌手朋友，當中有很多位都是第一次與校長合作。

在台灣率先推出的單曲《脫胎換骨》由校長和五月天合唱，樂隊出身的校長，很希望在這張《欣賞》專輯中能夠與樂隊合作，亞洲第一樂團五月天是不二之選。五月天表示能夠與校長合作是生涯成就的一個癲峰，校長也很開心終於可以跟五月天合作，他們是校長很欣賞的樂隊，更讓校長憶起夾band的點滴，充滿活力！


不可思議 接近奇蹟
2017難度最高的國語對唱計劃
譚詠麟
全新國語大碟【欣賞】專輯
樂壇校長夢想創舉 召集「欣賞」歌手
中、港、台、星、馬 十位天王天后共同合作

一個籌備了兩年多的計劃，一張大碟內有十個合作歌手單位，這是屬於樂壇校長的一個創舉。

譚詠麟出道四十年來曾與不少歌手合作過，這次在《欣賞》全新國語大碟中，集合了他一直很欣賞、喜歡、和極具份量的歌手們，分別來自五個概括了全球華人所認識樂壇的主要地區，包括內地、香港、台灣、新加坡和馬來西亞。這個計劃可謂一時無兩，最困難之處是時間上的協調，跨地合作，加上與不同歌手討論和錄音時間。校長十分重視這次合作，除了親自監製、作曲外，還擔當聯絡人角色，與不同歌手開會討論，希望把這張重要大碟做到盡善盡美。

《欣賞》中的十首歌曲有著不同題材，校長根據不同合作單位而定，因人而異，做到百花齊放效果。校長甚至發掘歌手們的特性，展現他們音樂上的另一面，帶來新鮮感。校長自己亦嘗試不同唱腔，與各歌手合作產生更多創新元素，遠超校長想像。合作的神奇之處，在於製造不同火花，很多效果都令校長有意想不到驚喜，校長表示：「圓了一個夢想，不是很多人做到這件事。」
這次校長點名欣賞、來自內地、香港、台灣、新加坡和馬來西亞的實力唱將包括
孫楠、譚維維、張學友、劉德華、陳奕迅、五月天、A-Lin、丁噹、陳潔儀和李幸倪，共十個單位，與校長共同演繹愛情、環保、生活態度、人生哲理等題材。

譚詠麟《欣賞》全新國語大碟即將於4月2１日在台灣地區正式推出，切勿錯過！

歌曲簡介:
1.《等一個可能》 唱: 譚詠麟+陳潔儀
曲: 譚詠麟 / 顧芮寧Elaine Koo 詞: 顧芮寧Elaine Koo
編: Johnny Yim 監: 譚詠麟 / Johnny Yim
《等一個可能》是校長在《欣賞》專輯中第一首寫的歌，也是最先完成demo的一首歌。這是一個男方苦等女方的愛情故事，男主角是校長，而女主角是陳潔儀(Kit)。校長直言這首合唱歌需要一把好聲音，而一直以來都渴望跟Kit合作，這次終於成事，就連Kit自己都沒有想過有幸得校長所欣賞。在《等一個可能》歌曲中，校長飾演苦等者一角，演繹情感由無奈到激動。而Kit因為曾迷失在愛，所以一直逃避，更不經意躲開對方的深情眼眸。《等一個可能》滿載感性而迷離的編曲，令兩位主唱者互相牽引，輕易進入狀態，將情感代入其中。

2.《型人道》 唱: 譚詠麟+李幸倪
曲: Johnny Yim 詞: 顧芮寧Elaine Koo
編: Johnny Yim 監: 譚詠麟 / Johnny Yim
校長夥拍新生代靚聲女歌手Gin Lee(李幸倪)，活出屬於自己的型人之道。
網絡世界主導領潮流趨勢，但往往出現很多不負責任的評論，言語批判是十分普遍的事情。要面對流言蜚語，「正面的態度就是最酷的回應」，不要被他人打擊自己，更加「不用刻意和別人共鳴」，即管做最有型的自己！做自己認為值得的事情，給自己按個「讚」，加點信心，要學懂對自己說由衷的讚美，才是「型人道」，校長+Gin Lee將正能量而具鼓舞的訊息發放開去，十分值得大眾學習。

3.《明天何其多》 唱: 譚詠麟+陳奕迅
曲: 譚詠麟 詞: 崔恕 編: Jonnic Lee Thompsett 監: 譚詠麟/ Johnny Yim
「明日復明日，明日何其多」人總會有種惰性，總想著還有明天，還有時間。於是就這樣一天拖一天，明天永遠是今天的藉口。與其沉醉於化成灰燼的過去，或是過度猜疑未來，不如主動出擊，不要只是口講缺乏行動，好好善用時間活出精采人生。以《明天何其多》為名的歌曲，由校長作曲兼執行監製、崔恕填詞及Johnny Yim監製，旋律輕快而且澎湃，當中不乏電子元素，兩位樂壇傳奇巨星充滿默契的演繹，成為一首充滿情感與爆發力的作品。

4.《脫胎換骨》 唱: 譚詠麟+五月天
曲: 五月天冠佑 詞: 林夕 編: 五月天 / 林依霖
監: 譚詠麟 / 五月天 / Johnny Yim
樂隊出身的校長，很希望在這張《欣賞》專輯中能夠與樂隊合作，亞洲第一樂團五月天是不二之選。由五月天冠佑作曲，林夕填詞，五月天和林依霖編曲的《脫胎換骨》是一首全新創作，校長十分滿意而且大讚是非常動聽的一首歌。漫長人生定必經過一個又一個的難關，黑暗過後會更加珍惜現在所擁有的東西，「脫胎換骨」後心境自然大為不同。五月天以慢板的Band sound編曲，令歌曲層次昇華，帶出不一樣味道，其中一句Lyrics「和不如意和平相處」既是點題，也是人生的座右銘。

5.《遊子》 唱: 譚詠麟+A-Lin
曲: 譚詠麟 詞: 崔恕 / 方文山 / 陳少琪
編: Johnny Yim 監: 譚詠麟 / Johnny Yim
對於一直在追尋夢想、離鄉別井的人，聽《遊子》這首歌會特別感觸。校長作曲的《遊子》，具有鄉村民謠色彩，啷啷上口的旋律，很自然浮現在腦海裡。A-Lin具有創造性的聲音，充分演繹出追尋夢想的同時要保持初衷，「思念 在紙上 /隨著落筆 回故鄉 /一句想家寫了 好幾行」正好道出了遊子很想跟家人分享成就。校長很欣賞A-Lin的鼻音和喉嚨音，二人合唱《遊子》擦出精采火花，送給正在努力打拼的人，共勉之！

6.《簡單是福》 唱: 譚詠麟+劉德華
曲: 譚詠麟 詞: 劉德華 編: Johnny Yim 監: 譚詠麟 / Johnny Yim
校長作曲、劉德華填詞，絕對是雙劍合壁，兩位樂壇天王共同演繹簡單就是福！二人同為殿堂級歌手，今次以好朋友對答形式互相呼應，無論人生遇到幾多狂風巨浪、苦戰逆境，都要勇敢面對，學習與困難共處。兩把具有不同特色的靚聲拼湊在一起，令歌曲的層次昇華，在簡單悠然的旋律下更顯非凡。「一切不易不難/ 浮雲從來就是短暫」一句總結，一切取決於心態，其實簡單是福，近在眼前。

7.《強者背後》 唱: 譚詠麟+譚維維
曲: 譚詠麟 / 顧芮寧Elaine Koo 詞: 崔恕
編: Johnny Yim 監: 譚詠麟 / Johnny Yim
《強者背後》由校長和Elaine Koo(顧芮寧)作曲，是一首具有濃厚民族色彩的作品。校長以長輩兼過來人身份領唱，道盡強者背後的寂寞脆弱，說穿了其實「一身華蓋都是枷鎖」。緊接譚維維以暸亮歌聲加入，與校長雄厚低沉的聲音形成強烈對比，將歌曲中段氛圍推至高峰，形成一個強者背後的遼闊壯麗畫面。卸下華麗衣裳做回自己，我就是我，不難發現傷口的痕跡。

8.《強求好嗎》 唱: 譚詠麟+丁噹
曲: 顧芮寧Elaine Koo 詞: 顧芮寧Elaine Koo
編: Johnny Yim 監: 譚詠麟 / Johnny Yim
被校長稱為「女版陳奕迅」的丁噹，曾經在活動後台與校長有一面之緣，想不到二人很快就有非常難得的合唱機會。丁噹有能力把歌曲唱得出神入化，校長與Johnny Yim共同擔任《強求好嗎》的監製，二人對丁噹的得心應手的聲音十分滿意。若果一段愛情長跑已去到瓶頸位，或許將故事改寫，彼此會有重生的可能。校長與丁噹在情感上發揮得淋漓盡致，將故事娓娓道來，這是有經歷和感觸過後的人才可產生的共鳴。

9.《觀》 唱: 譚詠麟+張學友
曲: 譚詠麟 詞: 陳永鎬 編: Johnny Yim 監: 譚詠麟 / Johnny Yim
同樣是校長創作的歌曲，陳永鎬填詞，邀得另一天王張學友共同演繹，唱出無盡、淨心境界。歌曲開首以一段流水聲引入，營造出山水花草的無涯世界，人在當中只是一顆微麈。校長與學友以聊天唱法，如你一言我一語，歌曲營造的回音(echo)效果，猶如一個虛幻朦朧的大觀園。只要心是無盡，不論唯心所做，或淡然放下，一切皆掌握在手裡。尾段二人一唱一和，伴以和唱的經文，具畫龍點睛的效果，歌曲意境更為圓滿。

10.《地球人醒來吧》 唱: 譚詠麟+孫楠
曲: 譚詠麟 詞: 譚詠麟 編: Johnny Yim 監: 譚詠麟/ Johnny Yim
以環保為主題的《地球人醒來吧》，由校長包辦曲詞，他找來中國最好男聲孫楠，共同將環保訊息宣揚開去，透過音樂向大眾灌輸正確知識。「不愛這地球 /等於背叛地球/ 人類把地球出賣」，簡單到位的Lyrics，讓聽者反思自身日常行為。校長希望透過這首作品，令大家更加珍惜大自然。激昂而有層次的旋律，配合兩位靚聲之人的演繹，各有千秋，時而澎湃的歌聲非常搶耳，令人忍不住跟著一起為環保發聲!
