【黑色彩虹】

你以為你懂得她？
有著精緻美麗的臉蛋、性感出眾的身材，總是笑意盈盈、熱情活潑，能歌善舞炒熱了周圍的氣氛…，我們眼中的蔡詩蕓，總是輕易成為大家目光的焦點，在女人羨慕與男人愛慕中，她彷彿得天獨厚地擁有著一切，我們也常猜想著：這樣的女孩，她的愛情她的世界，一定就像彩虹一般絢爛夢幻吧。

你以為你懂得她。
你以為她像洋娃娃一樣，可以一眼就看穿。
但是這一次，蔡詩蕓要打破你的想像，讓你知道，你對她的幻想，少了一點黑。

10首好歌全推薦----

1、【你知道這不是愛】詞：崔惟楷 / 曲：蔡詩蕓
感覺不在了，不如就微笑說再見！
蔡詩蕓2012年最灑脫率性的創作，要你坦率地面對自己、面對愛情。

2、【舞夜】詞：崔惟楷 / 曲：蔡詩蕓
別浪費此刻的感覺，不留白今夜的精采。
蔡詩蕓忘了時間、狂歡破表、電力十足的Party必備high歌！讓每個人都忍不住跟著快樂舞動。

3、【雨過後的風景】詞：崔惟楷、甘世佳/ 曲：蔡詩蕓
愛情離開了，留下來的人只能勇敢學著雲淡風輕。
蔡詩蕓的傷感情歌，整首歌滿溢一股心痛哀傷的情緒，陪你一起等待雨過後的風景。

4、【黑色彩虹】詞：黃文萱 / 曲：梁永泰
愛情像彩虹一樣，不能預測它何時出現或消失，它美得讓人迷惑卻同時也虛幻難以把握。愛上一個不應該愛的人，也許就像是天空裡出現了一道彩虹，它那麼燦爛美麗，卻是黑色的。
這首「黑色彩虹」編曲上一開始用鋼琴和弦樂鋪陳了一個寂寞感傷的世界，音色很輕柔、情緒卻滿滿地讓人難以呼吸。隨著音樂行進，漸漸加重渲染，釋放出心裡的痛。就像Lyrics中詩蕓面對愛情的態度--「就算會很痛，我也會捨得墜落」。

5、【吸血鬼】詞：陳妍希 / 曲：蔡詩蕓、梁永泰
「愛要願意 流乾所有血 / 只為一秒鐘的交會 / 就算全世界都毀滅 / 也值得 去追」
她的美麗很絕對，她的吸引力會致命，你敢不敢為了愛情，義無反顧縱身一躍？

6、【Hello Tomorrow】詞：王雅君 / 曲：蔡詩蕓、梁永泰
向過去告別，往明天跨越，只要我們夠勇敢，願望一定都能實現！
蔡詩蕓用音樂陪著你，一起Hello Tomorrow！

7、【你不懂】詞：王雅君 / 曲：蔡詩蕓
有時候，沒人懂的沉溺也是一種愛。
不論那個人已經走得多遠，歌還唱著、故事還沒完結，就有勇氣永遠站在原點。
蔡詩蕓2012年最深情的傻情歌，在虛擬的幸福中陪你等待愛。

8、【想像愛】詞：Ping / 曲：倪子岡
在每一個轉彎之後，都可能有一場浪漫的邂逅。
蔡詩蕓用天馬行空的奇想，勾勒出每個女孩對愛情懷抱夢想、依然憧憬的心情。

9、【影】詞：Ping / 曲：Skot Suyama、蔡詩蕓
一段很深又很痛的戀愛，會在生命中留下不會消失的烙印。
不管過了多久，走了多遠……它永遠如影隨形。

10、【藏起來的愛】詞：崔惟楷 / 曲：Skot Suyama、蔡詩蕓
「有些故事 結束太可惜 / 有些親愛的人 原來無法代替」
對的人在錯的時間遇上了，即使後悔、即使遺憾，有些愛只能是藏起來的秘密，逝去了就不會再回來。
帶著一絲懺情的味道，聽蔡詩蕓唱出面對生命中錯過的愛情，她的不捨與感慨。
