# 一樣 The Same
# 愛，本質無差別
# 把信仰還給愛


繼2016年張祖誠號召群星為流浪動物發聲，感性寫下《給牠一個家》後，今年母親節一曲道出愛要及時的《時間，請你走慢一些》，描寫『子欲養而親不待』的殘酷現實，引人熱淚滿眶，17年夏天張祖誠再次為愛發聲，寫下這首《一樣》，為所有曾為堅持所愛而磕絆受傷的人歌唱。
《一樣》由張祖誠譜曲填詞，嵐澄創藝的張博彥編曲與製作，歌曲為世上所有受歧視的弱勢族群發聲，無論是LGBTQ，或者任何因喜好而不受社會大眾所待見的群體，都不該因為『愛』而受到排擠與歧視。恐懼始終來自於不了解，在除去所愛的對象後，愛的本質並無差別，它是與生俱來的能力，也從來都不是種選擇，對愛的信仰不該是作為攻擊的武器。
身邊不乏同志朋友的張祖誠在一個個陪伴他們分享心中委屈與苦悶的徹夜暢談後，心有所感的在某個失眠的夜裡，譜出這首曲子。歌曲以清脆的鋼琴聲拉開序幕，張祖誠以直接而未有多餘技巧的唱法，道盡必須躲避燈光埋頭行走於黑暗街巷的孤獨感，只因追愛的對象差距，便被社會粗暴地以標籤化的方式歧視與攻擊；當鼓點漸漸滲入，是祖誠想要給所有迷途於自我與社會期待的人們一劑穩穩的強心針：『這條路始終不是你們獨自走著，請接受最完全而真實的自己，那就有如浪漫的彩虹，美麗而溫柔。』心因性的孤獨需要更多的溫暖與熱情來融化，為此張祖誠與製作人為編曲做了諸多努力，歷經許多版本才確定了現今的輕搖滾風格，為了便是給予這些在追愛路上受傷的人們，更多的支持與安慰，『你們從來都不孤獨，就勇敢地褪去所有偽裝，像彩虹般綻放吧』。
象徵多元的彩虹始終指涉著多元性，也是這個社會不斷追尋的美好價值之一，然而卻依然有人因他人的不同而給予負面標籤，歌曲命名為《一樣》，也是為了展現所有不同取向的人群在愛的本質上皆為相同，更何況他們也需要經歷生活與工作的壓力，張祖誠以陪伴與支持的心情寫下了這首歌曲，也作為即將上線的短片《Julia》的主題曲，影片闡述受排斥的變裝男子John所遭受到的攻擊與排斥，影片即將於盛夏上線，敬請期待。
