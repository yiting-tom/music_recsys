# 新春輕快歌〈神隊友〉 幽默無厘頭超神正能量
# 電子搖擺搖滾快活曲風 金曲最佳單曲製作人陳君豪操刀
# 卡祖笛俏皮開場 奠定慢慢說獨特音樂符號

金曲獎最佳演唱組合— 慢慢說（主唱利得彙、吉他手沈志方），將於2019年2月推出第三張全新專輯。首發心碎「慢」歌〈自導自演〉，也是超火紅電視劇《延禧攻略》片頭曲，在2018年底天后專輯齊發大亂鬥的強敵環伺下依然保有一席之地，在兩岸數位音樂平台深獲好評，被歌迷們譽為「年度虐心之作」，也勾起許多人紛紛寫出在過往感情回憶裡自導自演的心情留言。第二發開心輕快歌〈神隊友〉，重新召喚充滿「慢慢說式」獨特幽默無厘頭的詼諧曲風，為2019 一開年的華語歌壇注入快樂爆表的超神正能量！

電玩感歡樂助攻神曲 〈神隊友〉，電子搖擺搖滾 Electro swing rock 快活曲風，由利得彙 + 沈志方詞曲創作，金曲獎最佳單曲製作人陳君豪操刀製作，三人超級默契的音樂神隊友，鬼點子完全不設限發揮創意，神激盪出許多天馬行空的音樂新花火。〈神隊友〉編曲加了許多電子元素，樂器音色上以取樣 Sampling 技術經過處理、重建再運用別具特色。前奏則與第二張專輯收錄的〈喜歡待在家〉異曲同工，都設計一段卡祖笛 Kazoo 充滿喜劇電影感的俏皮旋律作開場，由利得彙親自演奏，讓「卡祖笛」奠定出代表慢慢說獨特的音樂符號。


# 電玩助攻神曲〈神隊友〉 大玩網路流行語
# 利得彙熱衷電玩 直播實況開線上音樂會
# 電玩如人生 一往無前才是做人真氣魄
# 感性主唱 + 理性吉他手 音樂上「最佳神隊友」

〈神隊友〉歌名與Lyrics充滿許多好玩逗趣的電玩與網路流行語，包含了凱瑞 carry、666、大腿、開掛、放大絕等等，源自於他們熱衷網路社群，利得彙喜歡電玩，也是遊戲軟體影音串流平台Twitch知名實況主，兩位常常會在直播間分享互動遊戲和開線上音樂會。〈神隊友〉Lyrics鋪陳在組隊打怪的電玩風格裡，像是夥伴之間的對話彼此打氣，但利得彙更想表達的是「電玩如人生」，許多做人態度都藏在其中。尤其是網路上常說：「認真就輸了」，但她這句話卻有不同見解：「因為這樣一來反而讓很多人不敢認真了，因為怕輸，卻忘了一往無前才是做人的氣魄和精神，這才是勇敢。」

對於心目中「神隊友」的定義，他們不約而同都認為：「每個人都有長處，補足隊友的短處。你沒有的，我來。更重要的是凱瑞 carry 對方後也不會流於表面的優越感。完全是義氣相挺很甘願，彼此幫忙不求功，才是真隊友。」他們互補角色是一個主唱加上一個吉他手，性格裡一個感性創作加上一個理性架構，不管是音樂上或生活上都是彼此的「最佳神隊友」。
