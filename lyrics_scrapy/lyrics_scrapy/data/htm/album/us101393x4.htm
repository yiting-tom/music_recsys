想念你 想念你的 Saya

　　好的歌聲不應該被埋沒

　　蟄伏兩年的等待 一個渴望被你重新聽見的歌聲

　　今年冬天想念凝結 只為聽見一首感動的歌

　　2005年冬天，想念凝結，只為聽見一首感動的歌—電影《世上的另一個我 Nana》主題曲中文版「想念你的歌」！Saya 發揮與生俱來的完美歌喉，淋漓盡致展現更具包容力的轉音唱腔，是近期難得聽見的超強好歌。

　　等待兩年 她的歌聲 渴望重新被你聽見

　　想念「唱歌」的單純感動化成「想念你的歌」

　　2003年「SOLO」專輯發片之後，Saya 並未停歇她在演藝圈裡努力的腳步，不斷累積實力，連續在「名揚四海」、「愛情風暴-美麗99」、「偷心Ⅱ」、「偵探物語」等電視劇中演出，更以「名揚四海」獲得金鐘獎最佳女配角獎的肯定。透過戲劇，看見 Saya 對戲劇的努力與成績，而一支支的廣告代言，更讓 Saya 變得家喻戶曉。然而，雖然在演藝成績上逐漸有所斬獲，但這些對 Saya 來說，都只是她多方面嘗試的收穫。在 Saya 的心裡始終覺得，她最想做的還是唱歌、跳舞。在她的血液裡，流動著渾然天成的歌唱本能，只有放聲歡唱的時候，才是最貼近自我、最開心的時刻。

　　蟄伏就是準備，時間就是考驗。由於遇到唱片公司內部結構調整，Saya 的發片日一延再延；2年的等待，Saya 曾經一度懷疑，是不是自己的歌聲將就此石沉大海，今後只能繼續在戲劇、廣告上努力，再也不能在自己最愛的歌唱舞台上盡情發揮。這些日子裡，天知道 Saya 有多麼想念能夠唱歌給大家聽的快樂。幸好唱片公司並沒有放棄 Saya ，終於在2005年底，讓我們再度聽見 Saya ，聽見「想念你的歌」。
