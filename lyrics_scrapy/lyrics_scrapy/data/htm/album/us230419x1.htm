先發投手小春 Kenzy 率先上場 頑童兄弟背後撐腰
廟宇文化 x 熱血棒球 x 好友助陣 x 提攜傳承
《啥款》EP超殺展示人生態度與音樂樣貌！

在小巨蛋爆滿的舞台上，頑童 MJ116 三人首度宣布單飛不解散計畫，引起聽眾與媒體的好奇與猜測。但對從組團開始感情就好到像親兄弟般的頑童來說，其實想法很簡單：當三人合在一起，就是頑童；當三人分開，從小春、瘦子到大淵，都有各自獨特的人生歷程、喜好個性和音樂想法，就會產生不同的音樂元素與創作的全新樣貌！而當三人分別發展自己的音樂特色後再度合體，也可能催生出完全不同的頑童！

從這個概念出發，頑童單飛不解散的計畫，在《董Don》專輯中初步嘗試三人各一首的 Solo 曲目《地痞》、《歹性底》和《質感老鄉》後，決定大膽放手執行！經過了兩年醞釀、三人各自嘗試音樂可能性一年的過程，終於派出先發投手小春，以 Project Kenzy 為名，為華語音樂圈投出了第一發行雲流水且華麗的變化球！

Project Kenzy 《啥款》EP，以小春 Kenzy 的口頭禪「啥款」為名，用全裸展現刺青的超殺樣貌為設計，充分展現要將 Kenzy 的一切展現在世人面前的決心！而專輯中的每一首曲目，都經過精心設計與挑選，只為準確表達 Kenzy 的人生歷程與獨特想法。從第一首《沒在怕的》就訴說自國小開始打棒球擔任投手的 Kenzy ，到長大後選擇了音樂道路，卻仍只要有空就奔往球場的熱血情懷！與中國有嘻哈冠軍 GAI 合唱的《保險櫃》，則展現了兩人從默默無名時開始建立的深刻友情，也隱含了兩人對嘻哈文化的定義與傳承意味！而同樣的傳承意味更展現在 Kenzy 與才廿歲的嘻哈新銳 K9 合作的《啥款》上，透過與年輕嘻哈世代的交流與共同創作，讓 Kenzy 學到許多新的想法與概念！最後則是以 Kenzy 從小接觸的廟宇傳統為文本，與李英宏共同描繪華人《香火》傳承文化，其中深刻而細膩的家庭情感描述，結合 Kenzy 流在血液中的陣頭節奏，譜出一首前所未見的嘻哈新樣貌！

首發即交出一張漂亮成績單的 Kenzy，談起這次 Solo Project 的點滴過程，不禁坦言一直以來都一起創作的三人不在一起時，心中還是會有許多不安與不確定性。而當各自創作到一個階段聚在一起時，才發現三人心中竟然都有一樣的想法！於是頑童三人決定互相參與彼此 Solo Project 的曲風、藝人定位、視覺與製作等決策，讓 Kenzy 像吃了一顆定心丸一般，勇敢地走出自己音樂的方向！Kenzy 說到：「因為我們能給彼此自信與力量，才能在一起這麼久！」這樣的過程也讓三人更加明白彼此的重要性，更期待三人各自在 Solo Project 成長後，再度重組的頑童，將會有怎樣驚人的音樂能量與轉變！

帶著《啥款》EP的概念， Kenzy 希望對每一個聽眾表達出，每個人都是獨一無二的存在，不管你的成長背景、文化傳統、個性態度、朋友家人是什麼樣子，都該不斷檢視形塑成「你」的每一個重要元素。而這些經歷、情感、機緣、省思，就是該不斷努力前進，甚至用力傳承下去的重要精神。透過音樂， Kenzy 完成了屬於他的傳承；透過人生，不管你是「啥款」人，都應該勇敢去找尋屬於自己的生命樣貌。也許在你踏出第一步後回頭望，才發現一路陪你走來的夥伴，就像頑童的音樂一樣，會始終在背後給你自信與力量。

先發上陣！棒球男孩 Kenzy 熱血吶喊《沒在怕的》

從國小就開始打棒球擔任投手的 Kenzy 小春，一直都是個熱血的運動少年，國中打了三年校隊的他，當年憑藉實力考上棒球強校強恕高中，但教練看到小春的刺青卻有些歧見，拖到開學前一兩天才通知小春報到。當時小春已在爸爸的安排下註冊了喬治高職，爸爸就下定決心不再安排轉校。沒想到這機緣竟讓小春後來在喬治高職遇到瘦子，開始接觸饒舌，並邀集大淵一同組成頑童，就此踏上音樂創作的道路！只因為當年的一個機緣與決定，一眨眼過了十五年，頑童一舉從街頭唱進小巨蛋，站上金曲獎的舞台成為嘻哈天團，讓人忍不住讚嘆命運安排的巧妙！

踏上音樂道路的 Kenzy ，並沒有因此放棄對棒球運動的熱愛，只要一有空就戴上棒球手套往球場跑，有任何比賽機會，只要他時間可以一定參與！此次因應世界 12 強棒球錦標賽， Kenzy 以一路以來對棒球的熱情，創作了這一首無所畏懼的震撼歌曲，激昂的旋律加上滂沱氣勢的軍戰鼓， Kenzy 吶喊出奪冠登王的決心！能將自己熱愛的兩樣事物結合在一起，Kenzy直呼創作過程感動又過癮。

對棒球的熱愛也讓 Kenzy 結交到一群喜愛棒球的好朋友，在 MV 中不但號招了平常一起打球的球友來擔任隊友，更請到擔任大聯盟投手的王維中一同參與！沒想到拍攝時王維中只是輕輕投球，就讓捕手接得辛苦，直呼大聯盟職業級的果然不一樣。一群人從晚上九點打球直到日出，充分享受打棒球的樂趣外，也再度了解只要有一群志同道合的夥伴一起努力，不管遇到什麼困難都《沒在怕的》！

廟宇文化混搭嘻哈音樂 一炷《香火》傳承世代情感

從最早的記憶開始，手上似乎總有阿嬤掛的一串護身符；家人拿香祈禱的背影歷歷在目，卻不明白他們在呢喃些什麼。從小在廟宇文化中長大的 Kenzy，三天兩頭混在廟口，參加八家將只因為好玩，直到長大離開家、玩音樂、功成名就後回頭看，才發現廟宇文化一直都存在他的血液中，一炷香火的信仰從未離開他的身體。

決心找回信仰的 Kenzy ，不但連續數年號招團員跟著媽祖遶境，更在三年前請回一尊挪吒三太子，成立屬於自己的廟「汝南堂」，還找來大淵當主任委員。談起廟宇文化， Kenzy 眼神發亮地說著各種官將神將的臉譜、媽祖廟香旗上的工藝，認為這些都是台灣很獨特的藝術文化，每天都在我們身邊，但對大多年輕人來說卻很陌生，是一件非常可惜的事。

直到近年才在廟宇文化中找到「傳承」、「延續」意涵的 Kenzy ，邀來具有「台帥」風格的李英宏助陣，一同深刻書寫一幕幕從小到大的香火回憶。以順口的嘻哈與R&B 入歌，行雲流水的旋律線配上台語國語交雜卻毫無違和的Lyrics，加上編曲中陣頭文化傳統器樂的聲響，彷彿帶著聽眾站在人生的十字路口、生活廟口，看著官將神將活靈活現地在眼前舞動上演著。廟宇文化與嘻哈音樂的另類結合，除了精準描繪著 Kenzy 與許多台灣年輕人的成長背景外，更帶著期望這樣獨特的台灣文化，能像一炷《香火》般，一首一首唱下去，一代一代傳下去。

嘻哈的態度就是這樣！Kenzy feat. GAI的超屌《保險櫃》

從一無所有到名利雙收，裝滿保險櫃的不只是滿滿的錢，更是硬派的嘻哈精神！頑童小春化身 Kenzy 與中國有嘻哈冠軍 GAI 首度攜手合作，在嘻哈音樂圈投下一顆超殺震撼彈。從兩人一開口就充分展現嘻哈態度，犀利的語句直逼聽眾別整天哀聲又嘆氣，一路苦過來的他們靠著天賦加努力，才能裝滿心中與手中的《保險櫃》。

雖然是首次合作，但 Kenzy 跟 GAI 早已相識多年，在Kenzy 開始玩饒舌音樂後，喜愛廣交朋友的他想知道各種不同的文化與想法，不斷與各地區的 Rapper 交流。在一次想介紹重慶的饒舌來台灣的過程中。結識了當時還尚未出名的 GAI，兩人一拍即合、惺惺相惜，約定好有天一定要一起做一首歌。

在 Project Kenzy 中兩人以《保險櫃》為題，述說一路走來的堅持與精神，更期望將這份眼界與魄力，透過這首歌傳承下去。《保險櫃》的意義不是要炫耀自己有多少財富，而是希望把擁有的資源與精神都存起來，照顧像當年的他們一樣默默無名的年輕饒舌歌手。打開兩位嘻哈高手的《保險櫃》，淋漓暢快的節奏與快嘴饒舌的高超功力迎面而來，搭配超殺超猛的旋律線，人生體悟與人情冷暖盡在其中！

是打招呼也是烙狠話 用《啥款》帶 K9 來跟你們見世面

《啥款》是 Kenzy 的口頭禪，意思接近黑人口中的”What’s up”，可以用來當作見面的打招呼語，也可以在耍狠時使用。從小受台語文化薰陶的 Kenzy 一直認為，台語中有許多用法其實很有趣，而在充滿西方元素的嘻哈音樂中，加入充滿本土意涵的詞句，是華語嘻哈應有的融合與創新。

Kenzy 也延續了頑童熱於分享、傳承機會的精神，以這首《啥款》與年僅廿歲的饒舌歌手 K9 合作，就像當年熱狗一手帶著還默默無名的頑童一般，期望熱愛饒舌又有才華的年輕人，能有機會在舞台上展露頭角。 Kenzy 與 K9 的爸爸因廟會活動而認識，提到兒子喜愛饒舌，就將 K9 託付給 Kenzy。年紀相差十歲的兩人在共同創作的過程中，不但沒有世代的隔閡，還彼此學習到了許多新的想法與創意。兩人第一次合作就完成了這首以電子音色為基底，超生猛直率的《啥款》，以犀利快嘴的饒舌和初生之犢不畏虎的姿態， Kenzy 帶著 K9 一起見世面，對大家狠狠地打聲招呼：《啥款》
