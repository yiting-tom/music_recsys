Album Intro：

HAOR許書豪 2019 創作專輯：Sound Lab 1《真心話冒險王》

【聲音實驗家 Haor 許書豪 致力聲音的精密研究，對聲音有著無遠弗屆的深度感受力與想像力。】
經過金曲獎的洗禮之後，耗費近一年時間重新探索自我，他問著自己「我追求的音樂究竟是什麼」「代表我的音樂又會是什麼」。他秉持勇於實驗的精神，嘗試截然不同的音樂創作過程，大膽且細膩的「採集」、「探索」、「挖掘」這個世界的聲音，融入創作之中，對他而言每一顆音符、每一聲聲響、每一句話語都在捕捉著難以言喻的真心話。新專輯Sound Lab 1《真心話冒險王》宣告展開一場盛大的聲音實驗。

【實驗精神，玩心大開。挑戰自我，突破極限。】
＊潮音才子 x 不敗女神 ＃ 許書豪 x 徐若瑄 激盪時空 經典之作《但願人長久》
＊金曲黑馬 x 金曲歌王 ＃ 許書豪 x 蕭敬騰 火力四射 震撼新作《別再叫我哥》
＊聲音實驗家 x 情歌王 ＃ 許書豪 x 陳威全 情感宇宙 爆發力作《真心話冒險王》

Fusion渴望 x 電子勇氣 x 民謠執著 x R&B浪漫
Haor 許書豪 Sound Lab 在一連串的自我辯證與音樂實驗中正式成立，蛻變後的首張作品《真心話冒險王》跨越舒適圈，展開多方合作，挑戰自我極限，穿梭城市狂響。為了全面實踐不被眼前所知所限的實驗精神，專輯視覺是實驗大膽排版，分割色塊表現鏡射的錯視維度，象徵書豪在平行宇宙中，冒險穿越時空找尋自我。《真心話冒險王》這張專輯就像是Haor 許書豪充滿無限潛力的能量魔方，帶我們聽見時間走過的聲音、萬物在空間裡的迴響以及我們蛻變的模樣。

《但願人長久》
＊潮音才子 x 不敗女神 ＃ 許書豪 x 徐若瑄 激盪時空 經典之作
＊實驗元素：潮音電子 x 經典之作 x 穿梭時空 x 無盡思念
「是什麼樣的力量，能超越時空的限制，將兩個人緊緊相繫在一起？」為了探究這樣的真理，Haor 許書豪大膽挑戰經典改作《但願人長久》，創作嶄新的旋律Lyrics將經典巧妙融入其中，音樂上將琵琶融入電子R&B X EDM X Future House，時間就在這首歌裡突然間拉開了數千年。在如此遙長的思念之中，Vivian 徐若瑄內外兼具古典現代的美就成了力邀合作對唱的不二人選，而Vivian在接獲合作邀請時，二話不說情義力挺獻聲作詞，潮音才子Haor 許書豪與不敗女神Vivian 徐若瑄，攜手詞曲編曲演唱從一而終實驗混搭，創造新世代經典對唱情歌。

《真心話冒險王》
＊聲音實驗家 x 情歌王 ＃ 許書豪 x 陳威全 情感宇宙 爆發力作
＊實驗元素：Urban R&B x 真心冒險 x 心電圖音色
「音樂 x 情感」產生無限融合變化式的化學反應，聲音實驗家Haor 許書豪遇上情歌王V.C.陳威全，聯手帶來2019 Urban R&B 必聽代表作《真心話冒險王》。作為書豪專輯名稱的同名歌曲，《真心話冒險王》取樣心電圖音色，將創作以及實驗動機精確定義：「真心 x 冒險」是驅動人持續更好的動能，進而帶出整張專輯都是跟「真心」密不可分的歌曲。不論你說的是真心話或是最逼真的謊話，都將迎來一場意料之外的大冒險。

＊Haor許書豪 ＃好人們，必聽專屬情歌三部曲。
第一回合「鍛鍊勇氣」：該如何揮別自始至終的「男二」情節？【不好說】就不要說了吧！
第二回合「勇敢行動」：【別再叫我哥】，我比較想當情人。
最終回合「愛如何愛」：當你最好的朋友，在你身邊默默守候【不再打擾你】。

《不好說》
＊Haor 許書豪「你的好人」首部曲
＊實驗元素： 好人曖昧 x Neo Soul x 音樂全開
許書豪詞、曲、編、混 音樂全開再炸一波，以擅長的Neo Soul風格為底，融合了Jazz、R&B、Chillhop等元素。「聲音」、「編曲」、「演奏」與「演唱」融為一體，『先倒杯酒欣賞夜的美』的時候能聽到倒紅酒的聲音；『明天我們請個假』之前的過門能聽見汽車發動引擎聲，直接用聲音讓你感覺出去玩的開心畫面，每個樂器角色在歌曲腳本中充分發揮。繼上一張專輯中《過一陣子打你》、《備胎》等自始至終的「男二」情節，這次秉持內斂、曖昧的態度，不好說就不要說了吧！轉個話題，要不要明天我們請個假去看晚霞？

《別再叫我哥》
＊金曲黑馬 x 金曲歌王 ＃ 許書豪 x 蕭敬騰 火花激盪 震撼新作
＊實驗元素：好心誠懇 x 玩心大發 x 金曲火花
從蕭敬騰擔任金曲主持人爆笑訪問最佳男歌手入圍的許書豪開始，許書豪除了金曲大黑馬、音樂全才生外，多了一個親切的「哥哥」稱號，更在金曲會後直播中，利用老蕭喊他哥哥的趣味訪問片段，即興創作了「哥葛Song」另類合唱老蕭，但歌迷呼喊不夠過癮，書豪2019玩心大發，利用直播當時即興創作的和弦、速度、旋律和音調，連結創作當時的新鮮感，呼應書豪一直以來從生活中取樣的創作風格，也依然一手包辦詞、曲、編曲，創作了一首狀似輕鬆談笑，卻掏心掏肺的誠懇之作《別再叫我哥》。

金曲會後，首次正式與老蕭合作，書豪坦誠有許多忐忑，但第一次碰面，老蕭就爽快直接進錄音室「撞一個」，間奏鋼琴solo更是老蕭即興現場創作和演奏，爽快拋出許多音樂上的交流，更俐落一次到位的錄完Vocal與和聲，透過音樂，默契和熟悉度毫無暖場寒暄的需要，全都是感受音樂當下最直接的火花！

《不再打擾你》
＊「你的好人」三部曲，毫無保留地愛一個人，化身「你的好人」最佳代言人
＊實驗元素：好人等待 x 期待回報 x 時鐘走動的聲音
當一段感情裡的好人，並不是怯懦的表現，而是在感情的牽絆中找到了一處秘密基地，在掙扎的痛苦中領悟如何去愛一個人的道理。在期待擁抱與不求回報之間，為了心裡最重要的人付出，同時也給了自己一處喘息的棲身之所。為了傳遞《不再打擾你》歌曲中，你我也曾有過的情景：一個人握著手機，在深夜裡漫長等待那個重要的人回應的感受。編曲的主軸線融入了「時鐘走動的聲音」，我們聽得見時間走過的聲音、遺憾在空間裡的迴響、以及身為哥的心裡，努力將最不好說的真心話，說成了：「不再打擾你」。他說：喜歡並不是獨佔的擁有，而是兩個人都能在這一段感情的過程中變得更好。

《聽不見你》
＊聲音實驗家 Haor 許書豪 x 電影詩人 Li Sean 李洹宇
＊實驗元素：聽覺 x 證明 x 存在 x 孤獨
「一旦證明了聽不見你，也就證明了我真的失去了你。」Haor 許書豪賦予情歌不屈不撓實驗精神 ：「看不見的如何證明存在過的可能？」音樂中設計了很多電子音色來取代以往的弦樂鋪陳，營造出現代城市的孤獨感。書豪說他對孤獨的想像畫面是一個人在熱鬧的餐廳吃飯，再一個人回到空蕩的家看電視，整首歌的最後取樣關電視的聲音。為了展現《聽不見你》歌曲精神，書豪攜手新銳導演Li Sean李洹宇遠赴日本仙台取景，踏上一段證明失去的旅程，用「孤獨的結束」迎向「未知的未來」，攜手詮釋視覺聽覺裡失愛者無垠無根的孤獨感。Haor 許書豪代替念舊的我們發問：「擁有的夢越美 如何承認 它消失不見」 從聽覺中，尋找愛與被愛的痕跡。

《就讓它漸漸死吧》
＊直指人心剖析時下私房愛情觀 暗黑系R&B死心力作
＊實驗元素：謊言 x 逞強 x 灑脫 x 如果
我們都有過一段不完整的愛情，也許是一廂情願，也許是曖昧，也許是把友情誤以為成愛情的美麗誤會。正因為這樣沒有開始的最美好，沒有曾經的最嚮往。說得一派雲淡風輕，會慢慢過去的、會漸漸忘記的，但其實只有面對在意跟無法忘記的才會說：「我會忘記，一切會過去。」許書豪將這首歌的創作概念定義在謊言，講述一個自己想說服自己的謊言。歌名「就讓它漸漸死吧」中的「它」代表的是回憶跟記憶。希望能讓揮之不去的回憶跟記憶漸漸枯萎死亡，從愛情中的難過跟失落之中完美轉身。 手機裡的照片跟訊息都是觸動回憶的證明，在編曲中加入了手機關機的聲音，象徵著悲憤地關掉手機做個了結。但就算對回憶唱得灑脫，終究只能用謊言和逞強的自我對話，既然一切不能夠從頭，就讓它漸漸死吧。

《半杯水》
＊音樂推向潛意識 同時扮演心理的矛與盾
＊實驗元素：半杯水 x 慾望 x 選擇
從一滴水滴滴落的聲音揭開這場心理實驗的序幕：遊走在無際沙漠的你，酷熱難耐寸步難行，眼前突然出現半杯水，你的心中會浮現什麼樣的思維？是「太好了，竟然還有半杯水」，或是「太可惡，誰喝掉了半杯水」。Haor 許書豪將音樂推向潛意識，同時扮演心理的矛與盾，「夠與不夠」的辯論交錯迴響在其中。書豪蒐集「杯 x 水」能創造的各種聲音，倒水、水滴到各種容器的聲音，像水一樣溫柔包容你，同時刺激你感官的最飢渴感受。

《阿君仔》
＊聲聲呼喚北漂遊子「返鄉潮已經到來」
＊實驗元素：東石海港 x 遊子時代 x 無根流漂
東石海港的海浪聲搖晃著熟悉卻又捉模不定的樂句，像是聲聲呼喚北漂的遊子「返鄉潮已經到來」。Haor 許書豪在《阿君仔》這首歌裡，將對音樂的執著與歌曲精神「認份踏實的力量」揮灑得淋漓盡致，考究經典歌曲《青蚵仔嫂》的時代背景，取樣青蚵仔嫂的歌聲、處理蚵的程序時傾倒蚵殼的聲音、工廠馬達啟動的聲音、象徵時代奔波的腳踏車鍊的聲音，在平交道警示音的催促聲中，敲響心中的嚮往，指向歸途。

《城市狂響》
＊整座城市都是我的聲音實驗室。
＊實驗元素：城市狂響 x 天旋地轉 x 真心話冒險王
聲音實驗家 Haor 許書豪全新創作專輯開門曲《城市狂響》，狂想率性地撐開天與地，展現真心話冒險王的堅定決心。呼應專輯 Sound Lab 的概念，不限於精密之地、不限於與世隔絕之境、不限於宇宙行星，只要帶著吉他和麥可風，整座城市都是他的聲音實驗室。歌曲中精彩的甩尾聲響由書豪親手在賽車場錄下，為專輯劃開最精彩失控的天旋地轉，譜寫2019全新豪語錄：「大不了偏離航線再重新定位」，邀你一同加入他的Sound Lab，迎接最痛快瘋狂的《城市狂響》。
