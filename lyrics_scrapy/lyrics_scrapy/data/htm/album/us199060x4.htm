婁藝瀟從來都是令人驚豔的那一個，她是《愛情公寓》裡機靈古怪、廣受觀眾喜愛的胡一菲，她是當下熱門綜藝《跨界歌王》裡唱功一流的“靈音唱將”，也是近期大型音樂劇《周日戀曲》裡一次連唱24首歌曲的當家主演。她會鋼琴、吉他、會跳舞、做得了詞也合得了音，不同的人可能因不同的方式認識她，但都會喜歡上這個才華與美貌並存的全能實力派。

綜藝節目讓婁藝瀟初次顯露了她的出色唱功，但其實本是音樂劇專業科班出身的她一直擁有出類拔萃的演唱實力，因此她能在近期出演大型音樂劇《周日戀曲》，全程沒有一句臺詞、只能通過演唱和表演來傳遞複雜的情感，也就不令人感到意外了。不過上綜藝、演音樂劇，終究並不能完整展現婁藝瀟內心豐富的音樂世界，因此她選擇跳出螢幕女神形象，霸氣宣佈「歌手」身份，更像是繞了一大圈之後回歸了歌者的老本行。

繼去年高調加盟華納音樂並推出首支單曲《野生情人》大獲好評以來，婁藝瀟的首張專輯一直都在緊鑼密鼓的籌備中。這一年，她投入了大量的精力在音樂事業上，盡可能做到每一個細節都親身參與，親力親為，甚至連全國巡演音樂劇也都成了她不斷修煉音樂素養的機會，舞臺上的學院派唱法、高標準的技巧要求、全國多場實戰巡迴演出，無形之中也影響拔高了婁藝瀟對流行音樂的把控能力和演繹水準。

「專輯終極命題——什麼才是真的？
所有的出發都可能回到一個原點，
感受到的，才是真的，是真的，才是最美的。」

在籌備自己的首張專輯中，婁藝瀟的腦中一直縈繞著這樣的問題：什麼才是真的？圍繞這個議題，她認真地展開了從企劃、選曲、創作等一系列角度的思考和探索，如何才能把自己真實的想法融入專輯的每個細節中，也是希望給久等的歌迷呈現出最真實的婁藝瀟，一個未曾謀面但就在那兒的歌手婁藝瀟。

無論是參與創作的專輯曲目，還是MV腳本策劃，婁藝瀟都力求融入她的感受，以及在藝術造煉下展現出來的最真實的她——那個定義了“多才多藝”的她。

「Scott Effman x Tushar Apte x Stuart Crichton x Steve Shebby
黃韻仁 x 楊傑 x 金大洲 x陳韋伶 x 夏侯哲x 黃鵬程 x 管啟源
—— 歐美知名製作人、華語資深創作人、樂壇新貴音樂人 一齊貢獻絕佳創作」

落實「歌手」身份之後的婁藝瀟，自然要在音樂上施展自己的雄心壯志，希望自己首張專輯可以是一份令人滿意的「答卷」，充滿豐富精彩多元化的曲風，全面展現婁藝瀟的音樂可塑性。專輯除了一口氣邀請了Scott Effman、Tushar Apte、Stuart Crichton、Steve Shebby等歐美樂壇炙手歌曲製作人前來打造前衛時髦的作品，更有新加坡金牌製作人黃韻仁、內地音樂人楊傑、新銳音樂人金大洲、內地鬼才音樂人黃鵬程前來加盟負責專輯詞曲、製作的打磨，臺灣“音樂女詩人”陳韋伶、內地資深音樂人夏侯哲、資深詞人管啟源也貢獻了自己珍藏已久的佳作……網羅歐美、華語各路知名音樂人，只為打造令人期待已久的「絕美新聲」。

「十首歌 六支MV 一本寫真 華納音樂 重金打造
美國洛杉磯、索爾頓湖、臺灣、上海 —— 遠赴多地拍攝
黃中平 x 黃子然 x 謝韻 x 馬銳 ——聯合傾力打造」

為了呈現最美的婁藝瀟，華納音樂也不遺餘力重金打造，10首單曲完成6支MV，遠赴美國洛杉磯、索爾頓湖、臺灣、上海等地拍攝，同時推出一本全新個人寫真；邀請著名導演黃中平、黃子然、新銳導演謝韻、亞洲頂級美學專家馬銳等資深大咖鼎力合作。

首波主打「非類」的MV力邀臺灣資深鬼才導演黃中平掌鏡、臺灣偶像劇標配男主陳楚河參演，婁藝瀟和工作人員更遠赴臺灣宜蘭進行了拍攝工作。MV採用意識形態的表現方式，展現出在愛情逐漸消失的過程。拍攝日恰逢雨季，陰雨連綿，也打亂了一些拍攝計畫，導演不得不在現場修改了部分拍攝內容，瀟瀟也主動提出了自己的想法，更給導演的腳本錦上添花。

作為專輯的第二波主打單曲，「好想他」是一首樸實深情、格外走心的情歌。歌曲MV由臺灣新晉導演黃子然拍攝，內地男演員李子峰加盟，與婁藝瀟一起上演了一個溫存美好又令人心碎的愛情故事。婁藝瀟坦言：“這首歌和故事，十分令人感同身受，看到自己曾經的影子。”

「李仲強 x Parcz_Tsai —— 知名視覺設計師、新銳服裝設計師 打造一流視覺呈現」

原定專輯封面主視覺是清新寫真的仙美風格，但是婁藝瀟更渴望展現她個性的一面、多彩的一面、真實的一面。多次合作的新銳服裝設計師Parcz_Tsai深諳婁藝瀟內心的想法，並為其量身打造了多彩絢爛的造型風格，專輯主視覺最終呈現出別具一格的迷幻風格，這也映射了婁藝瀟一直以來給大家帶來的千變萬化的風格，每一面都是精彩，都是驚喜。裝幀設計更交由獨具匠心的臺灣知名視覺設計師李仲強打造，專輯包裝採用大封套設計，精美無比，極具收藏價值。
