Album Intro：

充滿電影感的概念大碟《Dear Secret》
實力音樂班底打造新作《幸福門》、《美男子與香煙》
Gin Lee親自作曲《開燈。熄燈》剖白內心世界

一張屬於Gin Lee(李幸倪)的概念大碟《Dear Secret》，營造出充滿電影感的視覺效果，不論畫面和音樂都充滿張力。裡面包含Gin親自作曲和個人情感的作品，赤裸呈現，讓大家可以窺探她的內心世界，與她經歷一趟秘密旅程。

每個人家裡，總有一些不顯眼的東西、瑣碎的家庭用品，每件東西背後都有著一個故事，一個屬於家裡主人和物件之間的故事。新碟以家作主軸，圍繞這個地方發生不同的故事。家，其實也像是人的腦袋，家裡不同的陳設物件代表不同記憶。

「門」，是家裡最敏感的地方。開門，迎接；關門，歡送。開門與關門之間，迎送了多少人？開門迎來當初的那一位，到近來的一位，當中有多少次相親與相分，由陌生人變熟悉人，又由熟悉人變陌生人，開門關門記錄和印證了這一切的回憶。

「燈光」決定一個環境的氣氛。生活總有瑕疵，燈光太亮，會發現牆壁好多從前沒發現過的裂痕，有時寧可在半暗半亮之間，躲在曖昧的氛圍裡尋求圓滿。

「開燈熄燈」，每天都會做這個動作。每次開燈後，都會看見他/她/牠，若果對方從此消失了，我寧可是「開錯燈」，希望可以再次見到對方。

這次《Dear Secret》專輯由曾為王菲和陳奕迅製作很多膾炙人口作品的Alvin Leong擔任全碟監製，所有歌曲都由他的班底一手包辦，呈現统一貫徹的音樂風格和方向。Gin透過親自作曲《開燈。熄燈》(開燈版)、《開燈。熄燈》(熄燈版)和《開錯燈》，剖白她與離世愛貓的故事，從失去帶出珍惜的意義。


歌曲簡介：
1.《幸福門》
曲 : Vincent Chow / 澤日生 詞 : 何方 編 : Gary Tong 監製 : Alvin Leong
以門作主題，Gin以第三身角度說故事，有些人和關係希望留低，但最後都留不低。在現實中留不底的一切，最後只能變成回憶。所謂望門興嘆，門，也是讓人等待的道具。某次關門的時候，或者會忽然想起，後面再沒有人幫你鎖門了。門鈴響起時，也想不到是誰到訪值得驚喜。這首歌原本的編曲以鋼琴為基礎，後來歌曲的前奏和前半段加入結他和鋼琴協奏，營造了和諧的氣氛。其後逐步加入其他樂器，例如鼓和弦樂去打造歌曲後半段比較戲劇化的感覺，以適中的力度去襯托Gin的人聲(vocal)部分，恰到好處。

2. 《美男子與香煙》
曲：Vincent Chow 詞：何方 編曲 : Gary Tong 監製 : Alvin Leong
唱出一個情慾愛恨交纏的愛情經歷。明知愛情有毒，但仍是放不開；就算再污煙瘴氣，還是想捉緊那不可能的幻象。故事中的主角，因為現實沒有事情讓她留戀，如果可以讓她回到過去，那怕再一次陷入迷戀也是值得的。有人擁抱回憶，因為不想向前行，但回憶一旦被利用，就會成為痛苦的來源。Gin的演繹，也由一開首訴說別人故事的淡然，漸漸隨旋律加強了力度，投入於歌曲中的痴戀情感，交出了完美的唱功和演繹，一字一句均令人動容。

3. 《開燈。熄燈》 (開燈版)
曲：Gin Lee李幸倪 詞 : 何方 編 : Ted Lo 監製 : Alvin Leong
Gin親自作曲寫給已離世愛貓的歌，這是送給它的最後一份禮物，紀念一段刻苦銘心的旅程。在錄音期間，Gin憶起與牠的種種回憶，每一次唱都會淚崩，更無法完整地錄唱全首歌，這是她從未經歷過的錄音挑戰，也是她出道以來最「不懂」唱的歌。滿瀉的回憶不停在Gin的腦海裡徘徊，每唱一次，就像再經歷一次與愛貓的旅程，沉重的心情揮之不去。

4. 《開錯燈》
曲：Gin Lee李幸倪 詞 : 何方 編 : Ted Lo 監製 : Alvin Leong
「要是人亦似寵物　那又何謂主僕情深」是人又好，是寵物也好，道別一刻亦難掩傷心。畢竟你我之間有太多回憶，想念之情一發不可收拾。要放下執著和習慣的確不易，特別是這個習慣 - 每次回家開燈再喊出你的姓名，雖然到現在我仍會開錯燈，甚至不懂熄燈，但希望你不要操心，我會好好過。

5. 《別人的家》
曲：Vincent Chow / 澤日生 / Gin Lee李幸倪 詞 : 何方
編 : Fergus Chow / Derrick Sepino 監製 : Alvin Leong
這一個家不是那一個家，說到家這一個字，大家都不以為然地想起居所，但這首歌所提及的家，比喻了一個你認為可以寄托可以倚靠的對象。這個人提供一個空間予你去放鬆，做自己，可以發洩，可以療傷。可是，這個「家」令人感到不少壓力，既然如此，「我要擺駕　不帶走一絲牽掛」。

6. 《搬》
曲：澤日生 詞：許弘毅 編：C.Y. Kong思歪江 監製 : Alvin Leong
要離開的，始終都會離開。換一個伴侶就如搬家一樣，很多東西思前想後。覺得舊居住得不舒適，但執拾東西時會發現很多值得留戀的事件，又怕習慣不了新地方。種種矛盾都總會在決定去與留之際浮現。即使《別人的家》曾經有幾多開心回憶，到了一天覺得感覺不對，都終需要狠心的斷捨搬離。能帶給我有家的感覺，相當重要，便不會想再搬家了！

7. 《成家》
曲：Vincent Chow / 澤日生 詞：許弘毅 編：Gary Tong 監製 : Alvin Leong
小時候，大部分女生的父母都希望女兒可以嫁得好，目的都是希望她生活過得無憂。但風光背後，卻有很多說不出的苦。《成家》卻唱出一位活得自在且灑脫的女生，看破童話式愛情的陰暗面，不為愛情而活。不怕被標籤單身，沒有名牌，身份和品格依然可以高貴。愛情與麵包，其實在乎自己選擇。

8. 《開燈。熄燈》 (熄燈版)
曲：Gin Lee李幸倪 詞 : 何方 編 : Ted Lo 監製 : Alvin Leong
《開燈。熄燈》熄燈版，與開燈版相比，編曲經過多次修改，Gin亦重新錄音，最後決定以最簡單的鋼琴伴奏，及真摯不修飾的唱法把故事最原始的一面呈現，成就了這個熄燈版，彷彿是在親人耳邊的最後一次告白。貓不只是寵物，更是親人，經歷過失去的人相信都會有共鳴。
