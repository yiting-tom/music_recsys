Album Intro：

2012年初春，擁有挑剔耳朵的樂迷朋友們期待已久的唱作才女曹方，在沉寂一年多後終於推出全新EP《淺彩虹》,專輯由曹方歷時一年多時間，在旅居生活中創作完成。經過一段時間的獨立操作，曹方以大班音樂工作室主理人的身份發行了《比天空還遠》、《哼一首歌等日落》等音樂作品集並廣受好評。2012年度，曹方再度攜手《遇見我》專輯的音樂策劃人相征，大班音樂工作室誠意結盟新廠牌“NOVA娛樂”，重新出發，開啟全新的音樂旅程。

/ 創作背景 /
過去這一年多時間，曹方常背起行囊上路，帶著寵物鐵鍋，從北到南，去到許多國家與城市，無論是老撾的熙攘小道，還是東京夜晚的居酒屋，都留下了曹方和鐵鍋的足跡。沿路看到各種繽紛色彩的曹方，亦覺得這一年多的旅居生活給自己帶來了前所未有的感受和衝擊，經過一段時間的沉積和消化，曹方決定以顏色為靈感腳本講述五個故事，填上音符變成歌，於是便有了現在這張限量EP作品《淺彩虹》。

/ 生活感觸 全新方向 /
闊別近兩年重裝出發的曹方，令人驚訝於她所散發的全新能量，逐漸告別少女的稚嫩與單薄，越發具有女性情調的柔美氣質，擁有純粹的生活態度與質感的藝術品味，這一切都不同於大眾印象中小女生狀態的曹方，倒是在音樂主張和表達上，曹方依舊以勇敢的獨立姿態響亮發聲，更像是給陰霾的音樂行業綻放出一抹淺彩虹！始終堅持真誠創作的曹方，永遠給歌迷帶來極具誠意的作品，鼓舞著大家以個人微小力量，堅持夢想綻放。

/ 全新創作 全新表達 /
《淺彩虹》EP由女性視角出發，敘述著曹方行走歲月的私密情愫。經過近兩年精心準備的曹方藉由本張EP自吟自唱著淺彩虹一般的曖昧心境。陰霾還未散去，彩虹卻已劃破天空，憂傷與喜悅交織，希望與失望重疊，其間的微妙情緒在本張作品中纖毫畢現。本張專輯音樂製作由老搭檔王曉東先生操刀，作為合作了這麼久的默契搭檔，曹方與製作人都想表達出更多的音樂靈感，在風格上亦是做了許多不同嘗試，如專輯同名歌曲《淺彩虹》所呈現的復古搖滾風，《懷疑》一曲弦樂與人聲的大膽搭配，主打歌《傷心旅客》更是用鋼琴等看似簡單的配器營造聲場來承托跌宕的情緒起伏……這一切，相信聽過整張作品的你一定會有更為真切的感受。

本張EP限量7777張，每一張唱片均有獨立專屬編號，7777張也寓意7色彩虹，應和淺彩虹主題。此次EP採用獨特大開本設計，質感特種紙精美印製，封面可變四折小海報，並配有獨立Lyrics卡，Lyrics頁由不同淺色背景鋪就，寓意淺彩虹主題，更值得一提的是本張EP中特別搭載的概念別冊“60 seconds with icy”，以時間為容器，給你一個同曹方私密相處60秒的特別體驗，別冊中的曹方拋開大家所熟知的光鮮亮麗的藝人身份，卸下所有包裹和裝飾，盡情展示私人情緒，相信對於喜愛曹方音樂的朋友來說，這將會是次美妙的一分鐘旅程，跟隨icy一起感知時間的流逝。

/ 大膽嘗試 全新造型 /
為配合本張音樂作品的獨特氣質，此次NOVA娛樂專程找到知名女性藝術情調刊物LITTLE THING戀物志合作，專程為此次造型拍攝工作集結香港&內地知名造型妝發團隊，並量身打造“淺彩虹”主題的全新造型。經過精心準備和反復對音樂作品的聆聽感受，團隊成員們更是敏銳的捕捉到曹方內心的細膩轉變，在整體造型上更加注重細節，突出女性曖昧氣息，將曹方褪去青澀稚嫩，又未滿成熟的狀態展現的淋漓盡致，最終打造出精緻復古的驚豔視覺效果，力圖給歌迷呈現一個最具淺彩虹氣質的全新班長。

/ 全新玩法 互動體驗 /
為更好詮釋音樂情緒，此次曹方首度與傣族插畫家罕璿及知名app開發團隊KooWorx合作，以app的形式推出多媒體藝術互動作品《淺彩虹》，並成為國內首位推出獨立於唱片之外，全新創造互動藝術app的音樂人。傣族插畫家罕璿一直忠於與自然對話，以純淨的態度感知著自我與自然的交集，此次插畫作品全部由曹方歌曲為靈感創作而成。在插畫家筆下，曹方歌曲中描述出的女性情感世界幻化成頗具迷幻色彩的顏色，無論是插畫中的植物、動物還是小女孩，都透露著敏感而又獨立的姿態，像極《淺彩虹》中繽紛又曖昧的小世界，亦如一直以來堅持獨立創作的曹方，以小小的力量，始終倔強又敏銳的前行。
有別於其他音樂人在app store發售一款與實體唱片一般無二的電子版唱片的做法，《淺彩虹》app中的插畫、動畫、互動體驗等內容皆為全新創作，同唱片截然不同，相信會讓手指觸控式螢幕幕有淺色彩虹流出指尖的你充滿驚喜和感動！

/ 全新形式 全新巡演 /
隨著《淺彩虹》EP的發行，量身定制的主題巡演也即將於三月底啟動，並在4～5樂綻放全國。此次巡演Concert打破既有的演出形式，曹方將聯合新銳視覺藝術家“編號223”共同呈現一場精緻的演出，編號223負責整體視覺設計和舞臺呈現，將以大量多媒體藝術、裝置藝術等形式配合現場音樂作品表達，以藝術家的獨特角度解讀曹方，呈現出一場非凡視聽體驗，相信會給到場的樂迷朋友們不同以往的全新感受。
