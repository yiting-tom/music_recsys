# 「日常即創作」生活感唱作 慢慢說清新詼諧樂風再起
# 「符號即日常」現象觀察 萌生第 3 張專輯《 1＋1＜3 》
# 「漂符系金曲組合」新定位 漂浮於慢慢說音樂裡

第 26 屆金曲獎最佳演唱組合、由主唱利得彙和吉他手沈志方組成的「慢慢說
MURMURSHOW」，獲得日本 AMUSE 公司首簽的台灣樂團之殊榮，成為南方之星、福山雅治、ONE OK ROCK 和 PERFUME 的同門後輩。曾參加央視《中國好歌曲》節目以創作曲〈五虎〉躋身天后蔡健雅戰隊受到注目，發揮豐沛的創作力推出了《MURMURSHOW》、《驕傲地愛著》二張專輯，以復古俏皮、無厘頭的幽默曲風，秉持「日常即創作」的生活感唱作，樹立快慢皆宜的獨特清新詼諧風格，頗受樂評、DJ、樂迷推崇喜愛，更十分期待慢慢說的下一步，想要說什麼？

今天，你「社群」了嗎？現代人幾乎生活離不開 3C 與網路，訊息心情與人際關係極度仰賴社群世界。慢慢說兩人亦是重度網路使用者，喜歡打電玩線上遊戲，同時是遊戲軟體影音串流平台 Twitch 知名實況主，常會在直播間分享互動遊戲和開線上音樂會，善用許多符號來取代文字溝通，有時一個符號，一個圖示，勝過千言萬語，符號就是社群新人類的日常。這也讓慢慢說萌生了第 3 張符號日常創作專輯《 1＋1＜3 》的藍圖，用音樂詼諧探討社群世代對符號的愛與恨，描繪出人們使用符號表達生活裡喜怒哀樂的況味百態，省思在過度依賴、簡化的虛擬人際互動裡，如何保有最真實、也最人性的情感交流。

慢慢說 MURMURSHOW 在全新專輯裡有個俏皮新定位—「漂符系金曲組合」，不僅呼應專輯裡充滿「符號」與「社群」的概念，如同每個活在符號時代的人們一樣，每天訊息與心情在網路世界串動交流，就像是在虛擬雲端上漂浮。這定位同時十分貼合慢慢說的音樂是充滿日常感的，給人感覺輕鬆沒壓力，無厘頭幽默看人生，可以是嗨翻天在雲上的歡樂漂浮，也可以像水波低盪的沉靜漂浮。聽著慢慢說，能帶著每個聽者神遊內心想去的地方，不管開心或傷心，都可以恣意自在漂浮在慢慢說的音樂裡。


# 專輯名符號感十足 「1＋1」關係總和成愛「＜3」
# 以社群與符號為元素 組構各種愛的連結
# 一張愛與勇敢的音樂紀錄 社群世代的符號生活觀

「漂符系金曲組合」慢慢說第 3 張創作專輯《 1＋1＜3 》，乍看是個理所當然的數學關係式，其實符號感十足的專輯名稱是由「1＋1」和「＜3」所組成，其中「＜3」是在社群軟體打出這兩個符號，就會顯示出代表「愛心」的表情符號， 1 個「＜」加上 1 個「3」，組合成 「心」。不只用在符號上，「1＋1」的關係也可延伸到人和人的各種連結，比如：社群傳送者＋接受者、神隊友＋豬隊友、心儀者＋曖昧對象、心碎者＋前任......等等，投射到慢慢說的兩人組合就是 1 個主唱＋1 個吉他手的默契搭配。當然不只是人際關係，「1＋1」也可以是人和物、人和回憶、人和夢想......等等，而不管是怎麼樣的「1＋1」關係，最終總結都是「＜3」，都是愛的關聯。

人們總是太過於依賴符號，想快快講。忽略那些藏在符號背後的，其實才更該慢慢說。專輯《 1＋1＜3 》以社群與符號為出發點，唱出使用符號的流行趨勢與模式觀察，圍繞在愛的核心上，可以是愛情、親情、友情、甚至是對事物、夢想、寵物的愛。這不僅是一張探討「愛＜3」的專輯，更是一張「勇敢」的專輯，除了專輯名表面顯性的符號感，還蘊含著隱性卻很有力量的內在意念，也就是「1＋1」和「＜3」之間，其實藏在符號背後的是「勇敢」。愛的關係各有不同，有些愛是真的需要勇氣，要勇敢表達、勇敢追逐、勇敢說出來，但也有些愛反而是要不張揚，只需默默勇敢守護、或是為了對方而勇敢堅定的陪伴。讓專輯《 1＋1＜3 》不單單總成一種社群世代的符號生活觀，更是一張愛與勇敢的音樂紀錄。


# 「1＋1」金字保證 金曲演唱組合 ＋ 金曲單曲製作
# 利得彙錄音大玩角色扮演 沈志方獲讚誇「性感男聲」

金上加金，再造金級傑作！慢慢說第 3 張全新創作專輯《 1＋1＜3 》，由金曲獎最佳單曲製作人陳君豪操刀製作，Album songs大多由利得彙 + 沈志方詞曲創作與演唱錄製，金曲加上金曲，也是音樂上的「1＋1」金字保證！曲風多元，包含：電子搖擺搖滾 Electro Swing Rock 、東方感 Oriental R&B 、靈魂搖滾 Soul Rock、復古 Retro R&B、靈魂抒情Soul Ballad......等等，3 人超級默契的「音樂神隊友」，鬼點子完全不設限發揮創意，神激盪出許多天馬行空的音樂新花火！

利得彙透露錄音過程充滿歡樂：「我常在『角色扮演』，在錄音室裡君豪就是瘋狂導演，他會給我很多扮演指示，一下子是吉普賽女郎，一下是神祕性感女神，一下是神經質怪怪美少女，一下又是王菲、鄧麗君、椎名林檎！」而這樣看似遊戲的創意激發下，反倒開發出許多利得彙以往從未嘗試過的不同唱腔詮釋。沈志方這次在專輯中也加重許多演唱比例，自己很喜歡錄出來的聲音，更被讚誇為「性感男聲」，陳君豪與沈志方兩人都是吉他手背景，所以他很會「因材施教」，引導沈志方在錄音室配唱時可以假裝自己在吉他彈唱，這獨特的「空氣吉他配唱法」，成為沈志方錄音時表現自在的密技。


# 首發 心碎慢歌 〈自導自演〉 電視劇《延禧攻略》片頭曲
# 連二 電玩神曲 〈神隊友〉 MV創意翻玩遊戲元素
# 拉三 慵懶主打 〈失眠電影院〉 發想自空中環「搖籃 」
# 接四 勇敢主打 〈再見．快樂〉 椎名林檎御用混音師跨刀

強力新曲主打四連發！專輯暖身從首發心碎「慢」歌〈自導自演〉開始，亦是超火紅電視劇《延禧攻略》片頭曲，在2018年底天后專輯齊發大亂鬥的強敵環伺下，依然保有一席之地，在兩岸數位音樂平台深獲好評，被歌迷們譽為「年度虐心之作」，勾起許多同是情傷人紛紛寫出在感情回憶裡自導自演的心情留言。

第二發開心輕快歌〈神隊友〉，重新召喚充滿「慢慢說式」獨特幽默無厘頭的詼諧曲風，為歌壇注入快樂爆表的超神正能量！歌名與Lyrics寫入許多好玩逗趣的電玩與網路流行語，包含：凱瑞 Carry、666、大腿、開掛、放大絕等等。MV也向復古經典電玩遊戲致敬，創意翻玩許多遊戲元素，並加入大量可愛的表情圖示呼應專輯的符號概念。

第三發慵懶性感主打 〈失眠電影院〉，是首充滿內心小劇場的浪漫情歌，歌曲發想自利得彙十分熱衷的空中環運動的動作「搖籃 Cradle」，設定是睡前一段腦內溫習，彷彿利得彙優雅跳著空中環，眼前是沈志方欣賞共鳴共舞的電影情境曖昧想像，就算想到失眠也無所謂。

第四發抒情勇敢主打〈再見．快樂〉，混音由椎名林檎御用混音師－井上雨迩 Uni Inoue 跨刀合作。抒情搖滾曲風，旋律鋪陳隨著Lyrics前後情緒做不同轉換，前面段落偏向悲傷省思，後面結尾則變成激昂帶有堅定，代表堅決斷然跟過去告別，愛情不需要這麼卑微，勇敢跟過去說再見，希望自己快樂，愛自己才最值得。


# 有「符」同享 10首歌曲對應專屬符號顏文字
# 第 3 張專輯 巧妙呼應「3」「三」數字符碼
# 主視覺簡約清新不失前衛 符號藏在設計細節裡

符號人生，有「符」同享！呼應慢慢說充滿符號感的全新專輯 《 1＋1＜3 》，特把 10 首歌曲，賦予 10 種社群符號對應，並將大多數符號歌名巧思串聯 「3」或「三」有關的數字符碼，強化第 3 張專輯的連結感。先前曝光的 3 首新歌在單曲封面設計都埋下伏筆加入社群符號與顏文字，包含：電影感心碎情歌〈自導自演〉創意翻玩心碎符號「＜/3」的組合圖、電玩感歡樂助攻神曲〈神隊友〉置入代表五體投地的佩服符號「 m(_ _)m 」、催眠感慵懶搖籃曲〈失眠電影院〉則是以代表蓋棉被躺平的睡眠符號「 (:3[___]= 」。

新專輯主視覺呼應網路符號，在封面拍攝上力求簡約清新不失前衛感、大玩色塊變換，兩個人各自拿著 3C 產品拍照，象徵現代人幾乎生活離不開網路世界。專輯設計也頗具巧思，有歌名符號、顏文字、如密語般的反轉文字、符號情境圖小海報、視窗圖、下載傳輸符號……等等創意元素藏在設計細節裡。整張專輯從專輯名、到每首歌曲、設計包裝都巧妙對應社群符號，饒富趣味感，讓《 1＋1＜3 》成為名「符」其實的符號日常創作專輯，一起漂「符」在慢慢說音樂的雲海裡。


[1] 〈神隊友〉 m(_ _)m 神到佩服五體投地
詞：利得彙 曲：利得彙 / 沈志方

與其等待 777 的幸運神庇佑
不如善待 666 的幫運神隊友
電玩感歡樂助攻神曲〈神隊友〉， Electro Swing Rock 電子搖擺搖滾快活曲風，編曲加了許多電子元素，樂器音色上以取樣 Sampling 技術經過處理、重建再運用別具特色。前奏則與第二張專輯收錄的〈喜歡待在家〉異曲同工，設計一段卡祖笛 Kazoo 充滿喜劇電影感的俏皮旋律作開場，由利得彙親自演奏，讓「卡祖笛」奠定出代表慢慢說獨特的音樂符號。神隊友不是神， 卻總是很神，〈神隊友〉Lyrics架構在組隊打怪的電玩風格裡，像是兩個夥伴之間的對話彼此打氣，然而利得彙更想表達的是「電玩如人生」，許多做人態度都藏在其中。尤其是網路上常說：「認真就輸了」，但她卻有不同見解：「因為這樣一來反而讓很多人不敢認真了，因為怕輸，卻忘了一往無前才是做人的氣魄和精神，這才是勇敢。」

[2] 〈失眠電影院〉 (:3[___]= 捨不得洗洗睡
詞：利得彙 曲：利得彙 / 沈志方

自從認識你，失眠變成一件很棒的事情

催眠感慵懶性感搖籃曲〈失眠電影院〉， Neo Soul R&B 曲風，是首充滿內心小劇場的浪漫情歌，以八釐米膠卷音效聲營造電影放映開場，利得彙挑戰「性感女伶」的唱腔，編曲加入紐約管樂手跨國合作，更少見以 Bass 當主奏元素，成為貫穿全曲的靈魂，貼合整首歌低沉性感的氛圍。熱愛空中環運動的利得彙，一直很希望寫一首可以用在表演的歌，因此從曲調到Lyrics，圍繞著「漂浮」概念，有睡前失眠腦內的想像力漂浮，也有空中環優美舞蹈的肢體漂浮，讓這首〈失眠電影院〉成為名符其實的「漂浮歌」，對映著「漂符系金曲組合」的新定位稱號。

[3] 〈雷陣雨〉 ε= ε= ε= ┌(;￣◇￣)┘ 暴雨將至 速走快逃
詞：利得彙 曲：利得彙 / 沈志方

再狂的雨，總有雨過天青的一天
再傷的痛，總有良藥苦口的一帖
氣象感宣洩紓壓歌〈雷陣雨〉，Alternative Rock 另類搖滾曲風，編曲特意安排很多樂器之間的對話，比如：吉他和鼓的對話、BASS 和鼓的對話，營造雷陣雨情境裡大雨滂沱、措手不及的煩悶氛圍。〈雷陣雨〉是首負能量爆發釋放後的正能量歌，憤怒宣洩後心裡的大雨終會停歇，等烏雲散去重新找到出口，這也是繼代表作〈討厭下雨天〉後再一首以雨入歌，可謂是慢慢說之「雨的第二部曲」。

[4] 〈再見．快樂〉 (T_T)\(^ ω ^) 秀秀 哭完就雲淡風輕了
詞：利得彙 曲：利得彙 / 沈志方

勇敢跟舊的快樂說再見
才能跟新的快樂再見面
徹悟感勇敢放手情歌〈再見．快樂〉，Soul Ballad Rock 抒情搖滾曲風，編曲以前衛的電吉他彈法並增添 Wah Wah 效果器，製造音色衝突感，保有感傷氣氛卻不至於太過沈重，並加入 Organ 復古元素，為歌曲營造很有支撐感、有力量的音色。歌名〈再見．快樂〉有兩個含意，Lyrics前半的「再見．快樂」中間刻意用個實心點隔開，表達出面對放不下的戀情，要有勇氣跟曾經的快樂說再見，而Lyrics裡最後一個「再見快樂」拿掉實心點，象徵才能夠與新的快樂再見面，遇見下一次幸福。利得彙最喜歡其中Lyrics「錯把心動當成愛情築起 卻忽略心安才能過保鮮期」，人們總是喜歡一個人讓你心動，卻沒有想過這個人能不能讓你心安跟你一起走下去才是關鍵。 這也是她對愛情的體悟，對彼此有安全感才是愛情長久的重要依據。

[5] 〈痕跡〉 川 被愛劃的傷痕，就算痊癒還是會隱隱作痛
詞：陳信延 曲：利得彙 / 沈志方

時間是愛情的刺青師
在心上紋出回憶的隱形符號
旅行感復刻回憶療傷歌〈痕跡〉，Soul Ballad 靈魂抒情曲風，編曲以豎琴優雅開場，帶有歐式溫暖風。Lyrics由擅長描繪日常情境裡深刻而細膩情感面的名詞人陳信延創作，以「旅行」為情境元素，充滿異國畫面感的文字氛圍裡，復刻旅程行走在一起去過的地方，都會變成身上的印記，都是遺留在戀人身上的「隱形符號」，就像紋身刺青一樣，很美但卻也隱隱作痛。平日相安無事不彰顯，但一旦被喚起了，曾經相愛過的點點滴滴，就如人生跑馬燈般歷歷在目。一如Lyrics「我身上隱形符號 怎麼去刪也刪不掉」，愛情伸出尖爪在戀人身上劃下名為回憶的「川」型痕跡，只可惜我們都只是凡人，不像金鋼狼那樣能擁有隨時復原成無傷的超能力，我們只能在一次又一次的失戀裡受傷、治癒、再成長，用留在身上的痕跡，提醒自己那些愛教會我們的事。
