Album Intro：

剛完成單曲的蕭羽心和饒舌歌手BigOne朱秝樞 ( 韓國 C J E＆M 簽約 YouTuber ) 於2017年一同參加阿爾發唱片舉辦的音樂比賽，一舉得到冠軍殊榮。

再度聯手推出創作的他們，將迥然不同的音樂靈魂注入作品，這次合作會迸發什麼樣的火花呢？。

[ SHASHA 蕭羽心 ]

該類型風格是蕭羽心第一次嘗試，便一手包辦製作、編曲、混音，連同平面設計也不假他人之手。只為一首歌曲完美呈現，花費了近半年至一年的時間，就是願意為自己所堅持的一試再試，展現音樂上的主見和執著，是個獨當一面的韌性女孩。

蕭羽心完成單曲「我的主題曲」後，跟隨FIR阿沁老師到上海宣傳工作，對生命有了新的看法與感受，回到台灣的歸屬感激發新的創作靈感，馬上為自己的音樂藍圖再度整裝出發，積極的創作規劃，希望自己每年都能有不一樣的轉變與突破。

除了音樂製作的能力精進，她喜愛的設計也有不同凡響的表現，去年度為歌唱節目「希望之星 」設計節目LOGO ( 友松娛樂製作 )，也為許多歌手 ( 亦凡、戴愛玲、國蛋、男團 A-TEAM……等 ) 設計MV字體與專輯，無論手寫或平面設計都很有自己的風格和獨到的品味。

造型上獨樹一格的蕭羽心，留著一頭鮮豔的紫色長髮，一抹淡淡粉紅色的眉毛，幽靜的空靈氣息，時而個性時而甜美的模樣，只看外表似乎有些霸氣難親近，骨子裡卻是天然呆萌的善良直率。在她的作品裡完全能感受到她不易被打垮的個性，一手音樂一手設計，不願放掉所愛事物的倔強，讓人無法輕忽她的毅力與能力。

[ BigOne 朱秝樞 ]

BigOne朱秝樞在「尋找南拳媽媽接班人」比賽中受到矚目而發跡，並在MadStreet狂熱街頭、金曲獎國際音樂節、BigOne X OZI - This Side HipHopParty 等嘻哈活動中嶄露頭角，並與韓國最大娛樂公司「C J E&M」簽約合作。

BigOne在饒舌曲風上，變化多元，早在2015年就以現在流行的TRAP曲風，參加韓國YG娛樂公司所舉辦的比賽，獲得第二名的佳績。爾後更嘗試Dubstep、EDM、BoomBap等曲風，勇闖各大比賽，拼出卓越的成績。
舞者出生的他，經歷過萬人演唱會的洗禮，穩健的台風令他在饒舌這條路上更是得心應手。

也許社會上仍存在些許對嘻哈文化與幫派、毒品、情色密不可分的偏頗印象，從小受到B-Boy文化影響的BigOne，藉由饒舌度過兄弟的背叛、單親、負債、現實的總總壓力，轉化為自己的音樂態度，透過最真實的用字遣詞，及賦有辨識度的唱腔，傳達給大眾正面的理念及能量。

[ ZOE ]

靈生命代表的單字「 ZOE 」，為夏娃希臘文之名，代表著「一切生命的本源
」象徵活力之表現，亦為歌曲的主軸。

當我們面對生命雖有挫敗，偶然的低谷可能使自己失去奮起的勇氣，始終會發現當勇敢跨出痛不欲生那一步時，生命迸發燦爛花火的瞬間，自己將對愛與生命所付出的一切感動、榮耀。

此創作巧妙運用三首知名歌曲 ( 邦喬飛It's My Life、披頭四All My Loving、夏奇拉Shakira - Try Everything ) 的片段Lyrics，將文字重組分段並和節奏韻律一拍即合， 如此順暢的韻詞，跳耀式的創作思維，不難想像其中之意相當耗費心思。
