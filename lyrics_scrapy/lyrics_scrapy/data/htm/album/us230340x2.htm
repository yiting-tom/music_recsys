Melet Fan 范庭瑜
首張全創作專輯《最後一封情書》
從深邃灰黑尋覓重生純白
從盡頭尋找屬於我的樂園


為什麼哭了？為什麼笑了？
多久沒好好和自己聊天了？
多久沒誠實地面對自己了？

嘿，親愛的，
有時候，我也不知道為什麼會這樣。
有時候，我也不知道那未來會怎樣。

存在生命周遭，短暫、局部、甚至烙下疤痕的故事，
可能是我，可能是你，也可能是他。
那些平凡的小事與喜怒哀樂交織著生命，
擦出情節火花，隨時牽引下一秒無預警地爆炸。

我是短暫按下快門的掌鏡者，透過音樂留下瞬間永恆。
首張全創作專輯：給世界的《最後一封情書》。


「世界」有幾種模樣？
透過我的眼、我的筆，把對於世界的定義帶給大家，揭開身邊平凡又不凡的故事篇章。

「最後」並不真的是最後。
就像是在真正出現句號前都會有一個逗號讓人稍做休息。
或許最後可能像是逗號般存在，暫時告別。又或許只是一個階段的結束。

「情書」用來表明心意。
比起用說的，我更喜歡用寫的。
文字在書寫的過程中滿載情感份量，人們透過情書表露心意。
我則是將滿載情感的文字化成Lyrics，寫成《最後一封情書》這張專輯。


《最後一封情書》這張專輯擁有我最愛的三色彩：

「淡淡的土耳其藍 / 既憂鬱又平靜」

報導說藍色是最讓人心靜下來的顏色，
就如同我看著你的照片終於不會哭了，

不會再去思考為什麼了，心裡異常地平靜。
可是最想說出口的「我想你」卻再也沒辦法說出口，
如此的惆悵感，就像溫暖又憂傷的藍色。


「屬於夏天的黃 / 既開朗又充滿能量」

即使知道聖誕老人不存在，仍然相信著每一個明天。
認真地、勇敢地迎接每一個下一秒，你的每個不完美更顯得你多完美。


「夾雜著毛屑粉塵的灰黑 / 既又深陷泥淖的負面」

都會有低谷的，才能往上振作。
你的自卑和你的沮喪，還在深陷其中又或許不斷墜落，
就哭吧，就大吼吧，總會有對翅膀帶著你離開的。



「你，收到我的情書了嗎？」

就像盧凱彤的「留一秒」，
至少我還有「一雙手」，寫下所有的開心與悲傷，
勇敢接受自己的懦弱脆弱與自卑。

01.讓我離開我

「離開過去的我。」

每段時期的我都不太一樣，2018暑假我強烈地感受到自己的渺小，卻無法擺脫讓自己停滯不前的那個我。於是我開始練習面對自己的情緒，透過和不同人談話，漸漸找到面對悲傷及負面情緒的方法。到這個時候，我終於明白我早已離開：「離開過去的我。」

詞：范庭瑜 / 曲：范庭瑜 / 編曲：張哲瑋 / 和音編寫：柯淳恩 / 和音：柯淳恩 / 錄音：葉晉嘉 / 剪接：侯冠良 / 配唱：柯淳恩 / 混音：T.M.　


02.最後一封情書

「我最後能做的就是寫一首歌送給我們。」

也許是因為即將分手，也許是預感他即將離開，記得那時心中閃過一個念頭，只想好好和我的愛人告別，我最後能做的就是寫一首歌送給我們。明知不愛了卻還裝愛而緊抓不放，心裡明白，好好放手也是一種愛的方式，所以「我們都別再假裝」。

詞：范庭瑜 / 曲：范庭瑜 / 編曲：黃德霖 / 錄音：黃德霖 / 配唱：黃德霖、孫令謙 / 混音：T.M.


03.上癮

「想逃避，卻又被吸引。」

戀愛就像染上咖啡，輕啜一口一不小心就越陷越深無法自拔。曖昧世界裡的莫非定律，越是不可能就注定會發生；曖昧世界裡的偶然與巧合都是誰誰誰的精心策劃，讓人一步步掉入這叫做戀愛的圈套裡。

詞：范庭瑜 / 曲：范庭瑜 / 編曲：陳子文 / 吉他編寫：黃敔棠 / 錄音：吳書涵 / 配唱：柯淳恩 / 剪接：侯冠良 / 混音：T.M.


04.Baby Don’t Cry

「每一天的明天都會是全新的。」

天氣不好下了雨、出門才發現忘記帶錢包，很多時候因為這些瑣碎的小事影響整天的心情。長大以後發現自己越來越膽小，不像小時候，就算跌倒大哭，擦乾眼淚又繼續往前，都是那麼天不怕地不怕。每一天的明天都會是全新的，別讓這些小意外讓整天心情掉拍。這是一首鼓勵我和你的歌，一起勇往直前無畏無懼！

詞：范庭瑜 / 曲：范庭瑜 / 編曲：蘇唐 / 合聲編寫：柯淳恩 / 和音：柯淳恩 / 錄音：吳書涵 / 配唱：柯淳恩 / 剪接：侯冠良 / 混音：T.M.


05. 承諾

「很多時候約定總是會被打破。」

在霍爾的移動城堡裡鄰國王子對著荒野女巫說「變心是人之常情」。是啊！心總是會變的。
留在原地的人看起來好傻，或許「真心」在這個世代反倒不正常。

學習放過自己，當一個不正常的人也很好。我想對那些人說。

詞：范庭瑜 / 曲：范庭瑜 / 編曲：莫于庭 / 錄音：吳書涵 / 配唱：柯淳恩 / 剪接：侯冠良 / 混音：T.M.


06. Deep Down ( Acoustic )

「覺得受傷，卻讓我無法反駁。」

Deep Down是來自於我替房間冰箱除霜時，房裡冰箱的冷凍庫總是一直結霜，需要拿菜刀把它敲下來，不然沒辦法冰東西，就這樣結了又敲敲了又結持續一整個學年。其實冰箱就像我，即便我渴望有人能明白這樣矛盾的自己，但出現想要靠近我的人那時，我又把門關起。一直以來我是活在自己的世界裡，可我卻最討厭別人說我老是活在自己的世界。覺得受傷，卻讓我無法反駁。

詞：范庭瑜 / 曲：范庭瑜 / 編曲：黃德霖 / 錄音：吳書涵 / 配唱：柯淳恩 / 剪接：侯冠良 / 混音：T.M.


07. 樂園：

「當尖銳眼光，當刺耳聲響，你用彩虹的浪漫溫柔包裝。」張惠妹《彩虹》

不管什麼顏色，都能找到屬於我的樂園。

世界裡的紅橙黃綠藍，它們各自代表著不同的愛，而彩色是最難得可貴的。我愛所有的你們，我很幸運能擁抱這些所有不同的愛，就算愛的結局不完美，我仍在那平凡的過程中學到不凡的愛。這段愛的旅程，成就了不一樣的我。

謝謝你，特別的人。不論將來是否還會繼續在我生命裡，因為有你，才有我的樂園。

詞：范庭瑜 / 曲：范庭瑜 / 編曲：林哲 / 合聲編寫：柯淳恩 / 和聲：柯淳恩 / 錄音：吳書涵 / 配唱：柯淳恩 / 剪接：侯冠良 / 混音：T.M.




08.盡頭

「真的會有那雙手嗎？帶我墜落還飛翔？」

負面情緒襲來時，就像巨大黑洞一樣，讓人深陷其中無法掙脫。
深陷悲傷低落時，渴望被一雙手托住，告訴我說你絕不會放手。

真的會有那雙手嗎？
誰會是我的那雙手？
我能不能成為托住自己那雙手？

就像盧凱彤的「留一秒」，
至少我還有一雙手，寫下所有的開心與悲傷，
勇敢接受自己的懦弱脆弱與自卑。

讓這雙手：「帶我墜落還飛翔」

09.黑色毛團

「它會消失的，只是需要點時間。」

我常用混濁、模糊、灰暗感的名詞來形容各種複雜的心情，像是毛線、地下水、巧克力這類物件。
黑色毛線纏繞糾結成團，粗糙的棉絮有點黑有點灰，不快樂的時候便會浮現。它會消失的，只是需要點時間。

詞：范庭瑜 / 曲：范庭瑜 / 編曲：黃德霖 / 錄音：吳書涵 / 配唱：柯淳恩 / 剪接：侯冠良 / 混音：T.M.
